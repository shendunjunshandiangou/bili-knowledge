# 威胁玩笑当场报警核验

## 一句话定义

对公开威胁（如杀害宠物）先法律框架警示，确认非玩笑则当场报警，逼对方认怂道歉。

## 详细说明

连麦者称要杀狗吃。大冰：侵害他人财物、公众平台威胁，新法下有行政处罚；「确定是玩笑吗？不是玩笑我现在立马打聊城110」；对方道歉后仍批「开了玩笑又怂成这样……很没品」。并截图快手实名主页留证。
