# 迷茫与焦虑区分

## 一句话定义

对有业绩验证者，当其自称「迷茫」时，**质疑迷茫、改标焦虑**，逼其面对真实卡点。

## 详细说明

- 「你说迷茫我不太信，你说焦虑我信」
- 依据：2年赚95万说明具备能力，「大概率不是一个迷茫的人」
- 作用：避免用「迷茫」掩盖具体决策（义乌/考公/重复路径）
