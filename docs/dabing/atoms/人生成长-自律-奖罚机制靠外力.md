# 奖罚机制靠外力

## 一句话定义

自律、自控力极难，需**外在框架靠外力**；坚持的对象是奖罚规则本身，而非时间表。

## 详细说明

大冰要点：

- 「自律是超难的事情，不怕我们外在设置一个框架，靠外力作用」
- 「你要去坚持的不是这个时间表，而是对这套奖和罚的机制的坚持」
- 奖例：完成几小时学习才有资格看相应时长手机（类似番茄钟）
