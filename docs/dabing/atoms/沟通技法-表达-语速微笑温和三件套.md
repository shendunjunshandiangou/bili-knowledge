# 语速微笑温和三件套

## 一句话定义

社交受挫时先做**外化三调整**：语速慢下来、开口微笑、勿秀聪明改**温和**——大概率措辞小问题也不致反感升级。

## 详细说明

1. **语速慢**：避免灌输/倾倒的压迫感与侵略性；快则忽略对方语言与非语言接受度
2. **开口微笑**：软化可能招反感的措辞，「真的要笑出声」
3. **勿秀聪明**：越怕被发现不够聪明越爱证明→引发前两个问题；改「温柔/温和一点、慢一点缓一点」

大冰：用笑傲江湖**剑宗**（形式外化）而非气宗（根本重塑），「你不累，不会加很多心理包袱」。
