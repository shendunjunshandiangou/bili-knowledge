# 演练人际边界现场抬走

## 一句话定义

超长连麦末尾，**现场演练**人际边界：声明耐心与服务时长已尽，该阻断就阻断——「我不必对你有任何愧疚」，并教连麦者学会被阻断也无愧。

## 详细说明

女生发烧仍追问；大冰：

- 「人际边界就是我给予你的耐心和服务的时长足够多，接下来的问题我不打算跟你梳理，我觉得我尽到我的本分了，谢谢，这是我的边界」
- 「该阻断的时候我会果断阻断，再见」
- 对直播间：「而你要学到的是——我阻断了你，我不必对你有任何愧疚，你要学到这一点，孩子啊，不管对方怎么讲」
