# 问啥人说啥信息对称

## 一句话定义

下结论前先按对方已述信息逐项追问细节（谁介绍、消费、资产），大冰自称「我是问啥人说啥」。

## 详细说明

公务员述说感情问题后，大冰追问：介绍人是否领导、女方家庭职业、开什么车、用什么包、最贵支出等，再得出「不太像一路人但可谈」。避免在信息不全时给空泛安慰。
