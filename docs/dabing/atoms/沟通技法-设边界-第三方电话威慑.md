# 第三方电话威慑

## 一句话定义

面对权力型骚扰/欺压，**让第三方（男友、朋友、自媒体身份）冷静打电话**表明知情、有记录、望对方收敛——往往足以威慑怂货。

## 详细说明

对导师性骚扰：

- 男朋友客气礼貌打电话：「您和我女朋友说的话做的事，我及家人都知道，有充分记录，希望孩子顺利毕业」
- 备选：朋友自称自媒体200万粉丝，要求约束行为
- 「足以让他尿了」——真刚的不会对学生下手，多是怂货
