# 示好换认可的危险投资

## 一句话定义

替直属上司分担份外工作、打掩护，底层逻辑若是求**等价交换或认可**——这是小兵对上司的**危险情感投资**，轮不到你来做。

## 详细说明

- 连麦者：领导家庭变故后全盘接受、打考勤掩护，期望认可与绩效回报
- 大冰：年轻内心幼稚的「投资行为」——合适时机换认可；现实职场不是这套逻辑
- 「你对得好只能证明你人好，别的什么都证明不了，别指望换回来啥」
- 与 [人生成长-职场-大恩即大仇](/dabing/atoms/人生成长-职场-大恩即大仇.html) 同场：示好过度 → 对立面
