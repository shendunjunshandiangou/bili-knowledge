# 开口前琢磨超阈值

## 一句话定义

开口前**过度美化、琢磨**超正常阈值，同事虽未见词却可感知→升起**防备心**；你不是懦弱，是思量与小算计可被解读。

## 详细说明

- 连麦者「老想如何把话美化、别攻击力那么强」——话到嗓子眼说出却不符合预期。
- 大冰：「你不是懦弱软弱，有的是思量、考虑、偶尔迸发的小算计——周围人可感知。」
- 微表情、姿态、沉默时的反应都可被捕获；「别人知道你在琢磨。」
- 解法：分档，勿全员同一套琢磨模式。
