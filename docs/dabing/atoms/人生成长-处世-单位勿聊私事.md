# 单位勿聊私事

## 一句话定义

**不要在单位跟任何人聊私事**——婚恋、想不想结婚、家庭情况等尽量不要让人知道；有人主动聊你的私事也要少回应。

## 详细说明

- 由恋爱焦虑连麦展开的核心职场边界
- 闲聊勿把话题拐到自己身上——「开始谈论你，你就已经是一盘菜了」
- 可八卦不在场第三者，勿在现场八卦自己
- 与 [人生成长-处世-主动提建议者必是是非人](/dabing/atoms/人生成长-处世-主动提建议者必是是非人.html) 同场
