# 甩锅式开场改变链接方向

## 一句话定义

公共连麦中，预先甩敏感风险、要对方替承诺承担，属于**无用开场**——会改变沟通走向，也难建立真实链接。

## 详细说明

- 连麦者模式：先预警敏感→再表达强烈诉求→仍希望在本直播间获得帮助
- 大冰：若不说预警，主播可在过程中评估调整；一旦前置甩风险，等于完成自我评估后仍要主播背书
- 复盘：「你说的这个没用的话，直接导致了我们这个事情的走向，以及我们之间的真实链接，方向性的就转变了」
