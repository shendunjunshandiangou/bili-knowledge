# 博士身份勿自恋

## 一句话定义

踏上社会时须**最大化自我约束对学历的想象**——博士是加分跑鞋，但起跑线与其他人**相对拉平**；自恋身份会阻碍审慎判断。

## 详细说明

- 好听叫美化，不好听叫自恋——不会给当下动作正面加分，只把人框死在身份心态里
- 比喻：有人穿回力有人穿耐克，你是更专业跑鞋——差别在专项，非起跑线天然靠前
- 连麦者把漫长投资当必须快速变现 → 社会不如此
