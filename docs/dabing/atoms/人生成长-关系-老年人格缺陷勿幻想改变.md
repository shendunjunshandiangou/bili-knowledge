# 老年人格缺陷勿幻想改变

## 一句话定义

对六旬仍惩戒性离家出走、像「老巨婴」对待母亲的配偶，直判**人格缺陷不可改变**——能远离要远离，别幻想感化，「用不着你来给他当妈妈」。

## 详细说明

连麦者述丈夫第三次离家出走等。大冰：

- 「人格极度不完整……有人格缺陷，搭配上年龄 60 岁上下，很肯定告诉你这是不可改变的」
- 阴暗面比重无法被高光覆盖；「一般建议能远离要远离，不要有幻想去改变他……老年人不要有想象法去改变他」
- 行为像小孩子对母亲式叛逆；离家出走带**惩戒成分**，超出简单人格缺陷
- 「强烈建议这样的关系不宜长期持久」
- 对连麦者力量：「你不是隐忍，你懂得消化——这是你的力量所在」
