# 不谴责受害者再谈应做

## 一句话定义

对延迟报警/未行动的创伤倾诉者，**先明确不谴责**，再谈「应该做的事情」与未来避险。

## 详细说明

- 「完全没有理解你没有办法报警……没有人有资格谴责你，我先向你亮明观点」
- 理由：当时未长大、缺协作、9年义务教育不完整
- 但：「这也属于我们应该做的事情的范畴之一」——合适时机认真考量
- 目的：不为过往讨公道，为「最大化规避未来潜在风险」
