# 首份工作是社会教育

## 一句话定义

第一、二份工作的核心功能是**完成社会教育/职场教育** + **经济独立存活**，而非快速成就或持续正反馈；有成长空间即极好。

## 详细说明

- 「最初的工作不是为了让你成就……是为了完成社会教育、职场教育。」
- 同时实现：有经济收入能力的独立自然人身份能存活。
- 若求职阶段强依赖「挑战性→成就感」作动力，「大概率你要失望」——正反馈晚则热血不持久。
- 自负自卑交替是「上学你和刚上班你的状态」，进入正轨后会变。
