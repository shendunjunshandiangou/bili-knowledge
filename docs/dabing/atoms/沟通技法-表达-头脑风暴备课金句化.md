# 头脑风暴备课金句化

## 一句话定义

离开原职场后，聚拢**三五好友**头脑风暴：预设年轻人议题（平级、主管、酒局、被打压）→ 共同拟**话术** → **金句化**（半小时讲话中心思想一句话）→ 可卖课。

## 详细说明

- 不要独立完成备课；从出入办公室到见甲方、末位淘汰、有才华傲气的人
- 站在管理本位想：如何面对实习生、入职两三年员工
- 整好后直播间有粉丝与信服度 → 卖课；对标普通人家进事业单位的年轻人
- 同步树立人物姿态：社会面上普通人，小议题里可发言的**发言人**
