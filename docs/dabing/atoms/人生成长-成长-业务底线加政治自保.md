# 业务底线加政治自保

## 一句话定义

业务干部成长方向：**业务与服务群众是底线**，配上**及格的政治自保意识**，避免路子走歪成「整容比谁都六」的钻营型官僚。

## 详细说明

- 连麦者问：经历痛苦后如何不变扭曲、变更好的人？
- 大冰：你是具体干业务的人，「政治」意识有点弱；方向是——不管成长到哪一天，**业务放在首位**；服务好人民群众是底线；配上及格政治意识、该有自保动作。
- 「没有哪种成长不痛苦」；责任越大压力越大，一切指向成长。
- 反例：业务干部出身却「整容比谁的话都六」——「这种的话有几个好鸟呢？」
