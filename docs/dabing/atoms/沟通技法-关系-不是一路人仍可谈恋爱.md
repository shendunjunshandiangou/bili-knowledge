# 不是一路人仍可谈恋爱

## 一句话定义

价值观、审美不同不等于不能谈；若无原则底线问题，分手往往是正常摩擦而非终审。

## 详细说明

大冰听完消费与背景追问后说「不太像是一路人」，紧接着「但不见得不可以谈」；并指出「没有涉及原则及底线性的矛盾」，「情侣之间正常的吵架嘛」「真正的分手是不吵架、直接互删，没到那一步」。
