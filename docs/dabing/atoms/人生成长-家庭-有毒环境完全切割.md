# 有毒环境完全切割

## 一句话定义

极端有毒原生家庭：**停止联系、尤其母亲**，脱离环境奔前程，与是否报警并行但不互相替代。

## 详细说明

- 「接下来你应该去做的事情：不要再和他们联系了，尤其是你妈妈……自己去奔个前程」
- 「一切的前提是脱离那个人间地狱——哪有鬼，地狱就是人间；你妈妈不是人呐」
- 切割后：「该结婚结婚该生孩子生孩子」，不必受过往太多影响
