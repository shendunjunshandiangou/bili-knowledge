# 幻想放进时间段容器

## 一句话定义

走路幻想正常，**上课走神则规定每天固定时段专门幻想**，其余时间「摔平不响」——用「容器=时间段」装下行为。

## 详细说明

大冰分级：

- 走路时想当男主角 → 「很正常，谁没想象过盖世英雄，不是大事」
- 上课边看书边幻想 → 需管：「每天固定那个点专门幻想，其他时间控制；吃饭时想，想象成在用膳」

容器隐喻：「尿憋再急也得找厕所——把它放在哪个容器里，容器就是时间段，很小一件事。」
