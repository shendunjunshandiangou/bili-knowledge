# 误连麦轻松化解

## 一句话定义

连麦者误以为未连上、与旁人私聊被全场听见时，**幽默化解尴尬**（「78万人听得很开心」），不追责，迅速切入正题。

## 详细说明

- 「过去10分钟你们的聊天，78万人全听见了，没事儿没事儿，我们也没说别人坏话，请坐」
- 不羞辱、不放大事故，降低连麦者防御后再处理实质问题（导师性骚扰）
