# 没有哄不回来的女朋友

## 一句话定义

非原则性分手往往可修复；前提是你还喜欢对方，就去哄、去追，别冷战。

## 详细说明

大冰：「没有哄不回来的女朋友，就看你怎么哄；前提就一句话，你是不是喜欢她」。并区分真分手（互删不吵）与当前状态（仍见面、仍想对方）。行动层见 [沟通技法-收尾-话头嫁接约饭庸俗化解](/dabing/atoms/沟通技法-收尾-话头嫁接约饭庸俗化解.html)。
