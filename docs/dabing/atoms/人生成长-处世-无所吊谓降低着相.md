# 无所吊谓降低着相

## 一句话定义

日常人际：**过得好过得坏、对我好坏，最后都得死**——时间拉平一切；不必那么上心、着相、投射（**大是大非另论**，须泾渭分明）。

## 详细说明

- 连麦者学到大冰「说不」很爽——基于「不必那么在乎」
- 大部分人没那么在乎你，你也不必那么在乎人家
- 中国人中庸：日常生活能缩就缩；大是大非面前不含糊
