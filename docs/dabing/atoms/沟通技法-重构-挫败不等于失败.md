# 挫败不等于失败

## 一句话定义

长期躺平、无动力常是「挫败」，不应与「人生完蛋/失败」画等号；人生算总账。

## 详细说明

连麦者31岁多年打游戏、与社会隔绝。大冰：「失败是失败，挫败是挫败……挫败不等于是失败；在你这把失败等同于完蛋，那不是那个概念——挫败而已嘛，人生算的是总账。」并指出健身/考研与「明天没泡面钱、网快断」的切身紧迫度不同，故难坚持。
