# 亲子边界规则清单

## 一句话定义

青春期与异性长辈同住尴尬，用**白纸黑字规则清单**（敲门、马桶圈、穿着等）+ **母亲沟通父亲执行**化解，而非逃避。

## 详细说明

- 12 岁女孩担心与患病父亲在济南单独相处。
- 大冰建议与母拟「执行攻略」：细化到马桶圈抬起/放下、不可光膀子、进房间敲门等 → 打印贴门后，母沟通、母监督。
- 「什么叫大人——发现了问题，想办法去解决问题，制定规则，而不是逃避问题。」
