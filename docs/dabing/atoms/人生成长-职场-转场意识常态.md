# 转场意识常态

## 一句话定义

当下及未来，工作切换、迭代、断后重开是**常态**——无铁饭碗；抱隐形铁饭碗思维者会先被淘汰；体制内亦须放弃此思维。

## 详细说明

- 对直播间年轻人公屏强调：别害怕失业，要有**转场意识**
- 现工作可先混着，有收入、不累，找到下一技能对标前**不必主动辞**
