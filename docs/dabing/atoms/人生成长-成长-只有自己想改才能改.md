# 只有自己想改才能改

## 一句话定义

**不是**家长想改或用什么方法孩子才会改——**只有他自己想改时才能改**；收入够养则焦虑多余，「先养着」。

## 详细说明

- 孩子初中才回母亲身边，秩序与动力未建立——有前因，不能完全怪孩子
- 母亲焦虑对称给孩子只增压力，屁用没有
- 儿子也听大冰，若无主观改意愿，连麦无效
