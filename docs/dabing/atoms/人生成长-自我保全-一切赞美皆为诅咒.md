# 一切赞美皆为诅咒

## 一句话定义

公众人物从第一天就知道：**今天多认同，将来势必有一天站到对立面或批判面**——「反者道之动」；因此需要合理节点沉淀、收退，既自我保全也对观众负责。

## 详细说明

大冰自述：

- 「一切赞美其实皆为诅咒」——非常清楚，但无法一对一跟每个人讲清，「先受着」
- 「花无百日红，人无千日好」——千日约 3 年
- 每天直播像走钢丝，需要合理节点：该沉淀沉淀，该避锋芒避锋芒，该收收该退退
- 目标：慢慢衰贫、合理平安着陆，对大家好也对自己好

这不是悲观，而是对粉丝周期律的清醒认知，用于解释为何要规划退出节点。
