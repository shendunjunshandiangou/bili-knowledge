# 教条化导致社会化低

## 一句话定义

过于相信**书本规则**（如法律框架 1+1=2），忽视人间**灰色调**——社会化程度低，易Ideal化、易与人冲突；母亲话多、发起关联次数多，无意识易得罪人。

## 详细说明

- 「法律是道德的底线，道德本身有弹性」——100 年前道德≠今天
- 母子碰耍混才生气——认可的是教条规则，未必普世
- 解法：**让别人多说你多听**；别因噎废食宅家
