# 金钱观沙盘推演

## 一句话定义

连麦者想「退让上交」时，用**极端支配案例**做沙盘：若接受「钱全是她的」，她拿 80% 给哥哥你能接受吗——不能接受则「接受上交」是伪命题。

## 详细说明

准新郎曾想退让：每月上交，一年清点。女方要求工资上交且视为她的钱，共同账户也不接受。

大冰推演：

- 若名义上钱都是她的，她对这笔钱有完全自由支配权
- 「她把 80% 给了她哥了，你能接受吗？」→ 不能接受
- 则「能够接受每个月把钱上交=接受他认为这钱就是他的」是**伪命题**，归根到底接受不了
- 与人好不好无关，是**三观/金钱观不一致**
