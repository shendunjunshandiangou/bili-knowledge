# 别把直播间聊没

## 一句话定义

连麦者问**敏感/灰区/涉密**话题时，前置提醒**平台风险**，要求自行判断可否公开讲、尽量隐晦。

## 详细说明

- 「你自己评判有些话题适不适合在这个公益平台讲——你别再让我陪你聊完了直播间没了」
- 「我还有一些孩子等着服务……你这边服务了，直播间没了不好」
- 无力推动改变的现象类问题可以不答
