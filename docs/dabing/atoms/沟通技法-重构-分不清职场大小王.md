# 分不清职场大小王

## 一句话定义

协同岗位中单兵过强却**不服直接领导安排、冒尖未经授权**，即没分清与直接上级、平级同事的**大小王**——环境已「很危险，墙倒众人推」。

## 详细说明

- **主持工作=班长**：「不以年羹论，谁是班长谁说了算」；不会应对对立关系就「等着被拿捏」
- 与大领导欣赏、直接领导矛盾→夹心
- 「你不是最后责任承担者，服从安排听从管理……项目最后负责任的人是人家领导」
- 单兵冒进类比：「大家拿冲锋枪冲锋，你抡大砍刀脱离大部队，他人还得掩护你、打乱阵型」
- 自学 AI 提效未下工作单→团队能力被抬到八分、直接领导被将军
- 评估：职场环境「很危险……你出次错试试，墙倒众人推」
