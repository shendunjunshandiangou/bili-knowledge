# 苹果只属于今天

## 一句话定义

**没有明天**——最好的一切只属于今天；留给「明天」则像苹果一天天口感下降。

## 详细说明

- 「你没有明天……所谓的后天都是今天」
- 苹果比喻：不舍得吃、留给明天——「没领会苹果只属于今天，跟明天没关系；一天不吃口感降一点」
- 想有用的：今天怎么活好、怎么不活亏；别为昨天错失、明天没着落焦虑
