# 男教师适度邋遢设防线

## 一句话定义

中学男教师上课不必过度收拾帅气，适度「往邋遢偏一点」是工作稳定性与防线的行业经验。

## 详细说明

大冰题外话：北方很多男教师「刻意的往邋遢里边稍微偏一些」，为工作稳定性与「设立一定量的防线」。并引「少女的心比全宇宙都复杂」「释放魅力实际上对他们是一种剥削」——学生世界小，除父母外成年男性接触少，易投射。
