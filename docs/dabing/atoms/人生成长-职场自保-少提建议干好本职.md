# 少提建议干好本职

## 一句话定义

在复杂、可能有后台的公司里，职场自保：**财务别碰、干好技术工种、少提建议——让你干啥就干啥**。

## 详细说明

大冰给女硕士的具体建议：

1. 如果想跳槽，结合年龄没有特别好的选择，可在环境内看机会、自我提升（个人课题）
2. **财务这块不要去碰**
3. **干好你本身技术工种的活儿**
4. **少提建议，让你干啥你就干啥**

适用：已判断公司逻辑非正常商业公司、本人只是执行层、无改局能力时——先保命再谈发展。
