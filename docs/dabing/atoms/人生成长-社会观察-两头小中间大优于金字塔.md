# 两头小中间大优于金字塔

## 一句话定义

互联网等平台良性结构应是「两头小、中间大（肚子大）」，而非资源集中在塔尖的金字塔——后者等于在平台内再造阶级。

## 详细说明

连麦者认同「良性发展应该是两头小肚子大」。大冰希望砍掉塔尖、流量向下覆盖，让大量普通人直播间解决基础需求；并指出「金字塔模式永远是最不科学的模式，两头小中间大一定是最好的模式」。具体运营怎么做他自认无资格，但方向明确。
