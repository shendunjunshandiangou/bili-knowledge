# 效忠后断前联

## 一句话定义

决定向**现管老大效忠**后，**不得再与前老大**在事实或口头上做任何正向连接——「当你决定效忠，你已经没有权利和前老大再有任何正向连接了，没得选。」

## 详细说明

- 连麦者听前老大建议越级打电话给一把手诉苦
- 大冰重复：「当你决定跟一把手沟通/向县老大表达效忠，你已经没有权利说或不说了（供人名）」
- 「当汉奸也得当认真一点」——效忠路径与道义已变
- 标配应：不听不信传话，只观测现老大态度，逮机**重复效忠动作**
- 前老大信息不回=枪不好使、事实弃子
