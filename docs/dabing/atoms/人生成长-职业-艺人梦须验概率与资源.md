# 艺人梦须验概率与资源

## 一句话定义

普通工薪家庭孩子想当明星/艺人——须做**调研、验概率、看家族资源配置**；经纪公司面试路概率近买彩票，星探更不可赌；小有姿色反而易被祸害。

## 详细说明

- 举例宋雨琦被追问：形象、路径是否可复制 → 概率有没有的问题
- 家庭：父亲绘图员、母亲化验——与娱乐产业无千丝万缕资源，无力保卫
- 替代：大学话剧社、合唱团、脱口秀——感受舞台即可，别做明星梦
- 本集第二段连麦；**技法层**见调研验收式追问
