# 剑宗外化优于气宗重塑

## 一句话定义

长期创伤背景者改人际，优先**剑宗**——语速、微笑、温和等外化动作；少用气宗式「从根本改变自己」，避免加摘不下的心理包袱。

## 详细说明

- 连麦者有戒备、进攻性——大面「有自我是好事」，但具体场景招致反感
- 不必「扒皮抽筋把骨头看见」——系统工程，重新养育一遍需漫长时光
- 「对标具体情况，动作上做表皮浅层调整就好」
- 引用剑宗/气宗：很多人讲气宗根本改变——对多数人**不适用**；剑宗外化表达调整「你不累」
