# 钟点工解题化家务矛盾

## 一句话定义

家务卫生引发分手纠结时，用**可支付性**解题：「年收入够请钟点工，花钱解决卫生，感情是否不受影响？」——答「那也行」则非原则问题。

## 详细说明

博士连麦者因女友不收拾家、爱打游戏纠结分手。大冰：

- 青岛年收入 20～30 万「足够雇佣家庭服务员、定时钟点工」
- 关键提问：「如果就是因为卫生问题，花钱可以解决，是否这段感情不受影响？」
- 连麦者答「那也行啊」→「那就不是一个原则性的问题」
- 若花钱解决仍不行：「只能够证明咱配不上这个姑娘」
- 收尾：「挂麦去跟他商量找个什么样的钟点工，花不了你几个钱」
