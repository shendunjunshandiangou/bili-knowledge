# 大学资源非平均分配

## 一句话定义

**大学教育资源并不平均分配**——院士数、导师密度因校而异；高中「老师撵着你学」，大学同校同专业差距会非常明显。

## 详细说明

- 张雪峰强调：「听懂扣听懂——大学的教育资源并不是平均分配的。」
- 京城大学等会宣传院士数量——即非均分证据。
- 许多大学生是「高中体制和家长用鞭子抽进大学的」，到大学后是否主动学习决定分化。
- 非 985 院校更需本科期间**主动跟老师做科研**。
