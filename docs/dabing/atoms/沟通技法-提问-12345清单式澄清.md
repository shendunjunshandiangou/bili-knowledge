# 12345 清单式澄清

## 一句话定义

在要动摇对方核心信念之前，请对方**按 12345 逐条列出**「认为自己应承担的责任或目标」，用其**自排序**作为后续重构的锚点——比直接讲道理更难防御。

## 详细说明

大冰在命名身心信号后说：「接下来您诚实地回答我的问题：你到底对家里边应担负的应承担的责任是啥，来 12345 地讲。」

连麦者自列五项：

1. 让父母不再打工、过上好日子
2. 帮父亲处理赌债、不再担惊受怕
3. 有更多时间陪父母
4. 帮弟弟还网贷（强调非伏地魔、只帮一次、弟弟自己的责任）
5. 尽量 35 岁前过特别快乐的生活

大冰随即指出：**「您的第一项你重新重复一下」**——对方复述「让他们不再打工」——对比大冰给出的第一项：「你好好活着，以身心健康的状态存活……这是你为人子女面对父母应尽的第一项职责。」

技法价值：

- 清单来自对方口述，重构时不是「我替你说」，而是「**你的排序里缺了什么**」
- 自然过渡到「排序里看不到这个我字」
- 可复用于责任过载、目标冲突、主次颠倒的连麦场景
