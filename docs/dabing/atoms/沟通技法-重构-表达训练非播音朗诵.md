# 表达训练非播音朗诵

## 一句话定义

真正做好输出与表达，**最重要不是播音主持和朗诵**——「通通扯淡」；行内出身称易**教坏**扼杀秩序思考。

## 详细说明

- 大冰：「我是行内人啊……教不好，教坏了，大有人在，很有可能就敏杀了（扼杀了）你本来已经形成的秩序性思考能力。」
- 正路：大量阅读枯燥理性书（拿笔划重点）+ 科幻保想象力 + **一切小说不读**（青春期后再对接文学）。
- 兴趣培养是「无心插柳柳成荫」——积极拥抱但不指定开花结果场景（可能拍卖师、危机公关、或网络直播间）。
