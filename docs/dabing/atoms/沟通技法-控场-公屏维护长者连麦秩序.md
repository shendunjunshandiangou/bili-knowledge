# 公屏维护长者连麦秩序

## 一句话定义

长者连麦遭公屏不当言论时，暂停对话、公开立三条规矩（禁有罪推论、爱幼尊老、给说话空间），再继续。

## 详细说明

教授讲述童年创伤时，大冰对公屏说：谨慎「有罪推论」；中国人有爱幼尊老之心；「大概率好多年没有人给他机会认认真真听他来讲讲话」——请想听就听，不想听划走，「没有必要乱讲话」。
