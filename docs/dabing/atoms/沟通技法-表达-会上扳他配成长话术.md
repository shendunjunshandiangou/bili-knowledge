# 会上扳他配成长话术

## 一句话定义

事业单位内斗时，**冷静观察、最好在会上**点出对方行为事实或工作疏漏，并接「为工作负责、助同志成长」话术，而非私下忍或发疯。

## 详细说明

- 大冰用山东话「扳他」：小木子观察好了、有的放矢，**最好是在会上来办他**。
- 点行为事实、不作为、严重工作疏漏后，**必须追上一句话**：「我是为了对咱们的工作负责，也是为了帮助同志你更好的成长，你能理解我的良苦用心呀？」
- 德服人往往不够；立威后若仍不配合，即工作态度问题，矛盾可摆桌面，主管知情；自己零疏漏、积极推进需配合事项。
