# 可选择答与不答

## 一句话定义

在对方预设多题、拷问式提问的开场，先声明**有权选择回答与否**——「你可以问，但是我也可以选择答还是不答」——把互动从「必须应答的审讯」改为「可协商的对话」。

## 详细说明

连麦者开场：「请开始你的表演」「我能试问你，一个是问你四个问题吗？」

大冰接话：

> 「一个一个来可以吗？」「你可以问，但是我也可以选择答还是不答。」

**作用**：

- 提前设定边界，降低对方「连珠炮审问」的预期
- 与后续「你说得对但不解释」策略一致——既然可以选择不答，实际选择「答」时也只需最低限度回应
- 适用于直播连麦、公开 Q&A、面对带有挑衅意味的多题盘问
