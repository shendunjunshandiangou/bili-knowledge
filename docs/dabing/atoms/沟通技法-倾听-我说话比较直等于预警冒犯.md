# 我说话比较直等于预警冒犯

## 一句话定义

连麦者自称「性格直接」时，解读为：**不吝啬向外展现攻击力、预先预警「别惹我」**——之后说了啥对方不能反弹对抗。

## 详细说明

大冰：「但凡一个人上来说我说话比较直……我们这边统统解读为：我是一个不吝啬冒犯他人的人，我已经预警了。」

- 「我说话比较直啊，就你别惹我啊……接下来不管我对你做了啥说了啥，你不能够惹我、不能够反弹、不能够对抗」
- 指向长期职场人际问题，而非某一刻「嫉妒」爆发
- 连麦者若用对大领导的沟通方式面对直接领导，「不至于现在的处境」——说明并非不会柔软，是**没分清对象**
