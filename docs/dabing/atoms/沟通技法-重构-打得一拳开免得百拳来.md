# 打得一拳开免得百拳来

## 一句话定义

遭办公室小团体打压时：工作零疏漏是立身之本；**逮完全占理的一次机会果断捍卫**，力度到「得理不饶人边界再往前一小步」，有人劝立马停。

## 详细说明

- 新入职遭 C 位同事恐惧+小团体排异
- 第一：工作上这时期尽量不要疏漏——职场关系不直接决定工作存续
- 第二：「打得一拳开，免得百拳来」——残酷现实，非教办公室政治
- 须判断时机、完全占理；力度与烈度控制到得理不饶人边界
- 对方从背后走过贬低、有第三方在场——蓄意形成联盟
