# 不必防卫直接收下提醒

## 一句话定义

被点破问题后，阻止连麦者长篇自证「我不是那样的人」——目的在讲未来，「一句话谢谢提醒我有数就好」。

## 详细说明

大冰提醒：因帅哥可能脑补优良品质，未来恋爱多留理性。女生长篇辩解：在婚介所认识、是他追的我、不是外貌协会……

大冰解读：

- 「你不必做防卫动作……我们主要目的是来讲你的未来和将来」
- 潜台词：「你不要那么说教我……我不接受你的建议，因为我将来绝对不会踩第二次坑」
- 正确回应：「谢谢你的提醒，我有数就好了嘛」——「哔哩吧啦一长串解释，一句话就可以」
