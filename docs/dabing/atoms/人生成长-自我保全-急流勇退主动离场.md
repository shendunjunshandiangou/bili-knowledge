# 急流勇退主动离场

## 一句话定义

清楚自己的「下场」，但选择在被打倒之前**像以往一样主动悄悄离去**——中国历史上真正厉害的人，谁不是急流勇退。

## 详细说明

大冰对老大哥说：

- 「我知道我的下场，我一定知道我的下场」
- 但会在被打倒之前，自己主动选择悄悄离去
- 「中国历史上真正牛掰的人谁不是急流勇退啊，只要他不肯急流勇退的，那他一定不够牛掰」
- 即便无法全身而退，「死则死矣」

这与「一切赞美皆为诅咒」形成配套：不是被动等批判，而是主动在节点收退。
