# 发现短板现在补不晚

## 一句话定义

大学摆烂、无存量技能者，**承认短板并立即自我教育**，比抱怨社会或挑工作更重要。

## 详细说明

- 「发现了自己到底缺在哪儿了，现在补还来得及」
- 「耐住心房让自己长长本事，展开自我教育，孬好学点东西」
- 有因有果：「因处摆烂→果处收获结果；你早干嘛去了」
