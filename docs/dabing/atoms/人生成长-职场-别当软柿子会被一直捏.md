# 别当软柿子会被一直捏

## 一句话定义

职场里**一旦当软柿子**，会被持续拿捏，「一直把你捏呲了捏出汤来」——退让没有终点。

## 详细说明

- 大冰开场与收尾重复强调：「尽量别当软柿子啊，软柿子一旦当了软柿子的话，是捏不完的。」
- 连麦者代理科长遭同事架空、甩棘手问题、说她说话没底气；曾努力维持关系、天天加班哭。
- 错误自救：办公室发疯；正确方向：研究对手、会上扳他、向上对称、与领导联盟。
