# 饮鸩止渴隐喻

## 一句话定义

用**即时解渴但后续致命**的比喻，点破短利路径（如牺牲尊严换钱）的长害。

## 详细说明

- 「如果一杯水里大肠杆菌超标100倍，你喝吗？」
- 「茫茫大海特别渴喝海水——当时缓解，接下来会发疯」
- 对「有钱就能体验更多好」：「不是饮鸩止渴吗？可能又是毒药」
