# 孝敬要看真实实力

## 一句话定义

对父母的孝敬（经济帮助、陪伴）应匹配**当下的真实实力与发展阶段**，不是凭「多想」立刻实操；刚入职阶段「没有多少油水可挤」，硬挤压对自己和对方都可能弄巧成拙。

## 详细说明

连麦者见妈妈饭店打工、觉得妈妈老了很多，想孝敬。大冰梳理：

- 「你想孝敬妈妈这件事情还是和上一个一题一样，要看你的**真实实力**」
- 刚入职，若精力正儿八经投在工作上，「几年之内」才可能成为带团队、领年薪的人——**那时**再照顾妈妈
- 「妈妈等得及」——按年龄推，妈妈「没有老的那么快」
- **三岁搬煤气罐比喻**：幼时见妈妈搬重物很想帮，但只有三岁，帮可能压手、脱手砸腿——「你对他的帮有可能会弄巧成拙」

结论：「先对自己好，先发展自己的事业」；当下适当表孝心（买东西、红包）可以，但不是把自己挤压到崩溃。
