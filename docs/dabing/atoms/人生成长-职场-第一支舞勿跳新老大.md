# 第一支舞勿跳新老大

## 一句话定义

职场大忌：**永远跟着第一个带你去舞厅的人跳第一支舞**——勿从引路的老大跳到新老大，面上背叛难纠，他人对你界定偏差。

## 详细说明

- 连麦者由「前老大」带到公司，后转「现老大」部门（自认转技术岗）
- 大冰：「从钱老大那边跳到县老大这边，犯职场大忌……尽量不要做面上背叛行为」
- 他人解读：「从那一刻开始你已经有问题」——前老大口头答应转岗「那些东西不算数」
- 现老大清洗/边缘化；连麦者已成弃子
