# 答案只是阶段性工具

## 一句话定义

找答案、设目标（如买房）是**阶段性辅助工具**，不是人生终点——不能靠答案活着。

## 详细说明

姑娘长期靠「找答案」和「要有自己的房子」支撑，达成后失动力。大冰：

- 「我们靠答案活着吗？……它是可以阶段性来辅助你的一个工具」
- 「谁告诉你那个东西就是你的终点了呀？不能够靠答案活着」
- 正活法是：该睡觉睡觉、该吃饭吃饭、碰见心仪的人该表达就表达
