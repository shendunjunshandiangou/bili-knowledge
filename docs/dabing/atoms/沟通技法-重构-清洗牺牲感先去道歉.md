# 清洗牺牲感先去道歉

## 一句话定义

恋爱中若曾把选择失衡归因「为你牺牲」，须先彻底清洗此念头；若已对伴侣表露，**先去道歉**再谈改变。

## 详细说明

大冰判断连麦者对女友有「蔑视式」依赖（像对确信爱子的妈妈）。模式不可长期持续——职场压力会加码要求，女友包容有阈值。改变第一步：「放弃我是为了才有今天、为你付出牺牲——完全清洗干净」；「如果此前已经做了此番表露，你去给我道歉」。
