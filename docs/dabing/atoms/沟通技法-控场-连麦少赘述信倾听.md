# 连麦少赘述信倾听

## 一句话定义

连麦求梳理时**少赘述、信对方倾听与发问**——啰嗦=不信任倾听能力，反而显弱者；该闭嘴时闭嘴。

## 详细说明

- 连麦者怕框架说不清而长篇大论
- 大冰：「你说那么多干嘛呀……这个叫做不信任我的倾听能力，或者说是不信任对方的发问能力」
- 「这反而证明了你是弱者」
- 教：「不该说话的时候把嘴闭上；该说话时几句话简短说完，不必赘述；能直接给结论就不给解释」
- 话题跳转被纠正：「你必须得学会这一课……机会你自己浪费了」（指先结论课）
