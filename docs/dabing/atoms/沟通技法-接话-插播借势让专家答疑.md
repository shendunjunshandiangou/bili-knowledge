# 插播借势让专家答疑

## 一句话定义

连麦者的问题恰好可由**在场更合适的人**回答时，快速把提问权转给对方，并制造紧迫感促其开口，而非自己硬答。

## 详细说明

大冰与张雪峰视频通话中，随机连到一位家长咨询女儿专科报志愿。大冰没有自行分析，而是立刻转场：

- 「你赶紧问，千载难逢的机会，快点快点。」

**操作要点**：

1. 识别问题是否超出自己专长或他人更权威。
2. 简短说明机会难得，降低连麦者犹豫。
3. 把舞台让给专家，自己退到主持/搭桥位置。

本集张雪峰随即给出兽医、护理等具体建议，大冰后续再以祝福收尾，形成「专家答实质、主持人答情绪」的分工。
