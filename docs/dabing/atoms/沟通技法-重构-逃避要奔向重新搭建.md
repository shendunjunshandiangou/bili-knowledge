# 逃避要奔向重新搭建

## 一句话定义

换环境初期叫逃避不算错，但**一味只逃大错特错**——新环境须多维度「重新搭建」，可覆盖旧创伤，而非仅在单一维度（如学业）转移。

## 详细说明

藤校博士一生在换城市、换国家。大冰：

- 「逃避本身在初期阶段展开这个动作的时候没有错，但是如果你一味的只是逃的话，这个东西就大错特错了」
- 「逃开原有的问题，给自己搭建真的全新——而这个真的全新是可覆盖之前的宿病宿因问题的」
- 她之前的逃离只在学术/学业单一维度搭建「我」，未在其他层面滋养
