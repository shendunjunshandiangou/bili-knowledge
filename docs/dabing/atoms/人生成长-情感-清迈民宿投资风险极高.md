# 清迈民宿投资风险极高

## 一句话定义

通过清迈开民宿「正常经营赚钱」**风险值极高**；若为灰色转让费模式，出资方可能颗粒无收。

## 详细说明

- 三点：①正常经营赚钱风险极高；②目前去开「极不理智」——很多民宿想出手找不到接盘；③若非获客盈利而是转让费，与国内西南民宿性质不同
- 背景：疫情后中国游客赴泰人数下降，清迈是晴雨表，「宁曼路以前满大街中国人，现在基本看不到」
- 灰色地带：可找泰国人合伙、你主要出资——「从爸爸那边借来的钱可能颗粒无收，颗粒无收都是好的预判」
- 大冰依据：「你这些地方有我大量朋友吃过见过，相关案例见过不少」
