# 牺牲尊严换钱代价

## 一句话定义

以尊严为代价的「颜值红利」会在回溯时导致**自我认同极低**，且难逃社会原罪化压力。

## 详细说明

- 「不能以牺牲掉一些别的东西为代价，那个东西里面甚至包含尊严」
- 「若干年之后……自我认同、自我认可度会极低」
- 「不要以为有密不透风的墙……承受不了家庭压力、社会压力、别人对你的原罪化解读」
