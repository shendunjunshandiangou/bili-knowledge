# 连麦只收窄解一题

## 一句话定义

连麦者存量不清、诉求宏大时，**只帮解决一个问题**（如今日只做自我介绍），**不做具体职业推荐**。

## 详细说明

- 连麦者希望大冰直接告诉「适合干什么」；大冰：「我不是这样的主播……我们对您的硬实力在哪不了解。」
- 「今天帮你解决这一个问题行不行」——自我介绍；「自媒体工作室这块兄弟帮不上，能力有限。」
- 建议：先打工建立秩序与逻辑性，「再来考虑其他可能性。」
