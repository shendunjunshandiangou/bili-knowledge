# 道德是底裤只能自己穿

## 一句话定义

**道德是一个人的底裤**——可以自己穿上，不能要求别人也穿；爱心、公益是你自己的事，不能跨直播间借钱绑架他人。

## 详细说明

对自称借钱资助失足妇女实则嫖娼者：

- 「爱心是你自己的事，就像道德是一个人的底裤，你可以拿来自己穿上，但不能要求别人呢」
- 为什么不自己赚钱资助，而要借钱？
- 可在自己合理范围内解决，不必拿「很有情怀」的事要求别人
