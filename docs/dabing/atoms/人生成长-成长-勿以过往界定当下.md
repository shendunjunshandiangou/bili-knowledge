# 勿以过往界定当下

## 一句话定义

职场下放、竞争失败后，**勿用过往 glory 界定当下与未来**——「美好的仗打过了，它已经过去了」；不同阶段可有不同追求（如健康心态）。

## 详细说明

- 21 年后下放未必坏事，此前是安全着陆
- 求学拼学历、工作拼升迁；当下可拼心情、心态、身体——「何错之有」
- 与 [人生成长-择业-怎么选都会后悔](/dabing/atoms/人生成长-择业-怎么选都会后悔.html) 互补：过去不决定未来走法
