# 主观认定需客观依据

## 一句话定义

除主观论述外，须有**客观事实依据**；无法提供验证的，应去找完全相信你的人，而非在公共直播间求证。

## 详细说明

对小杨「被针对」「预知未来」：

- 「除了我们的主观认定之外，还要有客观依据」
- 报警不顶用可能说明理由不成立；没有证据则「别人可能没有针对你」
- 可当场预测彩票号码做验证
- 收尾：「我相信的是有本之者、有原之者、有用之者……如果没有，希望遇到完全相信您的人，抱歉我不是」
