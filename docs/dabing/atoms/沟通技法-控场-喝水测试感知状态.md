# 喝水测试感知状态

## 一句话定义

对高压激动连麦者，用**喝水仪式**观察其能否软着陆，语言信息有效往往≤30%。

## 详细说明

- 「去倒杯水……慢不下来什么事都没法聊」「听不见喝水声，再去倒一杯慢慢喝」
- 「语言信息有效作用不超过30%，语气/状态信号远大于语言本身」
- 测试结论：无法软着陆（「垂直砸下来」）→更需环境变迁
