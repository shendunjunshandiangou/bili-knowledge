# 别把所有事归因嫉妒

## 一句话定义

职场中被孤立，别全归因「嫉妒」——多与**切身利害**（如奖金由直接领导决定）相关；真嫉妒需业务线交错，「没那么多闲工夫」。

## 详细说明

- 连麦者放大「嫉妒」成分；大冰改词：可能是不喜欢、讨厌
- 「真实嫉妒在大部分职场环境会发生，但频率不会那么高」
- 需「切身利害发生关联」——追问奖金分配：「领导决定好了，那就懂了……牵扯到的利益不是小数」
- 「别所有事都往嫉妒上归，心态太中二……回缩到高中宿舍鸡毛蒜皮的程度」
- 与有利益关系者冲突、无利益关系者喜欢你——符合利害逻辑
