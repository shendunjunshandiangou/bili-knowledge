# 工作改变是成长切入点

## 一句话定义

失业或困局时，**唯一可落地支点**是谋生、参加社会劳动——先改工作，调整会辐射到感情与其他议题；不存在三个方法分治三个问题。

## 详细说明

- 「工作是你现在唯一肉眼可见的落地支点……以此完成的动作折射一定量的自我调整，辐射到其他议题」
- 无新对象时凭空谈感情是「画饼」
- 「没长大呀，孩子」——完成社会化从工作动作开始
- 与 [沟通技法-重构-一口井自我设限](/dabing/atoms/沟通技法-重构-一口井自我设限.html) 的扩大地域方案一体
