# 朋友的朋友勿增实体

## 一句话定义

**不必**把朋友的朋友都发展成哥们——大概率对方把你当资源方；「如无必要，勿增实体」，熟人即可，别给自己惹麻烦。

## 详细说明

- 连麦者老想与朋友的朋友也搞好；大冰：「不是病态，叫很二逼，没必要。」
- 老家朋友介绍的朋友若混得一般，「大概率还没有你高……把你当成具有资源的那一方，希望有一天你效劳。」
- 你没有看清交换点却「先揽住着」→ 累且易得罪一批人。
- 本乡本土多点熟人求医问药是好事，「但不宜走得太近。」
