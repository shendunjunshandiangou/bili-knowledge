# 社工扩大人际基数

## 一句话定义

深度关系/long-term 关系受阻时，通过**社工/NGO/义工**扩大人际接触基数，在短平快互动中寻求正反馈与可能的深入关系。

## 详细说明

对 CPTSD、记不住人、难维持关系的博士：

- 建议 gap 期间做社工类工作，「大量的去接触人……需要承受一点点压力但要去做接触」
- 「以此扩大你的基数，扩大基数之后看看有没有可能性有深入关系的产生」
- 美国可选无太多宗教背景的 NGO；国内可选成都、济南等烟火气足的城市
- 与仅旅行/独处/培养兴趣不同——须「通过做事产生人际链接」
