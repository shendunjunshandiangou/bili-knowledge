# 老好人包容度更低

## 一句话定义

职场**老好人**因底线设得低、自我弱，他人包容度反而低于「说话难听有标签」的人——稍不符合预期就被说「你怎么这样」。

## 详细说明

- 「但凡被贴标签说话非常难听……大家包容度非常高。」
- 老好人「说一点不符合预期的话，他们就受不了——因为感知到你底线非常低、自我没那么强。」
- 玩笑有度：超出度须表达内心不舒服。
- 分档后适度「难搞」可省去麻烦，非真侵略。
