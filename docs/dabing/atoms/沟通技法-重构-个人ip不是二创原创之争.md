# 个人IP不是二创原创之争

## 一句话定义

自媒体困境应重框为「做个人IP」，而非在二创与原创、追爆款之间摇摆。

## 详细说明

连麦者二创混剪流量下滑、原创爆款率低、心理过不去「营销号」坎。大冰：认知小问题不在二创/原创，「抓紧时间做个人IP」——「做个人IP不等于做原创」；泛原创/二创都会被「等爆款、10万赞」牵着鼻子走，走不远。
