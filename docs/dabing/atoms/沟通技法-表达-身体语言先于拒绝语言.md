# 身体语言先于拒绝语言

## 一句话定义

拒绝他人时，**先做身体语言**（摆手、摇头、转身、避眼神），再做口头拒绝——避免犹豫几秒后被说服。

## 详细说明

大冰教职场拒绝：

- 「身体语言先于你的唇齿化与语言，先做身体语言，再做语言对称，要不然中间空的这几秒钟会让你内心犹豫，偏向另外一个方向」
- 流程：判断不行 → 先摆手/摇头 → 再给「不行不行不行」→ 可选补理由
- 现场带练：模拟同事甩活，纠正「不够坚定」
