# 事业编非绝对稳定

## 一句话定义

北方人说的「编制」多指**公务员**；**事业编**只是相对非体制内较稳，并非不会被分流、并非铁饭碗——与公务员「两码事」。

## 详细说明

- 连麦者误解：单位经营不利也会分流，「永远有工作」不成立
- 相对稳定 = 与私企公司比；不能向上与公务员体制比
- 若奔赴事业编却谈绝对稳定，逻辑本身不太合理
- 与 [人生成长-职场-转场意识常态](/dabing/atoms/人生成长-职场-转场意识常态.html) 时代判断一致
