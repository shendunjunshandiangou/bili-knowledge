# 接住前辈关心如实汇报

## 一句话定义

当资深观众/前辈表达对其收入、前途、停播的心酸与担心时，**不回避、不给虚假安慰**，如实汇报计划、价值观与退场预案。

## 详细说明

老大哥担心大冰收入、快手受众、夏天后熄火。大冰：

- 认同对方预判同步，但不给对方压力（「你不要有压力」）
- 坦诚直播间人多了、自己也没那么放松
- 系统汇报继续直播的四条原因（勇敢、能量要用、不背叛过去/年轻自己、清楚下场仍坚守）
- 主动披露急流勇退与写书诉求，而非敷衍「放心我会很好」
