# 合则聚不合则散

## 一句话定义

梳理婚恋分歧时不评判谁错，只问**是否同频**——金钱观达不成共识「没法走在一起」，接受不了就「趁早分」。

## 详细说明

- 「合则聚，不合则散」——严重程度可比吃不到一块、睡不到一块
- 「这不是谁退谁让的问题……得找到基本盘认知相同的人，最起码对钱的认知相同」
- 对方想找「心甘情愿把钱给她、钱就是她的、再发点零花钱」的人——「你如果不是这样的人，不是忍多少就能同频，这是长期行为，往后余生都在这状态」
- 「接受不了的话趁早分……可以上升到三观了」
- 梳理不意味着一定谁做错，是**合不合适**
