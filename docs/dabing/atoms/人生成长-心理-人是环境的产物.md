# 人是环境的产物

## 一句话定义

**人出问题第一步是改变环境**；改变不了则重新选择环境，而非持续自我攻击。

## 详细说明

- 「人是什么的产物？人是环境的产物」
- 路径：改造原环境（对此人效果不抱期望）→ 重新选择环境
- 「只要不离开这个环境，所有议题都会下意识归因到自己身上」
- 复读≠脱离环境；上大学更可能脱离家庭场域
