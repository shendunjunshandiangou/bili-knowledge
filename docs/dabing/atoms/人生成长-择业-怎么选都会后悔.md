# 怎么选都会后悔

## 一句话定义

重大抉择（工作、婚姻）**怎么选都会后悔**——关键不在选，在**选之后怎么走**、创造什么可能性；怕选错而拧巴是误区。

## 详细说明

- 选钱多的或选清闲的，都取决于选后通道里的自我创造
- 连麦者以为选好了就有不变存量——并非如此
- 「真正想要什么」要上班一两年后再界定——社会教育才开始
- 与 [人生成长-成长-勿以过往界定当下](/dabing/atoms/人生成长-成长-勿以过往界定当下.html) 互补
