# 人不对离开事不对离开环境不对离开

## 一句话定义

三句决策框架：**人不对，离开人；事不对，离开事；环境不对，离开环境**——且须奔赴对的，而非消极逃避。

## 详细说明

大冰完整表述：

- 「人不对，离开人，去奔赴什么呢？对的人」
- 「事儿不对，离开事，去创造对的事」
- 「环境不对，离开环境，去搭建新的环境」
- 「这都是积极向动作啊」

对藤校博士：配偶不对、婚姻受阻、环境基数太小——可 gap 换场域，通过社工/NGO 做事重建链接。
