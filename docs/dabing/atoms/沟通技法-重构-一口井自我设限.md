# 一口井自我设限

## 一句话定义

把所有问题限定在**一口井**（如必须留某城）里求解，世界太小解决不了；不怕设限，怕设限后找一堆**合理化理由**——「不破不立」。

## 详细说明

- 解题：扩大求职地域，「哪有饭去哪吃」——概率基数问题
- 饭店比喻：打烊/菜不对口仍坚持在这家解决 = 自我设限
- 「你当然有权利选择原地躺下甚至往后出溜，但积极正向是往前试一步」
