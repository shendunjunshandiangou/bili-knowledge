# 表达不清也是修炼

## 一句话定义

连麦开场 normalize 不完美表达：逻辑清晰碰撞思考，表达不清「也不是坏事」——降低连麦者压力。

## 详细说明

大冰开场：希望连麦者逻辑清晰、用词精准，「但很遗憾没有太多这样的机会」；反推「如果上来表达不清晰、逻辑不清晰，难道真的是坏事吗？也不是」——「感恩诸君助我修炼，虽然修得不好，但肯定比去年好」。再请对方自我介绍进入正题。
