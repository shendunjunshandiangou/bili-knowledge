# 通过助人建立对外攻击力

## 一句话定义

当对外攻击力缺失、易对内攻击自己时，选**具体可执行的助人通道**（志愿者、社工、救助动物）在做事中重建力量感。

## 详细说明

大冰对两位连麦者均给出此建议：

- 儿童福利院、慈善机构、社区社工、流浪动物救助——「选一个通道，挂了麦去搜」
- 「帮助的过程是建构你自我当下阶段很好的一种方式」
- 反社会人格少女：「去帮帮别人，把你的力量往这个方向用」——与看暴力内容相对

比泛泛「去爱人、交朋友」更具体、更可执行。
