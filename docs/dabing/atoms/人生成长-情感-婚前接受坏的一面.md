# 婚前接受坏的一面

## 一句话定义

决定结婚前须观察对方好坏两面，并确认对「不爽的一面」可接受；否则婚后会不停要求改造对方。

## 详细说明

大冰两条：①足够长的时间发现好坏面，有包容度——「婚恋是一次选择」；②好的关系需双方共同成长，单方成长不算好关系。警告：「糟糕的婚姻是永远希望对方改变、改造对方」；「哪怕最美好婚姻也想掐死对方二三十回」，看接受度是否在婚前想明白。
