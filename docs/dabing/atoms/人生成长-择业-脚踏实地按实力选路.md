# 脚踏实地按实力选路

## 一句话定义

选专业、选工作要**按分数与实力**务实选择，不挑三拣四；有稳定工作、能在城市里，已属不错。

## 详细说明

张雪峰对专科女生家长送四个字：**脚踏实地**。

要点：

- 「有多少分说多少分」，不要「这不愿意干、那不愿意干」。
- 「实话说得咱成绩在这摆着」——承认分数约束，在此前提下选适合的路。
- 「有一个工作，有一个稳定的工作，能在城市里边，我个人觉得就已经很不错了。」
- 「一定要根据自己的实力去选择适合自己的路。」

与大冰「选什么专业都掉不了地上」的祝福形成互补：一方给情绪定心丸，一方给务实行动准则。
