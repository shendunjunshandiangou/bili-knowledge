# 学表情作职场第一步

## 一句话定义

i 人职场破冰**最低成本、最快正反馈**的第一步：对标优秀同事，**先学表情**（微笑/苹果肌），再学话术——媒体行业微笑是「标配小盾牌」。

## 详细说明

- 连麦者「扑克牌脸」；大冰：「第一步学表情……最容易上手。」
- 观察不同对象说话时的微笑姿态；「臭着脸的人都不认为自己是臭着脸。」
- 媒体单位：「微笑是标配动作……一面小小的盾牌，挡掉不必要的麻烦。」
- **分环境练**：采访对象处「脸可以笑烂了」获正反馈，再回单位对同事领导用。
- 模仿 1–2 名优秀女性外化表达，「演着演着假的就会变成真的」；勿学成欺下瞒上。
