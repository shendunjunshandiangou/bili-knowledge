# 先结论后补充

## 一句话定义

职场/连麦表达：**先落地有声给结论**，对方发问再补充；对方 get 到就闭嘴——别从开天辟地论证半天。

## 详细说明

大冰教 00 后连麦者（连麦啰嗦被纠正）：

- 「先给结论再补；结论得到共识不必补充；对方发问你再补充」
- 「一旦对方 get 到你的点，立马闭嘴」
- 反模式：「先论证半天最后才结论——永远控制不好说多说少，大概率说多；逻辑线越长越容易把自己说乱」
- 类比小学作文：先开题/立场，再展开再收；成年人平级交流「几乎没人能耐心听 2 分钟以上」
