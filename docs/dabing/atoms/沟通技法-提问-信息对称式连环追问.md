# 信息对称式连环追问

## 一句话定义

下结论前先通过连环提问把**时间线、资源、能力、动机**等信息对称到位，避免在信息不全时定性。

## 详细说明

- 先问具体支点：「此前形成积累的有哪些」「写小说发表过吗」「美术达高考水平吗」
- 再问关系结构：认识多久、对方年龄、在哪认识、提某方案几次、父母有没有钱
- 追问对方态度：「男朋友鼓励你读还是建议退学？从什么时候开始？」
- 信息够了再表态：「自始至终没有定性你是坏人」→再给出风险判断
- 职场困境版：城市→行业→公司规模→双职工岗位与薪资→学历→来深前职业→对方现意向（如出家），再谈休息/考编等建议
