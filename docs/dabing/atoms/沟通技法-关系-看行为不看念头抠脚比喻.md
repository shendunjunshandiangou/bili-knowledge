# 看行为不看念头抠脚比喻

## 一句话定义

有道德焦虑时区分「内心念头」与「 outward 行为」——行为防线到位则念头不必过度自责。

## 详细说明

大冰：「人非圣贤……重要的是看行为」。比喻：出门衬衫领带，回家抠脚丫子没问题；「只要不坐在地铁上当众抠脚丫子，你就不是个有问题的人」。老师确认行为从未逾越失德底线，大冰：「对对对就行」。
