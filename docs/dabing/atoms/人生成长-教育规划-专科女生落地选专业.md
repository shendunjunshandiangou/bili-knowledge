# 专科女生落地选专业

## 一句话定义

专科段女生报志愿宜选**就业落地、性别劣势较小**的专业；张雪峰本集建议重点为**兽医**与**护理**。

## 详细说明

**推荐方向**：

| 专业 | 理由（据张雪峰原话） |
| --- | --- |
| 护理 | 找静脉、打针等操作短期难被 AI 替代；老龄化社会需要大量护理人员 |
| 兽医 | 养宠人群增加；宠物看病多数不问医生学历，就业门槛相对不高；宠物医疗消费意愿强 |

**不建议方向**：

- 理工科研发：专科若真有研发所需学习能力，高考不至于只上专科；与高分生竞争研发不现实。
- 制造业一线「进厂」：工作条件相对艰苦；同等条件下用人单位往往更倾向男生。

张雪峰原话：「女孩子专科段，我强烈建议就是落地一点，一个就是兽医，一个就是护理。」
