# 错过皆非好物

## 一句话定义

**曾经错过的人事物（前男友、前工作等）通通不是好的选择**——重复强调以破「失去的是最好的」执念。

## 详细说明

- 「人生在世，只要是你错过的东西，通通不是好东西」——重复三遍
- 「只要错过了，就不是最好的东西……只要是后来她就不是好的选择」
