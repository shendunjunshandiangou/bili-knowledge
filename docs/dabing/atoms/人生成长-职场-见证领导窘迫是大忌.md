# 见证领导窘迫是大忌

## 一句话定义

职场中**了解、介入**领导家庭变故等窘迫私域——即便全员知晓、即便善意，也是大忌；别人知道却不示好，说明不该做。

## 详细说明

- 「你去了解这个东西干嘛？参与介入这些东西干嘛？」
- 被洞悉背施恩方心理压力大 → 易生 [人生成长-职场-大恩即大仇](/dabing/atoms/人生成长-职场-大恩即大仇.html)
- 连麦者成为「代表」——做得最多，领导潜意识希望你成对立面
- 对策：工作关系框架内沟通，超出框架别碰
