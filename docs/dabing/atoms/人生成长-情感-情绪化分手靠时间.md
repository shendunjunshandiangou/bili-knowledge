# 情绪化分手靠时间

## 一句话定义

逻辑解构、理性推导**成分极低**、易被感觉绑架的失恋者，走出伤痛**大概率靠时间**——非所有人适用，但对此类连麦者是诚实界定。

## 详细说明

- 连麦者「想到哪句说哪句、凭感觉梦到哪句说哪句」；大冰：理性推导成分极低。
- 拒绝逻辑解构路径→「大概率靠时间；时间还不够」——分太短。
- 下一段感情无保票：「酸甜苦辣都会有，合适就在一起，不合适就分。」
- 勿向公屏解释必要性——反而加强界定。
