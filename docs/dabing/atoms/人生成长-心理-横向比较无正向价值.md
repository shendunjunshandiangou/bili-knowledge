# 横向比较无正向价值

## 一句话定义

把同学「都很优秀」置换为「**过得比我好**」再追问——横向比较**无正向价值**，只会拖后腿；推动行动而非想象。

## 详细说明

- 大冰让连麦者重说：「我身边的人都过得比我好。」
- 「你确定他们过得真的比你好吗？可能他们拥有你没有的，你也有他们没有的——比如压力。」
- 「没法比……有人生下来继承上亿家产，要比吗？不要比。」
- 若比较力要用，则「夯实考学深造想法，趁着还有可能先往前走。」
