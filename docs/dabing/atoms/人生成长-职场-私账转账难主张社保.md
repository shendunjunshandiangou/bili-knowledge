# 私账转账难主张社保

## 一句话定义

**私人转账、无工资备注、非固定长期用工**时，事后主张补社保/劳动仲裁**举证困难**，「不大好弄」。

## 详细说明

- 大冰连问：是否私账转账、转账是否备注工资。
- 若无工资备注、非长期固定岗位、另有其他收入 → 劳动仲裁「可能不大好弄啊」。
- 多年旧事更建议「重新找个活干吧」——现已有工作则先往前看。
