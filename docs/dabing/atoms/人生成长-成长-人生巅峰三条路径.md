# 人生巅峰三条路径

## 一句话定义

人生走到顺了、该定「接下来怎么活」时，大冰给出三条次第：**① 活在过去加强自我认定**；**② 服务众生重建价值**；**③ 翻篇放下昨日之你**（推荐）。

## 详细说明

连麦者三硕士、工作平稳后情绪爆发，问「接下来该怎么活」：

1. **第一条**：让一部分自己活在过去，从外界获得认定认可，「一部分塞回过去」——可玩到生命终结。
2. **第二条**：做服务社会、利益众生、帮助他人，「重新建筑你的价值」。
3. **第三条**（稍高级）：「忘记之前好的坏的……昨日之你非今日之你……咋活都行，咋活都得活。」

建议：若心理咨询推第二条，可搭配第一条；大冰**希望选第三条**。
