# 弹幕借势化解资格攻击

## 一句话定义

直播间出现攻击性弹幕质疑自己「没资格帮对方」时，**主动念出并反讽该弹幕**，化攻击为话题，降低对抗后继续对话。

## 详细说明

弹幕：「大冰学历没你高，怎么可能帮你梳理这种问题」（对方是 985 硕士，大冰自称二本）。

大冰操作：

1. 念出留言和用户名「临界得星空」
2. 「拒绝了连麦申请，所以我只能陪你聊一聊好不好」
3. 反讽：「只有学历比别人高才能够帮别人梳理问题，这是不可能的……说不定有道理」
4. 立刻转折：「身处其中的人容易脑子混乱，跳出来……咱们只是瞎说说一种可能性」

效果：不硬辩学历，用幽默化解；同时暗示「跳出来看」才是价值，而非学历。
