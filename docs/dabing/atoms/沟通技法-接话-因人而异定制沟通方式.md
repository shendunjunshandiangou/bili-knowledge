# 因人而异定制沟通方式

## 一句话定义

对不同的人使用不同的沟通策略与姿态——不是「看人下菜碟」式的虚伪，而是**因地制宜、避险且有效**的一对一应对。

## 详细说明

大冰总结：「我在这个世界上接触的每一个人，我对他都有专属定制的一套方式。」

- 有知识、有修养的人：可以直接沟通，甚至指出错误
- 没文化、脾气暴躁、心理有问题的人：同样方法可能导致对方「崩掉」，把自己置于不安全位置

比喻：吃炒菜用筷子、喝汤用勺、吃螺丝用牙签——**不是一套器皿打天下**。

操作：每次接触具体的人之前，脑子里快速评估「该用什么姿态」，先立于安全盘、做避险；练到成为下意识反应。实践是检验真理的唯一标准。
