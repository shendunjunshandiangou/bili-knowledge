# 越对抗越加强

## 一句话定义

亲密关系小摩擦时，**越直接对抗越是在加强对方的气**；从热战变冷战，气撒不完，小事易放大。

## 详细说明

大冰对新婚夫妻：

- 女方一点就炸，男方从对抗→冷战——「你的应对措施不是安抚，而是对称冷暴力」
- 「你越对抗什么，越是在加强什么」
- 初期建议：小事「嬉皮笑脸骂过去」；后信息升级（扇脸）则另当别论
