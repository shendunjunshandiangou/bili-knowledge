# 一点点再坚持法

## 一句话定义

意志力低谷时**不要承诺长期目标**，只坚持「一点点」——三天、一个月、这一票难关——用微小正反馈当本钱续命。

## 详细说明

大冰对励志样本连麦者：

- 意志力「反人性」；普通人能量有限，「甚至都得死、能活一天多白赚一天」。
- 标的太远时：「那我就先坚持一点点就好……先一点点的坚持，实际上正反馈就已经有了。」
- 比喻投资：「总得有点本钱才能够去搏下一个一点点」；回头一看「好像我没有放弃的动作无限被拉长了」。
- 与项目管理同构：切小任务、完成一阶段一正反馈。
