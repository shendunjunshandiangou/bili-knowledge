# 知道就知道不知道就不知道

## 一句话定义

验证对方资质时，**拒绝接受模糊辩解**，要求明确知道或承认不知道。

## 详细说明

- 「知道就是知道，不知道就是不知道，好不好」
- 对方反复「多维度」「主攻人格」等辩解→「不知道是吧？你还回……」
- 不陪跑：无法通过基础验证则停止深入对话
