# 基础题验证对方高度

## 一句话定义

当对方自称某领域高手时，用**该领域最基础常识题**快速验证，判断真实水平。

## 详细说明

- 对方称研究哲学→「验证康德对人性的认知」
- 标准：「人性是复杂且不可论证的」——哲学系大一都知道
- 目的：「通过简单几个小问题，我得知道您处于什么样的高度」
- 答不出则判定：不专业，无法通过验证
