# 听三分知十二分主动梳理

## 一句话定义

当对方情绪叙述的模式已经清晰时，**主动打断冗长讲述**，声明「你说了三分我就知道十二分了」，提出由倾听者代为梳理，征得同意后再给结论——避免让对方在重复细节中再次崩溃。

## 详细说明

连麦者讲起去妈妈饭店、见妈妈变老、想孝敬等，情绪再次上来。大冰的处理：

1. **截断**：「你说了三分，我就知道十二分了」
2. **征得许可**：「我直接帮你梳理，要不要听？」「要要」
3. **不再要求讲完**：「你不用接着再往下说了」
4. **直接给梳理结论**：孝敬妈妈与前面「帮父母」是同一题——看真实实力，不是当下挤压；刚入职没有多少油水，等几年有硬实力再帮，「妈妈等得及」

要点：

- 前提是对话模式已重复出现（此前已讨论「帮父母 vs 先对自己好」）
- 打断不是冷漠，而是**减少二次创伤式复述**
- 梳理后接**具体比喻**（三岁搬煤气罐）巩固结论
