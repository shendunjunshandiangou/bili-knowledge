# 沙盘推演前先征得同意

## 一句话定义

在给当事人做分析或「推演」之前，先声明**可能不对、且不是替对方解决问题**，降低被说教感，换取倾听许可。

## 详细说明

大冰原话结构：「我大胆做一个沙盘推演，如果您愿意听的话，我帮您分析一下，**但一定不是解决您的问题好吗**。」

作用：

- 降低连麦者的防御（避免「你又来教我」）
- 把后续分析框定为**视角提供**，而非权威判决
- 为「我不认同你的自归因」做铺垫（如不同意「面试更软」的自评）
