# 深入链接是软肋

## 一句话定义

大冰自认软肋：**不可与人发生太深入的链接**，个体投入过多会导致投射、情绪外化不成熟、丧失直播间式的自我管理。

## 详细说明

自我披露要点：

- 以前不吝啬深链接，后来发现包括今天也无法完全驾驭
- 建立稍深链接 → 投射 → 动作/外化表露不成熟、情绪控制差
- 例：甘孜共同相处认朋友、喝三斤半青稞后对年轻人悲观预判；对大梦「梦见你交不起房租必须打钱」
