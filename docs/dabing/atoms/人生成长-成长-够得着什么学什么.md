# 够得着什么学什么

## 一句话定义

自我投资学啥**不取决于哪个最热**，而取决于视野内**够得着、碰得着什么**——亲戚人脉、资源辐射范围内，碰着啥学啥，别挑花眼。

## 详细说明

- 芯片、期货等不在选项内就不琢磨
- 先沙盘推演**已有能力**（否则现职干不了），再谈该长什么
- 大冰无法给具象「学啥最好」——只给方法论
