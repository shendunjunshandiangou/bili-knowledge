# 帮连麦者限时现场推销

## 一句话定义

在公开连麦中主动让诉求者利用当前流量窗口完成推销，并引导其自夸产品、说明卖点。

## 详细说明

大冰不仅同意打广告，还控场推进：报在线人数、催「抓紧做好」「开始吧」，并教连麦者「你得主动点夸一夸自己的黄瓜多好吃」，针对南方观众提醒「可能不吃这个东西，得好好推销」。

适用：帮助资源弱者完成**一次性的公开曝光动作**，而非代其卖货。
