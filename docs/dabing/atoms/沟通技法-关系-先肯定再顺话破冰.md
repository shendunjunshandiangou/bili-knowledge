# 先肯定再顺话破冰

## 一句话定义

无利益冲突时，**先肯定对方**（「您说得非常好」）→ 不露异见、顺着说 → 用 2–3 分钟无关闲聊**打破距离**，换取正式议题的空间——标准职场对称反应。

## 详细说明

- 呼啦圈环节：连麦者先肯定大冰/斌哥，再顺话聊 2–3 分钟
- 正向效果：人际关系距离打破 → 长自我介绍获更多空间与可能性
- 本质是**交换动作**；年轻人社会面上难直接学到
- 大冰：「在什么节点说什么话、有没有眼力见——都是有效积累」
