# 功夫在戏外求审美

## 一句话定义

艺术专业除技巧训练外，须长期投资戏外审美——看画展、读能折射人性/艺术向的美文，提升理解力。

## 详细说明

大冰通过听连麦环境判断在琴房，给钢琴生梳理：「功夫要在戏外求」；好画家音乐鉴赏更高、好写作者文字有画面感；建议搞明白洛可可、巴洛克等，「对于音乐理解一定有帮助」。审美「无比重要」，需长期主义，「技术赛道太拥挤」。
