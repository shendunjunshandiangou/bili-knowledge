# 抑郁症药疗心理干预并行

## 一句话定义

抑郁症恢复需**开方医生+适合的心理干预师**双管齐下；陪伴是被动性的，不能替代专业干预。

## 详细说明

- 「光吃药不行，需要心理疏导/心理干预老师和心理治疗师双管齐下」
- 咨询师要投缘：「不见得碰见每个人都该当你的师傅」；1000元/小时的不便宜，该继续找
- 推荐方向：搜「萨琳娜」女性咨询师（哥伦比亚国际心理学硕士），机构名不便在直播间报
- 对陪伴者：「身旁这个人对你最正向的作用是陪伴，但是被动性的，不能主要依赖」
- 好心理师会叫两人一起，给出阶段性方法论；「一家医院治不好就转院」
