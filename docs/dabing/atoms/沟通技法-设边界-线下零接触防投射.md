# 线下零接触防投射

## 一句话定义

自知建立深入链接会**大量投射、情绪失控**时，立规矩：**线上链接则线下绝对不见**——物理距离是基本保护。

## 详细说明

大冰自述软肋：深入链接后投射到对方，外化不成熟、难控情绪（甘孜喝青稞后对「朋友」弟弟的悲观预判；对大梦梦到交不起房租必打钱等）。

开直播后规矩：麦子阿姨、渐冻症男孩等所有发生链接者，**线下绝对不见**。

逻辑：「我知道我自己管不好我自己，最基本的保护动作就是不接触线下，咱们绝对不接触。」
