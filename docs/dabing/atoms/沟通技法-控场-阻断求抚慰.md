# 阻断求抚慰

## 一句话定义

大冰**一贯不做**情感抚慰；连麦者求温度时给客观梳理，并**迅速击破**其高期待——未来进入社会此模式会无数次重演。

## 详细说明

- 总结连麦状态：发起议题主要为获得抚慰；发现无温度梳理后试图掰话题
- 「我是绝对不会来承载你任何的高期待，不仅不承载，每当你有了之后我迅速给你击破」
- 今天算「比较温柔的一种态度」——预警职场同类场景
