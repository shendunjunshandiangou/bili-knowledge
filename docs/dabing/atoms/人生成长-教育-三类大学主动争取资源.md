# 三类大学主动争取资源

## 一句话定义

中国大学分**研究型（985，研>本）、教研型（研略低于本）、教学型（研远少于本）**；后两类本科更须**主动争取**导师与科研机会。

## 详细说明

| 类型 | 特征 | 本科策略 |
| --- | --- | --- |
| 研究型 | 985，研究生招生>本科生 | 资源相对多，仍须主动 |
| 教研型 | 如郑州大学，研略低于本 | 更须主动跟老师做科研 |
| 教学型 | 硕博点少 | 主动争取空间更大但资源少 |

- 脱颖而出：**自律能力强** + **主动争取资源**的人占优势。
- 「特别聪明的人一般不自律，自律蛮难的。」
