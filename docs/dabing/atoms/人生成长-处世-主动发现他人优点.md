# 主动发现他人优点

## 一句话定义

人际关系中稀缺的是「有需求才认可」——应尽量**发现对方值得认可的点并表达**；去功利化也是极好**保护色**，酒桌上更促单。

## 详细说明

- 连麦者婉转夸斌哥保留天真 → 解读为感谢；好话差几杯酒，酒桌管用
- 「比较稀缺的是有需求时才表达认可，实际上要尽量多发现值得认可的点」
- 可通过直播间教年轻人 → 少走弯路
