# KR全称验收是否真学

## 一句话定义

用**缩写全称**追问验收新人是否真学——答不上来说明把工作当「屁事儿」，没当提升自己或谈资。

## 详细说明

- 领导让学 OKR/KR 文档，连麦者学了但未深问
- 大冰：「KR 的全称是什么？……不能解释就说明你没有好好学」
- 原因：「你觉得那是领导交给你的个屁事儿，不是能够提升自己的角度」
- 若把工作当回事：「跟大领导、行业里的人聊起来算谈资；以后管理员工侃侃而谈，人家觉得不好骗」
- 「跟人说 KR 啥就年度计划吧——一听就是啥也不懂的人」
