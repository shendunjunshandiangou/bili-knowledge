# 主动提建议者必是是非人

## 一句话定义

扩大古语「来说是非者必是是非人」——在对方**未主动发问**时，主动披「好心」外衣提建议者（尤其涉私人议题），**必是是非人**；多为刷存在感。

## 详细说明

- 事业单位同事闲磨嘴皮子，拿结婚生孩子当唯一可称道之物，扮演人生导师
- 类比：家长扎堆支招别人家孩子——只为证明「我家养得比你们强」
- 价值感低、渠道少 → 逮机会指导他人
- 与 [人生成长-处世-单位勿聊私事](/dabing/atoms/人生成长-处世-单位勿聊私事.html) 配套
