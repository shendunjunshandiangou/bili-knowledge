# 帅哥光环加戏投射

## 一句话定义

点破社会教育塑造的「长得好=好人」：对帅哥会**加戏**脑补古偶/言情男主式优良品质——下次若被外在吸引，多留理性和理智。

## 详细说明

大冰判断前男友「长得还有点小帅」，女生在关系中「精神意义上的下跪」、跪姿挽回。

- 「长得帅的男生……我们大部分人都会犯这个毛病」
- 潜意识：相貌堂堂=好人；基于外在帅「给自己加戏，脑补出来很多优良品质」
- 社会教育海量信息塑造的察人方法论「是歪的东西」
- 建议：「如果一开始是外在吸引……给自己多留一点理性和理智好不好」
