# 反向提问听差异

## 一句话定义

对方从切片/二手内容来，先问：**「和真实直播一样吗？差异化在哪？」**——听感受再汇报，而非先辩解。

## 详细说明

B 站观众连麦，大冰开场：

- 「你在 B 站看到的切片，和我们真实连麦一样吗？」
- 「说一下差异化在哪里」

听众答：切片片面；直播间更有参与感、更真实生动、像朋友聊天、有疗愈功能。

大冰随后才做摩旅状态、没耐心标签等汇报——先收集差异，再解释。
