# 先给结论再补理由

## 一句话定义

拒绝或表态时**先给结论**（「不行」），再补理由；解释=授权对方根据理由发起质疑。

## 详细说明

- 「先不要找解释借口，先给一个结论——不行不行不行，然后你可以再补充你的理由」
- 「只要你解释，就等于是授权了对方来根据你的解释发起质疑」
- 进阶目标：既不需要肢体语言辅助，也不需要解释，「哎呀不行就好了」
- 可加「实在抱歉」维系基础友好，但结论不变
