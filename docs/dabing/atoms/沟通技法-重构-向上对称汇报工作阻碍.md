# 向上对称汇报工作阻碍

## 一句话定义

向主管汇报时，在表达负责态度前提下，**对称造成工作开展不畅的具体阻碍因素**——前提是告知领导「不是添麻烦，是为工作不出疏漏」。

## 详细说明

- 主持工作、掌握向上汇报权，「干嘛要浪费了这些机会呢？」
- 汇报结构：积极表达意愿与认真负责态度 → 对称卡点因素 → 本位前提：不是给领导添麻烦，是为工作不出疏漏；科室工作也是上级可被观测的内容。
- 大冰提醒：「你不去做，别人会去做」——不配合的同事可能先去向领导汇报，让你被动。
