# 菜市场早市复位

## 一句话定义

对焦虑/抑郁/人群恐惧者，建议赶**早市/菜市场**——食物琳琅满目刺激生存本能，摊主生命力可感染，是对死亡恐惧的底层抚慰。

## 详细说明

大冰对害怕人群的藤校博士：

- 「你下一回再碰见这样的场景……笑嘻嘻的往里边挤」
- 「赶着清晨的早市……连着给我逛个三天五天……为一把韭菜讨价还价半天」
- 「菜市场可治愈人了……充满了勃勃的生机」
- 逻辑：恐惧归根到底是对死亡的恐惧；食物刺激直接做「自我复位」
- 东北早市/南方菜市场两极端均可
