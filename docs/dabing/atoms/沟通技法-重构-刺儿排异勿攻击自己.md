# 刺儿排异勿攻击自己

## 一句话定义

失恋如**肉里扎刺**——拔刺排异对方，「这块肉没有错」；勿同归于尽式攻击自己。

## 详细说明

- 「让他意识本身排一下，意肉里边有刺儿就把它拔出来——是刺儿扎了你的肉，这块肉没有错。」
- 连麦者不舍得对方死，但要排异意识层面，非物理伤害。
- 若坚信「拔不出来，谁都帮不了你」——需保留可拔的信念。
