# 轻资产线上优于线下小饭馆

## 一句话定义

中年人失业拿积蓄创业，**第一层想到的小饭馆往往最赔**；互联网平台开号是轻资产、低投入、拼运气，即便干不成也不伤筋动骨。

## 详细说明

大冰在直播间三个月的经验：

- 人到中年失业、有点积蓄想开小饭馆 → 一定拦住
- 「你以前干过吗？」「没干过凭什么干成」
- 「一般人创业第一层想到就小饭馆，你不赔谁赔」
- 建议琢磨线上：轻资产、低投入、拼运气，相对规避现实生活创业风险
