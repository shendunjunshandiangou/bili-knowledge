# 悲悯不必痛心更不必实操

## 一句话定义

对年轻人自杀等新闻可以悲悯，但不必痛心到要拿自己经历去「治」别人——起了善念已有功德，实操走向判断不了。

## 详细说明

教授见年轻人轻生很痛心，想提供「病例」帮大冰治别人。大冰：「谁都帮不了，每个人有各自的因果业力」；「貌似帮人家，实际上帮自己」——可以悲悯，「大可不必痛心以及基于痛心去做说服指导动作」，因为「判断不好真实走向和药用价值」。「起了这个念动了这个心就已经很有功德了，未必一定要实操」。
