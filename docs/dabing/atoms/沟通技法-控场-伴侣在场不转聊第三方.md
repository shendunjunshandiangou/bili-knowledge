# 伴侣在场不转聊第三方

## 一句话定义

连麦对象明确为某当事人时，即使其伴侣在场要求对话，也**坚持只服务当事人**，不转开新对话线。

## 详细说明

- 男友被吵醒要说话：「没关系……我可以让他——什么没必要，我跟你聊来聊就行，我没必要跟他聊」
- 后续对男友单独开段时，仍锚定「我主要服务你女朋友」，只要求一个「坚定理由」来消除质疑
- 避免话题被第三方带偏、立场被稀释
