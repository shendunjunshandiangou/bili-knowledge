# 需求决定关系非父兄

## 一句话定义

把领导当父兄、凡事请教是**职场大忌**——真实世界是**需求决定关系**；对你好必有可回馈理由，无需求则关系不复存。

## 详细说明

- 连麦者慕强、认前老大如父兄，忍不住请教
- 大冰：「需求决定关系……他之所以对你好，一定有你可以回馈给他的具体理由，即便不是当下兑现」
- 前老大不回信息：棋子验证「屁用没有」，且触碰一把手真实立场
- 父位缺失→对高位者潜在恐惧→讨好行为（非单纯慕强）
