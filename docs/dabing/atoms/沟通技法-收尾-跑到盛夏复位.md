# 跑到盛夏复位

## 一句话定义

对长期躺平者，给**物理劳动+时间节点**（跑到盛夏最热时）作为状态恢复第一步。

## 详细说明

- 注册众包跑外卖，从现在跑到「盛夏时分——蚊子满天飞、一身汗」
- 「只要坚持到那一天，状态就完全回来了」
- 心态预设：「每天最起码碰壁三次」——接受不友善与心理折磨
