# 不承担他人因果

## 一句话定义

对不认识的连麦者，开场即声明：**不保护任何人、不承担别人的因果**——把帮助边界说清楚。

## 详细说明

自称被针对、能预知未来的小杨请求「保护」，大冰回应：

- 「不保护任何人，我不认识你，我不承担别人的因果的哦」

适用于无法核实身份与诉求、且可能将责任转嫁给主播的场景。
