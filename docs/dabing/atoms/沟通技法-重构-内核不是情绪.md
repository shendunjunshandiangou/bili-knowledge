# 内核不是情绪

## 一句话定义

应对冒犯时，**内核是原则**（如容差率、是否接受冒犯），不是生气/无语等情绪百分比。

## 详细说明

- 「你的内核情绪？不是我的内核——把情绪当内核是大部分人解决不了矛盾的主因」
- 示例内核：「不主动冒犯他人，也不接受他人轻易冒犯」或「有一定容差率」
- 反击力度须**忠实于内核**，否则跟着情绪走=凭感觉决定
