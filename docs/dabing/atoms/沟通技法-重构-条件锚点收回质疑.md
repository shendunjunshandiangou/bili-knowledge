# 条件锚点收回质疑

## 一句话定义

对第三方提出尖锐质疑后，设一个**可执行的交换条件**——对方满足则**全部收回**质疑话术，保留建设性叮嘱。

## 详细说明

- 大冰对男友质疑清迈民宿方案后：「收回刚才所有的话，您不必冤枉——锚点：只要你们不去清迈开民宿，刚才所有的话都收回」
- 男友答应后，叮嘱内容不变：督促女友找专业心理干预、药疗+心理干预并行
- 作用：给关系留台阶，把冲突从「人身质疑」转为「行为选择」
