# 表白一次勿死缠

## 一句话定义

追求时**表白一次即可**——对方意愿会直接告知；**勿死缠烂打**，基本尊重对方个体主观意愿。

## 详细说明

- 「你好，尽量不要做死缠烂打的事……基本的尊重，尊重对方的个体的主观的意愿。」
- 与「勿地铁要微信」配套：勿把女性当练手、勿用男性思维套女性。
- 「可以不当好人，但是不能够当小流氓。」
