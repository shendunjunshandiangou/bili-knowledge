# 开场尽量说点有用的

## 一句话定义

连麦接通后先定调：「尽量连上麦说点有用的」——避免无用开场（如甩锅预警）直接改变沟通走向与真实链接。

## 详细说明

抬走连麦者后，大冰向直播间复盘：

- 大部分人沟通时没意识到「多说少说一句无所谓」
- 但无用话（如预先甩敏感风险）会导致**事情走向**和**真实链接方向**转变
- 故连麦前先提醒：尽量说点有用的
