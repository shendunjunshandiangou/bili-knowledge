# 德服不够需配合立威

## 一句话定义

事业单位打开局面，**只靠德服人往往不够**；立威后对方仍不配合，即工作态度问题，矛盾可公开化，自己须零疏漏。

## 详细说明

- 大冰：「有时候往往你真正想打开局面，只靠德服人不够的。」
- 流程：收拾/扳他一次 → 若继续不配合 → 「这就是你的工作态度的问题了；矛盾都已经摆到桌面上公开化了，主管领导也都知道了。」
- 此时策略：自己尽量保全不出疏漏 + 工作积极推进，需配合事项对方仍不配合则立场清晰。
- 一切动作为服务好人民群众（弱势群体），匹配动作不出格。
