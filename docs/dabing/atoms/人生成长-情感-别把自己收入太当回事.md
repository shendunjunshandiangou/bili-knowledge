# 别把自己收入太当回事

## 一句话定义

嫌弃稳定、低野心伴侣时，用**本地婚恋市场**校准：此类姑娘「有的是人要」——别因自己刚涨薪就高估议价权。

## 详细说明

- 女友专科辅导员，年入约 6～7 万；男博士年入 20～30 万
- 大冰：「在咱们山东这类姑娘，在青岛远远可以找到嫁得比你更好的老公」
- 「这样的女孩子……青岛本地做生意有小成的人，大把等着娶」——「我给你制造点紧张感，你要不要，有的是人要」
- 理想画像：「开个小厂子年收入你四五倍……巴不得娶有份稳定工作就行，回家高高兴兴打游戏，别打麻将度脸整得和猴似的」
- 「你还真别把自己现在的收入太当回事」
