# 考试是模拟训练场

## 一句话定义

中高考前所有考试都是「吃鸡训练场」——测成果+找不足；已考完的分数是过去式，应关注下一场与错题。

## 详细说明

十岁男孩数学96后84、83焦虑。大冰：考试一半功能是发现不足；「7年半以后18岁高考可能是最重要一场，此前都是模拟训练」；「已经结束了就像前天那顿饭，不重要了」；应关注下一场和「错的题下回还会不会再错」。
