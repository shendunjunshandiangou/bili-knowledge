# 大恩即大仇

## 一句话定义

职场中对他人窘迫的洞悉与施恩，即便善意也会给对方巨大心理压力；**欠人情者易把你放对立面**——「N记仇N就是仇」。

## 详细说明

- 流行语精简：「N记仇N就是仇」
- 大部分人不愿尴尬时刻被洞悉，尤其职场；被施恩=被帮助一方心理压力大
- 领导可能解读成「你在可怜我」「情感投资雪中送炭」
- 你人好只能证明你人好，别指望真心换真心
- **本集补充**：帮领导打考勤掩护、全盘接活 → 见证窘迫是职场大忌；大概率是你**先上赶子**；粗俗比喻——帮换「沾屎裤子」后领导正常反应是希望你远离；「你是来上班的，不是来交朋友/走亲戚」
