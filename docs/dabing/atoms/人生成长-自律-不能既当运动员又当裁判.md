# 不能既当运动员又当裁判

## 一句话定义

自控奖罚机制要生效，**不能自己监督自己**——住宿舍请同学帮，在家请家人帮。

## 详细说明

大冰原话：「因为咱不能既当运动员又当裁判。」

操作：请同学或家人监督奖罚执行；奖罚规则需清晰（罚：没完成饿两顿；奖：学完才配看手机）。
