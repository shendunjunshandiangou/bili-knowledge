# 换城市注入可能性

## 一句话定义

在**陈旧存量环境**内躺平者已知无任何可能性（含婚恋）——**更换城市**代表可能性诞生；建议长三角等讲规则城市，慎选高海拔/过度下沉地域。

## 详细说明

- 秦皇岛 → 建议上海等长三角：更秩序化、讲规则，对教条母子未必错
- 拉萨：体重 280 斤有肺水肿风险；越下沉规则弹性越大
