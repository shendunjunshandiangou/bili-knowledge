# 纠正空泛大词让说人话

## 一句话定义

连麦者满口「基层」「意识形态高」「教育局限」等**大词套话**时，要求换成可听懂的具体表述，并警告大词会耽误本人。

## 详细说明

对大冰15岁初三男生：

- 「什么叫基层、什么叫意识形态高——这个听不懂」
- 建议：「你可以说我是过着很平凡的生活，父母收入很一般，尽量别有什么基不基层」
- 第二次提醒：「别学一堆貌似有道理的大词，很容易耽误你自己……25岁得跑飞机了」
