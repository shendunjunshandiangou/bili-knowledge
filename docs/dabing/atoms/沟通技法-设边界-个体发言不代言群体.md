# 个体发言不代言群体

## 一句话定义

当对方以「大多数网友」「大家都……」等集体名义施压时，温和指出其未必代表数据结论，请其**只代表自己**发言——「您代表您自己就行，因为如果您代表您自己的话，您说得就对。」

## 详细说明

连麦者称大冰的书「毫无内容」，并补充「是大多数网友的这个意见」。大冰回应：

> 「这个大多数这三个字的话，大概率不是您缜密的数据整理过之后得出的结论。所以我觉着，如果您以个体的身份来跟另外一个个体来对称相关信息的话，您代表您自己就行。因为如果您代表您自己的话，您说得就对。」

**与「你感觉框架」的配合**：

- 集体名义 → 打回个体身份
- 个体主观 → 用「你感觉」框架认同

二者共同避免陷入「你的书到底有没有价值」的内容辩论。
