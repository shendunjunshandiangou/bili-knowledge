# 快乐而糊涂的童年

## 一句话定义

经济过得去、家庭和睦时，尽量给孩子**「快乐而糊涂」**的正常童年——放弃天才幻想，切忌拔苗助长与早会式特长浇水。

## 详细说明

大冰建议：

- 「让孩子拥有一个快乐而糊涂的童年」
- 「大部分家长放弃孩子是个天才的想法，当正常孩子正常养育」
- 「切忌一切拔苗助长、违背常识及正常生长逻辑的动作」
- 舞台感偶尔体验即可，长期注意力偏移会快速心理落差
- 天真语言、不知道就不知道——「孩子就得有孩子的样」
