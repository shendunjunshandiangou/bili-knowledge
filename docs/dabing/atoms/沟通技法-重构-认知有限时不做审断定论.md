# 认知有限时不做审断定论

## 一句话定义

可以承认认知有限，但在**信息不足时仍做笃定结论**不科学——需点破「有限认知下的审视定论」问题。

## 详细说明

男生对艺考、教育模式发表笃定意见，大冰：

- 「你的认知有限……你才有很笃定的言论的话，请注意这两个议题你都是在目前已承认认知有限，但是在认知有限的情况之下很笃定的做出了一定量的审视之后的定论，它科学吗」
- 信息不清楚时（如艺术院校学费）：「您省省你的意见啊」
