# 晒太阳跑步节阳气

## 一句话定义

心情低落、焦虑到动不了时，**先从生理调节**：固定早起跟太阳跑步流汗，再谈心理。

## 详细说明

- 「管你悲不悲观，第二天早上06:35起床，跟东边升起来的太阳一起跑，晒晒太阳流流汗」
- 「先从生理上调节——节阳气；回来洗澡大吃一顿」
- 最浅表的心态转变方式，先于复杂心理分析
