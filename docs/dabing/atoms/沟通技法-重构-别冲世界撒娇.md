# 别冲世界撒娇

## 一句话定义

面对 offer 变动等无常，成熟是接受变化、自我调整，而非要求世界按你的意志配合——「我们都不是地球中心」。

## 详细说明

连麦者因工作调动失衡。大冰：「别冲世界撒娇」；「真实的世界不以我们的意志为轴心给我们匹配资源」；不成熟常表现为「对世界撒娇——你为什么变，为什么不能变化」；要「接受变化、配合变化、自我发生变化」指向成熟。真自我面对小捉弄会「一切都是最好的安排」。
