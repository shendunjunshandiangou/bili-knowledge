# 12345 清单后重排责任序

## 一句话定义

在对方用清单自陈责任后，**指出排序遗漏**，把被挤到后面的项（如「好好活着」「对自己好」）提升为**第一项**，并追问「以你目前状态，第一项能完成吗？」——从根上动摇「牺牲自我=尽孝」的真理。

## 详细说明

完整序列：

1. **预告动摇**：「我要尝试动摇一下你的真理」
2. **12345 清单**（见 [沟通技法-提问-12345清单式澄清](/dabing/atoms/沟通技法-提问-12345清单式澄清.html)）
3. **重排**：「你对他们应承担的第一项天然的职责，叫做你好好活着……探讨一切议题，先不要着急说是我想怎样为他人」
4. **点破模式**：「您的排序里边看不到这个我字……通过压制自我，然后来做一个奉献及牺牲——这个是有严重问题的」
5. **一句话回收**：「你应对他们尽的第一项责任是啥？」→ 对方答「好好活着」
6. **状态自检**：「以你目前的这个状态，你觉着对他们应尽的责任的第一项，可以很好完成吗？」

后续展开（同一重构线）：

- 父亲债务：法律底线是将来赡养，之上每多一分是情分；「你爹有你爹的议题，他欠的债不是你欠的」
- 弟弟网贷：「第一次就不该帮，若帮了第一次就没有第二次，有一有二就无穷尽一」
- 孝的前置：「父慈子孝」——父慈在先，不是无条件当血包

与「只劝休息」的区别：不是症状管理，而是**改责任框架**。
