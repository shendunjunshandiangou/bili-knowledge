# 游牧思维替代农耕思维

## 一句话定义

面对行政前台等普通岗位，勿用**农耕思维**守一块地（必须长庄稼、患得患失）——应换**游牧思维**：逐水草而居，草吃完了就转场，「此处不留爷」找丰美之地。

## 详细说明

- 行政前台非强技术岗，更不宜锁死一家公司干 15–30 年
- 公司 5–10 年后能否存活都是问题——随时做好转场准备
