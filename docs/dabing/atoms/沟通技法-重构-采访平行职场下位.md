# 采访平行职场下位

## 一句话定义

同一人在**采访场景**（平行/略高位→安全）与**职场**（明确下位→畏缩）的关系位置不同，导致「跟采访对象能聊、跟同事领导不能」的分化。

## 详细说明

- 采访对象：记者身份下「最次是平行关系，面对老百姓 often 高一点点」→ 心理安全，沟通无碍。
- 职场：领导、资深同事在各维度指向**下位者**→ 第一层感知畏缩恐惧，担心做不好。
- 改良方向：假装扮演强者、向外模仿；「认为自己是一个强者，起码大家都是」。
- 童年缺健康成年男性榜样可模仿，「现在对标补上不算晚」。
