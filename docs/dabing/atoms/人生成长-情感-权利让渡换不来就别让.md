# 权利让渡换不来就别让

## 一句话定义

恋爱中若已到仰其鼻息、察言观色、对方一举一动左右喜怒哀乐，且**权利让渡没换回想要的**——让渡本身已出问题；不是献祭，「我就是我，你是你」。

## 详细说明

- 怕分手=「我字没有了、权利最大化让渡」
- 大冰：「谈恋爱……目的是共同成长、经历美好时光，不是把微薄的自我完全投射给你，让你决定我的人生，永远扮演跪姿」
- 投射过多：「不是投入，是投射太多……all in 交给他，命运裁决权完全在人家那边」
- 「脸给多了……你把权力也给他了……培养了一个侵略军」
- 老百姓总结：「有点逼数」——对自己、对对方
- 区分：自我缺失≠主体性缺失（主体性指情绪观测掌控能力）
