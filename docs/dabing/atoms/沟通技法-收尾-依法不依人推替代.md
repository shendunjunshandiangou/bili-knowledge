# 依法不依人推替代

## 一句话定义

当听众表达离不开某一人时，用「**依法不依人**」「一鲸落万物生」引导其寻找替代，降低对个体的迷信依赖。

## 详细说明

大冰对粉丝焦虑的收尾策略：

- 「世界离了谁都转」——一定找得到替代直播间
- 「依法不依人」——过于迷信、相信单一的人有逻辑问题
- 「一鲸落万物生」——可能恰恰因他挡道，观众没接触更多优秀主播和更新鲜观念
- 同时承诺：合适时间该播还会播，话没说死

既安抚（有替代、可能还会回来），又推独立（勿把人生链接绑在一个人身上）。
