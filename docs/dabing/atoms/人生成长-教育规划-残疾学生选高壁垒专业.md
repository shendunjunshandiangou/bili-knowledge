# 残疾学生选高壁垒专业

## 一句话定义

身体条件有限的学生报志愿，应优先选**专业壁垒高**的方向，避免低壁垒、易被同等条件下优先淘汰的岗位。

## 详细说明

张雪峰在汇报烟台残疾女孩帮扶时补充报志愿建议：

- 弄清「残疾的点在哪里」，结合具体情况选专业。
- 「报一些高壁垒的专业，不要报那些低壁垒的专业。」
- 不能道德绑架社会或企业；同等条件下，低壁垒岗位可能更倾向身体健全者。
- 应选「只要我努力学习、找我能干会有更好效果，找别人干就不会有这样效果」的专业——用技术能力形成不可替代性。

同时提到：直播间有身体残疾孩子的家庭，公司会在学费、就业等方面帮扶到位；烟台女孩求学维度也会持续帮助。
