# 生活化追问建罚则

## 一句话定义

对方有抽象规划却无执行约束时，用**具体生活问题链**（吃早饭吗→没完成就饿两顿）当场建惩罚规则。

## 详细说明

连麦者：有时间表但无奖惩，规划无用。

大冰追问：

- 「抽烟吗？不抽。」
- 「吃早饭吗？吃。」
- 「头一天没完成，第二天没资格吃早饭，能不能做到？」

从日常习惯切入，惩罚可感知、可执行；再补「还要设奖」「坚持的是奖罚机制不是表」。
