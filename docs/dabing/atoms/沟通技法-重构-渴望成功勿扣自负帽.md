# 渴望成功勿扣自负帽

## 一句话定义

内心**渴望成功**不是错，但勿给自己扣「自负」帽——HR 说「有傲气」常是点你**姿态偏高需谦卑**，非真自负。

## 详细说明

- 连麦者自负自卑交替；大冰：「不重要……不是下一个阶段的状态。」
- 面试官说傲气：「语态措辞礼貌，姿态可能有点高——点你稍微恭敬谦卑一点点。」
- 「你并不是一个自负的人，而是内心有一个你很希望成为别人认为挺成功的人。」
- 可说「我渴望成功」——比「我是自负的人」少心理暗示、少自我攻击。
