# 娶妻非雇保洁

## 一句话定义

「你是娶个媳妇，不是说娶回来必须给你当保姆或保洁」——家务谁有精力谁干，收入够则可花钱买服务，不必把卫生问题上升为原则性分手理由。

## 详细说明

- 博士因女友不收拾、打游戏纠结；大冰：年收入够请钟点工，「家里边乱你可以花钱解决此类问题」
- 女方合租多年，终于有小家，「放松放松玩一玩没有什么大过」
