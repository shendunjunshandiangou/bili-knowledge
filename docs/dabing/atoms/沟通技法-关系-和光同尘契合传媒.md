# 和光同尘契合传媒

## 一句话定义

明知不必迎合大众传媒规则，外化姿态仍须**多多少少契合**一部分——「和光同尘」，虽无聊但可避免不必要的麻烦。

## 详细说明

大冰谈「没耐心」标签：

- 「大众传媒有潜在游戏规则，即便不受捆绑，也不希望受捆绑」
- 外化表露的自我管理样貌要契合一部分规则
- 「和光同尘，很无聊……但没办法，你要去契合一部分」
- 同时说明摩旅晒肿等真实处境，非单纯「装耐心」
