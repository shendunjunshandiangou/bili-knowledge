# 少说凝练短语训练

## 一句话定义

与权威/陌生人沟通紧张时，**少说、凝练成最短准确短句**，献丑不如藏拙。

## 详细说明

- 核心指令：「少说」——五小短句凝练成最短一句
- 「少点发表意见、少点剖析、少点对称个人感觉——越说越慌越乱」
- 现场训练：问完不马上答，用短语回应；「说话的时候少说——那不就很好吗」
