# 别信情商智商测试分数

## 一句话定义

用自黑与逻辑拆穿心理测验的量化幻觉——感情与情商不宜打分，测出来低分可能是卷子有问题。

## 详细说明

教授做卷子情商59、智商147，大冰：「相信个六」。反问感情能否量化到90几分；自曝认真做智商卷「70分都没到」，结论「写这套卷子的人智商有问题」。延伸：「宁可全世界都错了，不要觉得自己错了；最害怕的是自己为难自己」。
