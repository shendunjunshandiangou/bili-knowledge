# 客气易被读作示弱

## 一句话定义

职场里**过度客气**在许多人眼中是**示弱**——因「没有攻击性（加引号：自我、原则底线）」而低于众人预期，易沦为被排异对象。

## 详细说明

- 「客气在浅表是礼貌，但在很多人眼里……是一种示弱，因为你没有攻击性。」
- 攻击性「不是侵略性，是自我、原则底线」；人人需一定攻击性。
- 低于众人预期水平线→「很容易沦为职场末尾排序、被瞧不上、被打击或共同排异的人。」
- 与「分档设底线」配套：该直则直，该反弹则反弹。
