# 先澄清误解再展开

## 一句话定义

听众带着误读或焦虑提问时，**先拆解误解**（停播≠退网、不连麦≠不直播），再展开完整论述。

## 详细说明

粉丝连麦核心焦虑：「夏天后不直播了」「退网」。大冰回应顺序：

1. 「也没有完全说死」——否定绝对化理解
2. 「夏天之后不怎么连麦了，没有说是不直播」——层级澄清
3. 「有可能还开着直播播放摩旅、骑摩托车」——给出具体替代形态
4. 然后才展开 AI 预判、被看见价值、依法不依人等深层内容

若直接讲预判，听众仍卡在「你要跑了」；先澄清误解，情绪才能降下来听进去。
