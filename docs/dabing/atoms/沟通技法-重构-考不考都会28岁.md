# 考不考都会28岁

## 一句话定义

用**时间不可逆**推行动：「不管你考还是不考，你都会 28 岁」——无更好路径时，考试是成本低、风险低的选项。

## 详细说明

- 县城编制女想考回省会，焦虑 28 岁与催婚。
- 大冰：「你愿意给自己多一次机会也没有别的更好的道可走的话，考试对你而言成本最低、风险最低。」
- 「至于考完了你是不是 28 岁——你不考你也会 28 岁。」
- 只做基本逻辑处理，减少内耗式空想。
