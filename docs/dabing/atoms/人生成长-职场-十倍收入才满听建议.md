# 十倍收入才满听建议

## 一句话定义

过滤同事职场建议：**收入达你十倍**可满听；差不多或略多叫**竞品**，有利益相关更须岔开收入——刚上班别被扰乱心智。

## 详细说明

- 同事姐姐聊赛道建议，连麦者差点动摇
- 家人：「给你建议的人是不是公司小虾米？」——大冰：「你家里人比他们聪明」
- 「如果这些给你建议的人是你的十倍收入，你可以满听听；如果他还赚的没你多或者和你差不多，这叫竞品」
- 「多一点的不足为奇，不到十倍，他不配给你提建议；他跟你有利益相关」
- 刚上班就扰乱心智、丧失校招身份瞎试机会——「这不可能是朋友」
