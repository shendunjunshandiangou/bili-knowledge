# 揭穿关系中的加戏

## 一句话定义

当一方用「我是为你而来的，你得对我负责」绑定对方时，直接判定为「加戏」，并说明机会曾给过即可、接下来看自身胜任与否——**性别颠倒亦然**。

## 详细说明

女方男友称来深「一半因感情、一半觉得跟着她不会吃亏」；女方愿分一半存款兜底，反复说需负责。

大冰回应：

- 「你曾经提供过机会就可以了；接下来看他自己，胜任就胜任，不胜任不存在他说的话就完全正确」
- 「我是奔赴你而来的，你得对我负责任——不是的，加戏了，属于加戏了」
- 「即便你们的性别完全颠倒，都不是啊，加戏了」

并延伸：若事态不按预期（如无法待下去），连麦者需直面自己的**挫败感**是否那么恐怖——往往比「他能不能接受」更需处理。
