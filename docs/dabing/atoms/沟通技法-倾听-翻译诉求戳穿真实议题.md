# 翻译诉求戳穿真实议题

## 一句话定义

用大词包装的个人焦虑，用「**让我翻译一下你所说的话**」还原为具体个人议题。

## 详细说明

男生谈「教育局限」「人人理想生活」，大冰翻译：

- 「当下的学校培养方式，让我觉着指向清晰变成人上人可能性又不大，而且大家都这样，让我又有竞争的恐慌，是这个意思吧」

把宏观批判拉回个人竞争恐慌与资源匮乏，再对接「先拿门票」建议。
