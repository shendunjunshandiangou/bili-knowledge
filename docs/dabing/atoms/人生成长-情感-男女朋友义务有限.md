# 男女朋友义务有限

## 一句话定义

无法律婚约的恋爱关系，一方对另一方的义务**未必有那么多**；双方认定看「未来是否互相纳入」，而非单方面牺牲叙事。

## 详细说明

- 大冰：「现在并没有一个法律所约束的婚约，男女朋友，你需要对他担负的义务有那么多吗？大概率没有那么多吧」
- 女方自述：多年校友朋友重逢，以朋友相处；对方「上头」后走到一起，「像抱只小猫小狗」的依恋，「有吧也不多」
- 男方最大志愿是出家；女方初衷是「让他快乐幸福」，但说不清感情是什么——「有缘分就对你负点责」
- 工作机会：女方曾犹豫仍招他进团队，**提供过导入即可**；不胜任不等于女方需无限兜底
