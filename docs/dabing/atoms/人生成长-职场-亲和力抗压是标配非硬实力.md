# 亲和力抗压是标配非硬实力

## 一句话定义

亲和力、抗压、大场面从容等是**社会劳动标配**（及格线），不是HR选人的**硬实力**，不能指向获得新工作。

## 详细说明

- 连麦者自评三项：亲和力、抗压力、大场面从容
- 大冰：「这三点并不指向新工作的获得……剥削价值不在这三项上」
- 「它是标配——要求你具备这些才表明你是及格、可胜任基础工作」
- 若强调统筹力等硬实力，需岗位职务名称等**信任背书**，不能只听嘴上自证
