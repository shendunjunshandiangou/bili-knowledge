# 办公室发疯坐实软柿子

## 一句话定义

被同事拿捏后**在办公室发疯**，不是变强硬，而是坐实「你是软柿子，而且是有点烂了的软柿子」。

## 详细说明

- 大冰：「尽量别当软柿子……软柿子一旦当了，是捏不完的，一直把你捏呲了捏出汤来。」
- 连麦者曾以发疯自救；大冰纠偏：「只要你在办公室发疯，不仅事实佐证了你是软柿子，同时是已经有点烂了的软柿子。」
- 正解是「做好战斗准备，认真地研究你的对手」，冷静、有的放矢，而非情绪爆发。
