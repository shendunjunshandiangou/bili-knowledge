# 需求决定关系减沟通成本

## 一句话定义

**需求决定关系，没有需求保持友好**——对方对你无需求时，自我介绍越**清晰**越等于**减少沟通成本**，而非人为设置障碍。

## 详细说明

- 大冰：「没有那么多人给你那么多的耐心……您的身份、自我介绍的清晰，越清晰越等于减少沟通成本。」
- 30+ 仍不会自我介绍是「硬伤」，「尤其打不上你的年龄，太容易暴露个体基础实力」。
- 连麦者多份底层工无有效成长 → 先踏踏实实打工建立秩序，再谈方向。
