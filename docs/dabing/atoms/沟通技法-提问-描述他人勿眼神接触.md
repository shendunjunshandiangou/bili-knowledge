# 描述他人勿眼神接触

## 一句话定义

请连麦者评价在场亲属时，要求其转过头、避免与对方眼神接触，否则会修饰语言。

## 详细说明

父亲描述母子关系时，大冰：「眼睛不要看着他妈妈，因为你在描述任何一个人的时候，只要你和他发生了眼神接触，你就会修饰你的语言了——转过头去，不要看着他妈妈，直接跟我说，不要思考，直接说开始。」
