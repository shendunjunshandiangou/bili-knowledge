# 不为他人目光被迫减肥

## 一句话定义

伴侣若仅为迁就父母/他人眼光被迫减肥，应拒绝；若为穿婚纱等自我目标可减——「先给自己活」。

## 详细说明

哥哥父母要求微胖女友半年内减肥否则反对婚事。大冰：「她如果自己想减是她的自由，但来自你或父母的命令多伤人心」；除非理由如「过半年穿婚纱没有我的尺码」——「若只为迁就他人目光，凭什么减？凭什么给谁活？先给自己活」。并指出五年未向家人透露已造成伤害，「别欺负好人，我站她」。
