# 打破房价反弹侥幸

## 一句话定义

对方不愿亏本卖房时，直接打破幻想：**「不要做梦，明年今天更亏，后年更更亏」**——济宁济南不行，北京上海非好位置也不行。

## 详细说明

连麦者：济宁降价厉害，挂出去要赔很多，怕折腾。

大冰：

- 「放弃侥幸心理」
- 「现在卖亏，明年更亏，后年更更亏，不要认为还能翻盘、回到购买价——做梦」
- 「能卖出去就烧高香，能保住多少本保多少；关键是先解脱巨大经济压力」
- 「哪怕赔一半，卖出去第二天去泰山给老奶奶烧高香」
