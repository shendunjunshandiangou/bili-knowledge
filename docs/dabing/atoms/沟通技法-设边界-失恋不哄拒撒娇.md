# 失恋不哄拒撒娇

## 一句话定义

处理失恋议题时**不进行适时抚慰**——「别从这个世界撒娇」；失恋不可怜、不必因失恋享特殊哄劝。

## 详细说明

- 连麦者多「但是」、求哄；大冰：「我不负责哄你……失恋了我们在梳理，但不对你进行适时抚慰。」
- 每个「但是」=对梳理逻辑的不认同→推演为更需要抚慰，反而加强「靠时间」界定。
- 与「被辜负=自攻」「刺儿排异」配套：帮处理问题，不替世界撒娇。
