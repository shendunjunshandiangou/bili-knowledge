# 质朴诉求当场放行

## 一句话定义

连麦者简单直接说明真实诉求（如打广告）时，不摆架子、不绕弯，当场同意并给时间窗口。

## 详细说明

农村小妹第一次连麦就说「想借着你这打个广告」，大冰回应「没问题，来打吧」，并告知「这会儿7万多人，抓紧做」。随后评价「简单直接的上来就讲自己的诉求，特别的好」。

核心：**诉求清晰 + 态度诚实 → 先给机会**，再补执行条件（如挂小黄车）。
