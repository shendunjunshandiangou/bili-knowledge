# 逆向站在对方立场对冲

## 一句话定义

连麦者标准像「要么和我一样高收入，要么回来当贤妻良母」时，先声明**站在对方角度**「对冲一下」，揭单边选择感。

## 详细说明

男博士要求：要么有上进心，要么把家收拾干净——女友两方面都不占。

大冰：

- 「我先站在女孩子的角度，我们先来说两句啊，来对冲一下」
- 解读连麦者无意暴露的标准：「要不然你和我一样收入高，要不然你就回来好好给我当个贤妻良母」
- 「听起来挺像是单边选择……恋爱谈得很像是你很给他面子一样，这个我接受不太了」
- 补充女方视角：合租多年终于有小家，下班打游戏放松；「不是天天把家搞得这么乱，你不在家，收拾那么干净给谁看」
