# 比例上交留自由裁决

## 一句话定义

若仍想谈折中，唯一可谈方向：**按比例**（1/3、1/5、1/2）交给对方，其余保留自由裁决权——且明确仍是**婚内共同财产**。

## 详细说明

大冰免费赠送收尾话：

- 「各自让一步……你按照比例，你不心疼的比例，1/3 或 1/5 或 1/2 交给你」
- 「剩下的话你让我有一个自由裁决权」
- 「但这所有一切都叫做婚内共同财产」

前提：双方金钱观仍有谈的空间；若对方坚持「全是她的」且沙盘推演接受不了，则折中不成立。
