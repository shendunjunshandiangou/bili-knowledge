# 五秒停顿组织短语

## 一句话定义

对方说完后**不必立刻接话**——正常人可耐受约5秒停顿，用来组织最短准确回答。

## 详细说明

- 「您不是下一秒钟立马就要张嘴——可以停顿，五秒钟足够组织一句短语」
- 不够则明说：「嗯我想一想啊」——再争取8秒以上
- 沉默被感知为「在认真咀嚼你的话」——多数人不排斥
