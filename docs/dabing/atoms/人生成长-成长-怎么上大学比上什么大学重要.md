# 怎么上大学比上什么大学重要

## 一句话定义

张雪峰：**你怎么上大学比你上什么大学重要**——自律 + 主动争取资源，比校名本身更能决定大学阶段分化。

## 详细说明

- 「主动真的蛮重要的。」
- 同校同专业入学时看似相同，大学阶段差距会非常明显。
- 与「七次机会」「资源非均分」配套：好学校不自动等于好结果。
- 爱好须「相对擅长」，非仅喜欢——「天天听歌不叫爱好音乐」。
