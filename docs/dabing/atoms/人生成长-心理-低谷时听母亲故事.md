# 低谷时听母亲故事

## 一句话定义

人生低谷/security 感往往来自与母亲单独相处、听其讲述青春经历，而非继续分析自己的困境。

## 详细说明

帝师回忆赌球欠债等低谷全由母亲解决，「那几句话就有了安全感」。与大冰「回到妈妈身旁听故事」处方呼应——适用于中年危机、债务、迷茫等「自己解决不了」的阶段。
