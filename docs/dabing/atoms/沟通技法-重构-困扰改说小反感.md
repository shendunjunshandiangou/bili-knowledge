# 困扰改说小反感

## 一句话定义

把「对不起造成困扰」改理解为**小反感**——可规避、可操作；「困扰」预设不平等下位关系。

## 详细说明

连麦者担心「出于好意却造成困扰」。大冰：

- 「不是困扰，叫反感……小反感这个东西可以规避的，不是一个多大的问题」
- 「对不起我对你造成了困扰——你们关系位置非常不平等，对方处于下位概念」
- 对标具体场景在**动作上**浅层调整即可，不必扒皮抽筋式心理工程
