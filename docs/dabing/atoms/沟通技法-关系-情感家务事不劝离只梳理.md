# 情感家务事不劝离只梳理

## 一句话定义

对婚姻/情感家务事，**明确不替人做离婚决定**，只帮梳理关系结构与趋势，结论由当事人做。

## 详细说明

- 「我不能够劝你分劝你离，这不是我们该干的事，我只是帮你梳理分析一下」
- 梳理后给判断：「不是很乐观」「趁着还没有要孩子，您做出一个明智的选择吧」
- 但不替人按下按钮：「你要能受得了，你就接着受」
