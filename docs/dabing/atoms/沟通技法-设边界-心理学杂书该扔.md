# 心理学杂书该扔

## 一句话定义

当对方被多套心理学/元认知概念绕乱、把小事放大时，直说：**「该扔全扔掉，整砸了」**——多体系在脑子里打架。

## 详细说明

连麦者提「元认知」监督幻想，大冰没听说过；判断：

- 书看杂了，日常感悟写着写着不知道写啥
- 「好几套不同体系在你脑子里打架，把其实特别小的事放得很大」
- 幻想事不大，不需要元认知来解决
- 「不太建议这年龄段多接触，别把多重人格诱导出来」

用降维处理替代继续堆概念。
