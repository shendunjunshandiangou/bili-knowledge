# 吃饭不问意义类比

## 一句话定义

反驳「须先找到人生意义才能活」——如同吃饭不为赋予每顿饭哲学意义仍会吃，生存是自然需求，不必意义前置。

## 详细说明

厌世程序员追问为何而活。大冰：你吃饭前会思考这顿饭在生命长河中的抽象意义吗？「没有就好，如果有就不仅是虚无，叫脑子有问题」。婴儿尚无独立意识时并未思考为何而活却照样活、「少喂一口奶都会哭」——意义不是活的前提；若非要追求，两字「充实」。
