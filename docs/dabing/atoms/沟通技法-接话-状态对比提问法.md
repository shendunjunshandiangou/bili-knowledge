# 状态对比提问法

## 一句话定义

通过对比「当下连麦状态」与「问题场景中的状态」（如面试），帮对方看见**可被他人感知的外显姿态**，而非只纠结内容或能力。

## 详细说明

大冰在连麦者归因「反应力不够、回答不上来」时，连续追问：

- 「今天跟我交流的方式，和面试时一样吗？」
- 「面试时比现在更硬一点还是更软一点？」

进而指出：HR 感知的是**情绪姿态**（是否凶、是否难管、是否有对抗性），无法解读真实心理；能力可能被低估，但更可能是在面试关被判定为「减分项」。

**操作要点**：以当下为坐标，让当事人自己对比两种状态下的差异，再引出调整方向（如表情管理、语言软化）。
