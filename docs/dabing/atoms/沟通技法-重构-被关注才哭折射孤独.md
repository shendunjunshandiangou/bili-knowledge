# 被关注才哭折射孤独

## 一句话定义

连麦时情绪起伏大、哭泣——往往因**感知到被认真倾听**，而非事情本身多严重；折射在大城市生活**极度孤独**，工作寄托了几乎全部生活。

## 详细说明

- 「只是事情本身不至于起伏这么大……你清晰感知到被关注，所以才有情绪起伏」
- 对领导讲也会哭——谁认真听都会如此；**必须改**
