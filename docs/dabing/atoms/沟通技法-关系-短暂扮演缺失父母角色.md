# 短暂扮演缺失父母角色

## 一句话定义

在有限连麦时间内，**扮演连麦者内心渴望而缺失的父母角色**（常是父亲），补功能而非替代终身关系。

## 详细说明

大冰对18岁女生：

- 「我刚才不过是努力扮演了一下你内心当中真正希望有的那个父亲的角色」
- 因父亲在养育中「犯了这个错误比母亲要大」
- 「我的这个扮演只能在我们有限的聊天短短十来分钟」
- 接下来鼓励她自己给自己当父母
