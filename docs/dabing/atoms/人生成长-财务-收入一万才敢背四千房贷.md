# 收入一万才敢背四千房贷

## 一句话定义

房贷月供不应吞噬绝大部分收入；**月入约 1 万才合理背 4000 月供**，否则生活品质受本质影响。

## 详细说明

济宁工人案例：5000 月入、4000+ 月供、30 年贷、为学区前年买在高点。

大冰判断：

- 所谓刚需、给孩子留房常是心理安慰，**经济账没算明白**
- 合理结构：1 万收入里拿 4000 还贷，生活才不崩
- 先挂房卖解主要矛盾，再谈换工作提收入；仅换工作除非过万否则不够
