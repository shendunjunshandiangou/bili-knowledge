# 被辜负即自我攻击

## 一句话定义

心里产生「**我被辜负了**」，往往已启动**自我攻击**——隐藏版是「我瞎了眼了」；宁可攻击对方意识，勿攻击自己。

## 详细说明

- 29 岁女分手五年男友，觉被辜负生恨；大冰：「只要你心里边产生了我被辜负了，实际上你已经开始了你的自我攻击。」
- 老百姓版：错付、瞎了眼；极端版：同归于尽=自我攻击成分更高。
- 「宁可是他该死、让他死去……你也不要攻击自己。」
