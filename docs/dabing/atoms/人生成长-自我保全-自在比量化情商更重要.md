# 自在比量化情商更重要

## 一句话定义

到一定年纪，「怎么舒服怎么来、怎么自在怎么来」比追求情商分数或持续「积德」更重要；别较劲、别硬找烦恼。

## 详细说明

大冰：「都这岁数了，是不是没有什么比自在两个字更重要」；「什么事让你难受就不碰；谁说你不对了不搭理」。对修行过度：「烦恼即菩提，但别给自己没烦恼硬找烦恼，这他妈就是最好的菩提」；「别太较劲」——持戒也是告诉你别较劲，不是碰都不能碰。
