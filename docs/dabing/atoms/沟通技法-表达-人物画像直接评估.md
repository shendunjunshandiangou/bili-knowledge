# 人物画像直接评估

## 一句话定义

对连麦者提出的重大选择（如出家），不顺着对方愿望走，而是用**已展现的能力与意志**做基础评估，给出够直接的判断。

## 详细说明

女方问出家有无推荐、指路。大冰：

- 直播间不适合、也「没法吃这个」
- 用现有表现反推：「四个小时的工作都难以胜任、身心俱疲」→「我不相信他能一坐下来（修行）」
- 正规路要求：背楞言、凌晨三点半四点起床、三坛大戒——「出家不是逃避、不是避世、不是当香云野鹤，很苦的，挺累的」
- 直接结论：「我不太相信他有太高的追求，这个话说的够直接吧」
- 金句：「最勇猛精进的人、最勇敢的人才会选择走修行路」——想滋润过活却选此道，「磨走这条道，走的那条道也未必是正道」

适用于：对方理想与行为画像明显不匹配时，避免用意志力更强者的标准代推他人。
