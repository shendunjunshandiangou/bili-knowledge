# 措辞丝滑度识背景

## 一句话定义

听连麦者复述特定词汇（如「清华北大」）是否**丝滑无滞涩**，可推断其真实教育背景与经历层级。

## 详细说明

女生复述大冰名言时提到「清华北大」：

- 「你刚才复述我之前的话提到了清华北大两个字的时候，那么丝滑顺滑的，没有任何滞涩的来说出来，只能证明你的学校应该比清华北大还好吧」

对方随后承认在美国藤校。适用于连麦者模糊背景时的快速识人。
