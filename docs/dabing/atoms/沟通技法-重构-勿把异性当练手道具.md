# 勿把异性当练手道具

## 一句话定义

以「多经历」为由把女孩当**练手道具**心态不对——「那些女孩子成了你的道具了，那不行，拿她们练手不好。」

## 详细说明

- 工程师承认地铁加人是为多经历、知方法少；大冰先肯定想经历，再划边界。
- 与互联网搭讪课（快速占便宜）切割：「发心目的不一样……甚至影响社会风气。」
- 应通过志趣场域真诚建联，非纯玩技术。
