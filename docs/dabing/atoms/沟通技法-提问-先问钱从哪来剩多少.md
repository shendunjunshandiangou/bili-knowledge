# 先问钱从哪来剩多少

## 一句话定义

面对自称创业/理财的连麦者，**先问收入来源、周期、结余与资产配置**，再谈规划。

## 详细说明

- 「95个是靠什么赚来的？多长周期？实际赚到的现在还剩多少个？」
- 追问理财：定期还是取出、买黄金还是积存金、什么价位
- 信息到位后再判断：「2年赚近100个——大概率你不是迷茫的人」
