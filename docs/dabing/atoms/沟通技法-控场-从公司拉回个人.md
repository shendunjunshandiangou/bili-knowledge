# 从公司拉回个人

## 一句话定义

连麦者不断展开公司/行业内幕时，用「**你们公司我不关心，关心你**」将话题从八卦拉回对方个人处境与选择。

## 详细说明

女硕士连麦中详细描述：断崖式项目、人才流失、AI 布局扯淡、老板体制内背景、小领导制衡等——信息量越来越敏感。

大冰叫停句：「你今天找我聊的是啥？你们公司我不关心，关心你。」

随后才给个人建议：财务别碰、干好技术、少提建议、自我提升是你自己的课题。

作用：防止直播间变成行业黑料传播场；也把对方从「分析公司」拉回「我该怎么办」。
