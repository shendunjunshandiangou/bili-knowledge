# 被选择者先扩可能性

## 一句话定义

能力/资源未积累时，当事人是**被选择者**而非选择者——不谈「选行业」，谈扩大被选择的可能性。

## 详细说明

- 「选择权不在你这儿，你叫被选择者……尽量让自己被选择的可能性大一点」
- 比喻：「家里只有二两肉，非得给十口人包饺子——拿什么包？没有就炒菜别琢磨包饺子」
- 标准降到极低：「可触碰，甚至不要选——你能碰到啥就学啥，学了比不学强」
- 反对「选好再努力」的偷懒：「火烧眉毛才强调选择，你早干吗去了」
