# 找爹式关系投射

## 一句话定义

亲密关系或职场关系中，把对方当可托举、可保护、可承载的**「爹」**——初期设定不平等，对方承担不了期望就会「变了」；像投资，没有只赚不赔。

## 详细说明

- 感情：找爹式恋爱；初期由连麦者推动关系模式，后不愿承担破裂结果
- 职场：对主管像孩子面对父亲，试探包容上限——「是不是在找爹」
- 应先「成为相对完整的自己」，再建平等情侣模式
- 区别于 [沟通技法-重构-需求决定关系非父兄](/dabing/atoms/沟通技法-重构-需求决定关系非父兄.html)（职场慕强请教），本原子侧重**情感依附投射**
