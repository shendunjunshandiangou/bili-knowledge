# 反对小主持人班

## 一句话定义

针对 6–8 岁「小小主持人/口才班」，大冰以**前媒体人+早期儿童综艺**经验**强烈反对**：教坏的多、教好的几乎没有，本质是表演感与被关注，非真沟通。

## 详细说明

核心论点：

- 孩子喜欢被关注（≈被爱），易变形为舞台表演
- 天性被泯杀、小大人儿、塑料感成长违背常识
- 背稿式表达长大影响正常沟通——「不说人话」
- 时代变了，报幕式表达社会不接受，弊大于利
- 真正有良心的前媒体人往往不对儿童做定向培养；生计所迫者难对未来负责
- 出口多是省市级无聊朗诵赛，正向加分看不到

「您完全可以不接受，但我忍不住必须发表观点。」
