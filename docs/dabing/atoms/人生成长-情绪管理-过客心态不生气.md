# 过客心态不生气

## 一句话定义

把一次性冲突中的对方视为生命里的**过客**——「有可能这辈子我们只聊这一次天」——因链接浅、对己不重要，故「没有必要生你的气」。

## 详细说明

连麦者摊牌后，大冰用两句话总结：

> 「你觉着咱们俩的生命链接能够有多深？……有可能这辈子我们只聊这一次天，只说这一次话。」

> 「在我的生命当中，我的认知是因为您是我的过客……所以我没有必要生你的气嘛，因为归根到底对我不重要的一件事。」

**与「选择性生气」并列的观点**（同集）：

- 奔五之后「没有那么多值得我去生气的东西」
- 唯一举例的真生气：山东父亲阻挠儿子上大学——涉及年轻人前程、且事不关己时仍会动气；「如果是我自己的事的话，大概率我是不会生气的」

**适用**：网络陌生人挑衅、一次性连麦、无需建立长期关系的场合；不适用于需维护的核心关系。
