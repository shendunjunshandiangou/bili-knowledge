# 世界不会哄你

## 一句话定义

对期待社会同情的年轻人，直接点破：**同情不解决问题**，唯有发现短板并行动。

## 详细说明

- 「这个世界和这个社会不会一直哄着你……成年人要为自己买单」
- 「几万人在线说你可怜——屁用没有，不解决你的问题」
- 「解决你的问题就一句话：发现自己短在哪了，现在开始不晚」
