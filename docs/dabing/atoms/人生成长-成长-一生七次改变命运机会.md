# 一生七次改变命运机会

## 一句话定义

张雪峰引述说法：人一生平均会遇到**七次改变命运的机会**，把握一次即可；差别在于能否识别为机会、是否**主动**争取。

## 详细说明

- 「你把握住一次，就足以改变命运。」
- 主动 vs 被动：有人需鞭子抽，有人「相对比较机灵，会去各种尝试」。
- 与大学资源、自律、爱好深度等议题串联——机会需配合行动。
