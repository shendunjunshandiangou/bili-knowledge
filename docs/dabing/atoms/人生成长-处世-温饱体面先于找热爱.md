# 温饱体面先于找热爱

## 一句话定义

不知自己喜欢什么**不害怕**——在试错空间狭窄时，先**奔温饱**，温饱之上还有**体面**；一辈子不知所爱也可保证二者。

## 详细说明

- 「在不知道自己喜欢什么的情况之下，您就先奔着温饱去……奔着温饱体面去就是了。」
- 「如果你一辈子不知道你到底喜欢什么，也不害怕——最起码把最基本的温饱体面保证了。」
- 勿只想结果忽略过程；已有考学/深造规划则「奔着这个方向去就好，过程中想太多没有意义。」
- 前提：不影响当下已获温饱和体面。
