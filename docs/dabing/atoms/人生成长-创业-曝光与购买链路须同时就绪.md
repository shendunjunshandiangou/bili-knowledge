# 曝光与购买链路须同时就绪

## 一句话定义

获得流量或口头推荐后，若没有小黄车、联系方式等购买路径，机会等于浪费。

## 详细说明

小妹有几十斤腌黄瓜和大冰直播间曝光，但不会挂小黄车、不会自夸，大冰总结：广告做了却落不了地。行动清单：开购物车/留购买方式 + 能简短介绍产品。
