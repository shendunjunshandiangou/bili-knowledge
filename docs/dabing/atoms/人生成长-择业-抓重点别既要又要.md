# 抓重点别既要又要

## 一句话定义

择业不能**又要清闲又要高薪又要成就又要空间**——抓最主要一项，其余稍一稍；天底下全无此类好事。

## 详细说明

- 要清闲就别指望大钱；要赚钱别嫌夜班累；要机会空间就选空间大的，别纠结钱少
- 连麦者「既要又要」——大冰两层框架之一
- 与 [人生成长-择业-怎么选都会后悔](/dabing/atoms/人生成长-择业-怎么选都会后悔.html) 配合：抓重点后仍可能后悔，但须先选
