# 差异化人设破局

## 一句话定义

自媒体/破冰时，**身份与行为的反差**（如局里局气穿行政夹克转呼啦圈）比行为本身量级更抓人——「重点不是转几千个，是谁在转」。

## 详细说明

- 运动员转呼啦圈正常；「张主任」西装转 = 太牛逼
- 穿着越商务/行政，差异化越强；可叠加背头、眼镜、分析形势
- 大众传媒逻辑：受众关心**身份标签** + 反常识行为
