# 职场哭坐实弱者

## 一句话定义

职场倾诉时**哭泣**并非占理——反而加强他人认知「你是弱者」；潜意识是讨可怜、求饶，须收起。

## 详细说明

- 「并不是你哭你就有理了……哭加强大众认知：你是弱者」
- 连麦者平常日子不哭，被倾听才哭 → 见 [沟通技法-重构-被关注才哭折射孤独](/dabing/atoms/沟通技法-重构-被关注才哭折射孤独.html)
- 把精力放在该用的地方（如注会）
