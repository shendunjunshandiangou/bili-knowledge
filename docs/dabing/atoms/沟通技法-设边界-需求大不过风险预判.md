# 需求大不过风险预判

## 一句话定义

连麦者再强调「迫切需要指点」，若风险预判已拉起，明确：**后者比前者大，跟你想不想说没有关系**——礼貌抬走。

## 详细说明

连麦者反复说「迫切需要您给我指点」。大冰：

- 「您的需求以及多么迫切，它大不过咱们对于这个风险的预判了」
- 「后者的话比前者大，它跟你想不想没有关系了」
- 给出替代：其他直播间、主播、心理咨询师、心理疗愈师
- 「我们先聊到这吧，你再找找别的一些路径好吗」
