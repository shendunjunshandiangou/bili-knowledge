# 励志多因没回头路

## 一句话定义

外界看到的「励志」往往是**生活一巴掌打过来、没有回头路只能往前走**；走过后再自我美化成奋斗故事。

## 详细说明

- 「我不太相信励志这个东西……所谓外界看到的励志往往都是生活一巴掌打过来，你没有回头路而已，你只能往前走。」
- 「不走会死掉、会被干掉……生如逆旅单行道，哪有岁月可回头。」
- 若干年后他人界定「哇你很励志、意志力真坚强」——当事人可能想：「对，谣言。」
- 与连麦者共识：「很多时候只是一个标签而已，每个人情况不一样。」
