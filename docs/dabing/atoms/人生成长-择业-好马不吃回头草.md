# 好马不吃回头草

## 一句话定义

曾被原单位**拒绝**后又被召回——大概率需求只是阶段性；**去而复回**有额外成本，他人会重新审视你；真缺人才第一次就该让你回。

## 详细说明

- 连麦者内心想回轻松岗位，但回去照样后悔、遇领导不欢迎
- 「好马不吃回头草」——看到的是待遇轻松，忽略评估成本
- 与 [人生成长-择业-怎么选都会后悔](/dabing/atoms/人生成长-择业-怎么选都会后悔.html) 同场：选回也会后悔
