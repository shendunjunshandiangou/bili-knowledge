# 收入上升后的精神俯视

## 一句话定义

伴侣矛盾在上班赚钱后突然放大，常因**不自觉的精神俯视**——把寒窗苦读的「应得回报」打包进女友匹配度，失衡会越往后越麻烦。

## 详细说明

连麦者博士毕业前不困惑，上班后纠结女友懒与收入低。大冰：

- 「上了班能赚到二三十万之后，你自己都没有意识到，开始多多少少有一点在精神层面俯视他了」
- 「俯视必须从源头处斩断，越往后你会拿她和你不停比较，进入‘你配不配得上我’阶段」
- 「质朴算法」：多年投入应得回报，粗浅算法里**打包上了女朋友**——匹配度不高则心里失衡，「影响做人根本，特别不好」
- 对照：博士毕业压力期 vs 现期，「审视他的角度一致吗？」——「恋爱时期的你和现在你是两个你」
