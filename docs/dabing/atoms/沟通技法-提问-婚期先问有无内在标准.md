# 婚期先问有无内在标准

## 一句话定义

对方问「什么时候适合订婚」时，若其自身无基本标准，先指出「这一点挺可怕」——此刻不适合推进。

## 详细说明

连麦者问何时从恋爱到婚姻。大冰：「我不知道什么阶段合适，但你目前的阶段一定不合适」——「关于要不要步入婚姻……你还需要出来问人，自己并没有设立基础标准」。应先有足够时间发现好坏面、测彼此包容度，而非「谈好多年就得结」。
