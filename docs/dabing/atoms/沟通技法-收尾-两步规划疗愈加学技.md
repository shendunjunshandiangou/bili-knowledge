# 两步规划疗愈加学技

## 一句话定义

对创伤青少年：**第一步心理疗愈**（如创可贴），**第二步学技术找学徒**；梦想（如写作）排序靠后，与生存不冲突。

## 详细说明

- 第一步：心理创伤需疗愈，非「有心理疾病」，像划口贴创可贴
- 第二步：学徒/后厨/机械等，不宜高频大人群（外卖分拣等）
- 写作等放第四五步：靠写作吃饱饭不易，先有稳定收入再靠拢
