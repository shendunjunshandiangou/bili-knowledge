# 审慎寻求只指出问题的咨询

## 一句话定义

社会面不少心理咨询**只帮你看清问题、不掌握方法论解决**——审慎去做，否则陷入「发现问题又解决不了」。

## 详细说明

连麦者曾Seek心理咨询，分析出更多问题却无办法。大冰：

- 直播间「很少告诉你去接受心理咨询」，尤其技能背景判不准的
- 「他懂一些，会帮你指出问题，但大部分并不会帮你一起来把问题给解决」
- 「与其这样，反而有些问题本身让你看清楚了，这个动作要审慎地去做」
- 否则「发现问题又解决不了，不掌握方法论」
