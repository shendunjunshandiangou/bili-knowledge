# 涉刑案件帮不了

## 一句话定义

涉及**刑事边缘/暴力幻想/复杂抚养权诉讼**时，**当场声明帮不了**，转介律师等专业渠道，不在直播间深聊。

## 详细说明

- 「实在抱歉，我可能帮不了你……不是我们能够帮得了的」
- 建议：找律师打抚养权官司
- 「我不是讨论这个问题」——拒绝展开仇恨/暴力细节
