# 紧急案件摇专业资源

## 一句话定义

直播间非法律/救援本职，但涉及**极幼女性侵害、人口买卖、自杀风险**等时，立刻协调认识的律师、志愿者等专业资源下场，不隔夜处理。

## 详细说明

16岁被卖婚女孩案例：

- 定性：不是婚姻，疑人口买卖、幼女性侵、非法拘禁
- 路径：浙江当地妇联协助报警（优于网友直接报警）
- 行动：深夜联系徐律；边言边语找账号、要联系方式、做切片
- 向观众说明：「我们虽然不是法律人士，但是不见得我们不认识……这是一个孩子，必须让专业朋友下场」
- 跟进：徐律已加求助者微信，等候聊天记录协助报警
