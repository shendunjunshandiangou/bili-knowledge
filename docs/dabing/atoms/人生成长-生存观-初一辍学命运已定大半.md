# 初一辍学命运已定大半

## 一句话定义

寒门子弟在当代靠学业提升改命；初一下学基本错失主路径，后续只能惜命式维持，而非幻想轻易跃迁。

## 详细说明

23岁外卖员初一辍学。大冰：「你的命运在你初一下学的时候就已经决定了」——错失唯一可改变命运的路径；互联网农村大火概率太低；无完整教育与职业培训，「社会分层就是这么残酷」。现策略：学手艺没时间、9000是全家收入——只能锻炼身体、买保险、**惜命**，「你垮不起、生不起病」。
