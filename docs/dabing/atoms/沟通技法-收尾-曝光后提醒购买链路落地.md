# 曝光后提醒购买链路落地

## 一句话定义

帮对方做完口头宣传后，指出缺购买入口会导致「广告落不了地」，并给出下播后的具体动作。

## 详细说明

连麦者没挂小黄车，大冰指出：「你打完这广告，人家别人想买也没有地方买啊」。收尾建议：下播后主页会有人来留言私信，「尽快上一个小黄车能把你的黄瓜卖出去」。

金句：「上来打过广告的话，你连个小黄车都没有，我替你打完这广告都落不了地啊。」
