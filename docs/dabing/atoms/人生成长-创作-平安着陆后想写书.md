# 平安着陆后想写书

## 一句话定义

大冰的世俗追求不是回电视台当主持人，而是**若平安着陆，回到安安静静写书**——享受过去十年写书的状态，而非大众传媒。

## 详细说明

- 老大哥预言电视台会请他，大冰回应「其实已经请了，只不过现在不想想时机」
- 个人诉求：「如果可以平安着陆，有一天就像你自己可以开台」——但开台太累；若世俗追求，更想写书
- 写书状态：「安安静静的，每天吃我妈做的饭」；写书很苦、赚不了什么钱，「散财的速度比赚钱要快」
- 行业现实：写真东西往往不能卖；作家杂志多年未发专刊
