# 子女第一项职责是好好活着

## 一句话定义

为人子女面对父母，**第一项职责**不是让父母过上好日子或替他们还债，而是**以身心健康的状态存活、让生命健康延续**——「己先达而后达人，己先自度而后度人」。

## 详细说明

连麦者把「让父母不再打工、过上好日子」列为对 family 的第一责任。大冰重排：

- 第一项应是「你好好活着……能让自己的生命健康的延续」
- 探讨一切议题前，「先不要着急说是我想怎样为他人」
- 「以己度人、以己推人」——**先把自己搞好搞明白**，至亲父母也是「他人」
- 若排序里「看不到这个我字」，是通过压制自我做奉献牺牲，「有严重问题」

与当前状态的关联：莫名哭泣、躯体化、多份兼职——说明第一项职责**已无法很好完成**；只劝「休息放松」不够，要从责任认知上动刀。
