# 对方着急时先降温

## 一句话定义

代述者或连麦者因紧急情况**不断插话、急成一片**时，先要求对方允许你把话说完，否则无法面对问题。

## 详细说明

16岁被卖婚案例中，网友着急打断。大冰：

- 「你是不是允许我先把刚才我要说的说完呢」
- 「你先不要着急……如果你上来急成慌成这样，我们怎么来面对这个事情呢」
- 「那只能够证明你的年龄也不满18岁吗——你有18岁吗」

先恢复对话秩序，再给妇联/报警路径。
