# 直播间倾诉别回家说

## 一句话定义

连麦梳理出的真实想法（含对伴侣的嫌弃、市场比较）**别带回家对伴侣说**——「当你女朋友一个屁别放」，否则对方可能真走。

## 详细说明

大冰收尾叮嘱：

- 「在直播间所有的这些想法说的这些话，当你女朋友一个屁别放啊」
- 「但凡放完了之后，人家姑娘主动找去了，人家才不要你呢」
- 可执行下一步：挂麦去商量请什么样的钟点工

适用于：公共连麦中说了伤关系的大实话，需区分「梳理场域」与「亲密沟通场域」。
