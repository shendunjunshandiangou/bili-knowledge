# 纠正过度客套拉近距离

## 一句话定义

当对方用「汇报」等拉开距离的词，或对话变得过于官方时，**直接点破并纠正**，把关系拉回平等、自然的说话方式。

## 详细说明

**纠正用词**：张雪峰说向大冰「汇报」烟台残疾女孩后续，大冰立刻回应：「首先不能用汇报这个词儿，特别的感谢我。」——先否定不对等的 framing，再表达真诚感谢。

**打破官方腔**：两人直播对话变客套时，大冰笑问：「在直播间这么多人看着咱俩说话，都这么的官方吗？」「你不能这么客气，你这么客气咱就不会了。」张雪峰接话「该怎么样怎么样」，对话随即自然展开。

**操作要点**：

1. 对明显不对等的词（汇报、请示等）当场温和纠正。
2. 用半开玩笑的方式点破「太客气」。
3. 示范自己希望的关系温度（「该怎么样怎么样」）。
