# 标配互动动作

## 一句话定义

学校教育之外，社会教育要求每个人学会在生态环境里**与他人建立连接**；其中一套「标配」是：可以不让人特别喜欢你，但要**尽量去掉让人不喜欢的理由**。

## 详细说明

大冰在讲「回答前先软化」时抬升到这一层：完成学校教育后，需要一定量**社会教育**——学会跟世界、跟他人发生关联。

面试面对 HR 的对标：把对方当人，在语言上给出浅层的**双向认可**（肤浅的浅 + 潜水的潜），属于生存标配，不是马屁。

与「因人而异定制沟通」配合：标配是底线，具体对象还要细分姿态。
