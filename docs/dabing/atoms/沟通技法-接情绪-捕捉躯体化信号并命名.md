# 捕捉躯体化信号并命名

## 一句话定义

在对方长篇诉困后，**提取并命名**其身心危险信号（莫名哭泣、开会冒冷汗手抖、枕头哭湿等），将其与对方信奉的「家庭责任真理」挂钩，点明「迫在眉睫」——而非仅安慰「别太累」。

## 详细说明

连麦者描述：下班独自哭、公司厕所哭、半夜哭到枕头湿、开会突然冒冷汗手抖、同事一句关心就绷不住大哭。大冰没有停在「理解你不容易」，而是：

1. **复述关键信号**：「莫名其妙的哭」「会上出现的那个情况，已经不是单纯的躯体化」
2. **与行为模式串联**：卷王外号、大厂本职已累、公余时间全打工、还帮弟弟还网贷
3. **命名根源**：「这一切都指向了这套所谓的真理给你的绑架……到了一个迫在眉睫、必须抓紧去解决的时期」
4. **后续强化比喻**：「没有任何一个人是可以无限上弦的琴弦……这个弦再拧两下就要崩断了」；「只告诉你休息放松，屁用不管，只能从根儿上来动刀」

适用：对方用「小太阳」「卷王」等人设掩盖透支，或把哭泣归因为「矫情」时——先** legitimizing 严重性**，再进入认知重构。
