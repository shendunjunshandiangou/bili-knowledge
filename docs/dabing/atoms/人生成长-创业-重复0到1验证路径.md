# 重复0到1验证路径

## 一句话定义

已有创业验证的人，新动作应**重复曾走通从0到1的路径**，而非盲目换赛道交学费。

## 详细说明

- 「如果你打算来赚钱/创业，最好依据你曾验证成功、走通过从0~1的路径，尝试重复那个东西」
- 去义乌若奔「长期积累」自圆其说；若奔同样多钱+愿意交学费则可去
- 「大可不必再去重复做一个从0~1的动作」——尤其已有95万验证
