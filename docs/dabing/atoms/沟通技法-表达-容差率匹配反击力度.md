# 容差率匹配反击力度

## 一句话定义

先选定**容差率**（能否接受冒犯），再用天平评估本次冒犯是否超出容差，从而决定反击力度。

## 详细说明

- 冒犯事实 = 天平一端砝码；对照容差率看是否超出
- 在容差内：「未必要跟着一块傻缺」；超出：该反击反击
- 软刀子 vs 硬碰硬取决于内核，不是「想反击又怕戳太疼」的情绪
