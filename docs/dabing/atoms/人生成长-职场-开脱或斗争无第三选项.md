# 开脱或斗争无第三选项

## 一句话定义

职场被孤立、被恶意时：要么**开脱**（强化「我来领工资」、像零零后不管闲事），要么**树立斗争意识**（谁骂骂回去，没理搅浑）——**没有第三个选项**。

## 详细说明

- 大冰判断连麦者做不到真战斗 → 适合反复加强「来赚钱」认知 + 注会
- 斗争定义：办公室政治基础斗争；有理得理不饶人，没理把水搅浑
- 与 [人生成长-职场-来上班赚钱关系差点无妨](/dabing/atoms/人生成长-职场-来上班赚钱关系差点无妨.html) 同向，本原子强调**二选一框架**
