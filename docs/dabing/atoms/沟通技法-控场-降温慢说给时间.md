# 降温慢说给时间

## 一句话定义

当对方情绪升高、语带指责时，用「别激动」「慢慢说」「喝口水」「咱们有时间」等话术**主动降速、给缓冲**，同时对比自身状态（「我现在一定比你正常」），避免被拖入对骂节奏。

## 详细说明

本集多处出现：

| 情境 | 大冰话术 |
| --- | --- |
| 对方说「别激动，我这是好好跟你说话呢」 | 「不要激动啊，下一个问题是啥」 |
| 对方想不起第三题 | 「慢慢想，没事，咱们俩有时间」 |
| 对方组织语言 | 「别紧张别紧张，喝口水慢慢说，咱俩有时间」 |
| 对方称阴阳怪气、能否正常聊天 | 「你不要激动……我现在一定比你正常……慢慢说慢慢说好不好」 |
| 等第四题 | 「你要不要再组织一下语言，我可以等你一下，我点根白将等你一下，慢慢想不着急啊」 |
| 对方更生气 | 「您不要生气啊……消消气消消气」 |

**核心逻辑**：不跟对方抢「谁更占理」，而是控**节奏**和**情绪温度**；配合「你感觉框架」使用，让对方在慢节奏下难以维持攻击性势能。
