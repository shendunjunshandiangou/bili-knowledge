# 回到妈妈身旁单独聊

## 一句话定义

自己过不去坎时，回到母亲身边、清场单独相处，听妈妈讲年轻时的事——不聊自己的难题。

## 详细说明

帝师问迷茫/负债年轻人怎么办。大冰：「回到妈妈身旁，不要让别人在场，和妈妈单独呆两个小时」；「不要去聊那些你自己遇到的问题，你和妈妈聊聊天，问一问妈妈年轻时候的一些事情，听妈妈讲故事给你——特别管用」。大冰自述每过几年会和母亲认真聊一晚上（如赤脚医生接生故事）。
