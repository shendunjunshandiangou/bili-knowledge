# 职场分档设底线

## 一句话定义

勿把同一套「琢磨、美化、客气」对标**所有人**——画人物谱系**分档**，该摔贫摔贫、该生平生平；到预警线**反弹亮底线**。

## 详细说明

- 找本子画人物谱系，分几档；「不要把现在的琢磨状态对标职场所有人。」
- 对一些人「直来直往」；设立预警机制——到程度该反弹、亮明底线。
- 平级同事冒犯尊严可翻脸：「都是爹生妈养的……我的自我必须第一时间告诉你碰到底线了。」
- **勿跟发薪人翻脸**；新工作/调动=重新开始，「现在设立不算晚。」
