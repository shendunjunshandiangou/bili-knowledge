# 艺术路窄别幻想改行

## 一句话定义

从小单一投入的艺术生，本科混出后改行难；怕的是「上不着天下不着地」时父母已老。

## 详细说明

大冰：学艺术「实际上窄的很」；音乐比美术更窄，「专一下真改不了」；「手不能提肩不能扛」，困难时期干体力活都怕伤手。若只混本科不进研，可替代性强、竞争大，等父母老时无学历背书更麻烦。已在211优势位应继续深造。
