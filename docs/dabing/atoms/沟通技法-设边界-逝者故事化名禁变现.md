# 逝者故事化名禁变现

## 一句话定义

基于真实逝者创作须化名、改城市与时间线，且不商业变现，避免对家属二次伤害与道德攻击。

## 详细说明

程序员想用AI做与已故未婚妻的动漫。大冰：信息对称时代，家属看到等于「伤口被第二次扒开」——须换名换城换时间线；「尽量不要给他们对称这个信息」；「不要有任何商业变现可能性」，否则道德洁癖者会攻击。生者「不见得一辈子只能活在这一段故事当中」。
