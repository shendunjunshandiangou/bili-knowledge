# 预判不说死留弹性

## 一句话定义

在传达趋势判断或计划变更时，**反复强调「可能性」「没有把话说死」**，避免观众将预判当作铁律，同时保留自身调整空间。

## 详细说明

大冰多次使用这一模式回应「夏天后停播」焦虑：

- 给出专业预判（AI 替代、三季度节点）后，立刻补「有很大的可能性」「话也没有完全说死」
- 区分层次：不连麦 ≠ 不直播 ≠ 退网；可能摩旅、吃播、低频开播
- 留条件出口：「如果 AI 没那么好用」「社会仍需要，该回来还会回来」

作用：既传递真实判断，又避免因「说死」引发集体焦虑或日后被打脸；对双方都是缓冲。
