# 敏感政策直播点到即止

## 一句话定义

劳动社保等**敏感、易误导**的议题，直播间只做方向性提示（找仲裁/法律援助），**不展开具体操作细节**，避免「影响不好」。

## 详细说明

- 连麦者问 3 年私账用工能否补社保；大冰分析举证难后：「算了别去找人家麻烦了」「聊了半天了祝你顺利」。
- 建议真在乎则去劳动仲裁咨询，但强调「这些我不专业，你最好找专业的问一下」。
- 切片语境：大冰「不想解释太清楚」——直播非法律咨询场域。
