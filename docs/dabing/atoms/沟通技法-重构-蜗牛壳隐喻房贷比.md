# 蜗牛壳隐喻房贷比

## 一句话定义

用「**蜗牛壳太沉**」形容收入/房贷严重失衡——赚 5000 还 4000，生活品质被壳压垮。

## 详细说明

大冰算账：

- 5000 收入 4000+ 房贷 → 「壳儿太沉」，剩下不够养娃吃饭
- 敢背 4000 月供，至少月入 1 万——「交 4000 生活品质才不会受到本质影响」
- 壳会压 10–20 年，影响家庭质量提升；换工作除非月薪过万否则解决不了

视觉化让听众感受比例荒谬，而非抽象「压力大」。
