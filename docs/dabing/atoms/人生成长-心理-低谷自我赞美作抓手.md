# 低谷自我赞美作抓手

## 一句话定义

低谷时**越不能否定自己**——从过往找小战绩甚至适度夸大，作「自我赞美」抓手，从光滑深渊壁往上爬。

## 详细说明

- 「积极的自我认可……包含很重要的一条叫做自己赞美自己。」
- 还没成事的人「越没有成事，你越得积极地认可自己」。
- 低谷还往前走：一没回头路，二「越是在人生低谷，越不能否定自己……找闪光点甚至夸大化——我他妈成过事儿，我不是那么糟糕。」
- 「活着一场经常需要自救……事没来之前先把自救机制建立好。」
- 可主动向外讨要赞美加强自我认可，「前提是我们都是普通人，不需要当圣人。」
