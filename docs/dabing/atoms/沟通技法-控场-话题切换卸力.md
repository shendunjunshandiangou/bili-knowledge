# 话题切换卸力

## 一句话定义

对峙僵局、对方持续生气时，**主动切到无害的日常话题**（如饮食、地域），用闲聊打断攻击循环，为双方降温——「咱们聊一点轻松的，昨天晚上吃的什么呀？」

## 详细说明

连麦者在多轮「你说得对」后仍称「你这种语气就越来越让我生气」。大冰在「消消气」之后转向：

> 「咱们聊一点轻松的，昨天晚上吃的什么呀？」

随后展开面条、是否炒菜、成都肥肠面等饮食话题，连麦者情绪明显缓和，并 eventually 坦承「我刚刚在跟你开玩笑……看你会不会生气」。

**操作要点**：

1. 先简短安抚（「消消气」）
2. 用开放式、低 stakes 问题切换频道
3. 顺着对方回答自然延伸，不强行拉回冲突话题
4. 适用于网络连麦、线下争执后需要冷却，但**不适用于**必须当场解决的原则性问题
