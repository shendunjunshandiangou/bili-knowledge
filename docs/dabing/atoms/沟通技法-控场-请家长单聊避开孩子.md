# 请家长单聊避开孩子

## 一句话定义

需对家长讲敏感批评时：**「让孩子出去，不要让孩子听到我们接下来聊天」**——公开段给童面子，批评段无童在场。

## 详细说明

流程：

1. 与 7 岁女孩短聊、听朗诵
2. 「我可不可以跟你妈妈聊一会儿？你去喊一下妈妈」
3. 妈妈接麦：「让孩子出去，我有事找你，不要让孩子听到」
4. 展开对小主持人班的强烈反对
5. 收尾再强调：「必须避开孩子讲，在孩子面前一丝打击都不会有——归根到底不是孩子的事」
