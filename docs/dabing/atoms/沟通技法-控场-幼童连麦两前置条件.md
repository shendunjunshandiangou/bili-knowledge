# 幼童连麦两前置条件

## 一句话定义

幼童上连麦前立**两条件**：① 尽量不要让该年龄段看直播（成人信息伤童真）；② 家长把手机交给孩子后**离开房间**。

## 详细说明

大冰对妈妈：

1. 「尽量不要让这个年龄段小朋友看直播……接触太早，担心影响正常长大，过早抹杀童真因素」
2. 「把手机交给孩子，您离开这个房间，好吗？」
