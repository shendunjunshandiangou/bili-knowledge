# 护短自己人硬边界

## 一句话定义

对**威胁、骚扰自己体系内成员**（兄弟、小弟、合作方）的人，不留情面，直接硬边界驱逐——「我不骂你，但我就俩字，滚蛋」。

## 详细说明

连麦者用自杀威胁大冰小屋老板（钟不悔，妻重病、幼子一岁）：

- 「这是我兄弟，我小弟，我罩着的是跟过我的人」
- 念其威胁留言：「在我离开之前让全网知道社会虚伪冷漠」「会去你们小屋自杀」
- 「我不骂你，但是我实在忍不住就俩字儿——滚蛋啊」

与对普通连麦者的耐心形成对比：碰自己人底线则零容忍。
