# 东边不亮西边亮副业清单

## 一句话定义

主业保身份与五险一金、**西边副业**按个体能力列清单试——切片带货、写手接单、个人公众号等，「哪个能赚到钱就把精力投射过去」。

## 详细说明

保留工作前提下双议题：

**西边（副业）**：
- 图片类切片挂链（可配大冰或他人图像+摘句）
- 刷热度公号应写手/P 图招聘，按件计酬
- 25 岁基层媒体人自述公众号：「好的坏的我都敢说」

**原则**：
- 依据文字、拍摄、时政公文等**已有能力**延展，非空算主意
- 「几点都可以同时做，开头全都学，哪个成就把精力主要往那块投射」
