# 伤害了人要登门礼数

## 一句话定义

不小心弄伤同学后，除学校和解外，应上门当面道歉+带点心意；仅打电话「来得轻」，礼数不能缺。

## 详细说明

男孩课间兴奋撞伤同学头破流血，已送医务室、家长电话。大冰：「你爹应该领着你买点心上门看看」；「打电话来得轻，得面对面，这是个礼数」；「不能在礼数上有缺」，否则易留矛盾根子。认错应针对对方伤害，而非纠结自己「没面子」。
