# 摇人脉落实具体帮助

## 一句话定义

不只给建议，**现场打电话**对接可信的本地负责人，把设备、资金、对接等落到可执行的人身上。

## 详细说明

- 卧床单手打游戏少年需双机直播：翻朋友圈找南昌大冰小屋负责人小江，当场拨电话
- 交代清单：华为系适合游戏直播的手机、小炕桌、3米充电线、高支架、补光灯；了解房间架设环境
- 「钱你先垫付，我路过南昌当面还你」——降低对方顾虑
- 说明为何不找台州本地朋友：「这个点做生意的都睡了」
