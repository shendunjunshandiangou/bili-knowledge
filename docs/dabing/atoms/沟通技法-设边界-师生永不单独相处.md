# 师生永不单独相处

## 一句话定义

异性师生场景硬底线：永远不要单独相处；必须接触时须第三方在场（如办公室开门、喊其他老师）。

## 详细说明

高中男老师担心对女学生好感。大冰：「永远不要有单独相处；必须的接触一定要有第三方在场」——「很多老教师都有经验，男老师批评女学生必须开着办公室的门」。这是「最终防线」，因「人性复杂、自我防线极易被攻破」。
