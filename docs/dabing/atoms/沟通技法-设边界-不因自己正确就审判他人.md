# 不因自己正确就审判他人

## 一句话定义

自己做对的事值得鼓励，但**不能因自己正确就认定他人错误**——避免二元对立式道德审判。

## 详细说明

男生帮老人捡纸，批判同学不入团不搞志愿。大冰：

- 「你有做出这个行为一定值得鼓励，甚至认可，但不能够因为我们做对了，我们就一定去批判别人做错了」
- 「评价的前提不应该是建立在我做对了所以你们做错了，这个太二元对立了」
- 「十五六岁，他们也才十五六岁，给他们一点成长的空间」
