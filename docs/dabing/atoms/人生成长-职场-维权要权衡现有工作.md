# 维权要权衡现有工作

## 一句话定义

主张社保/仲裁时须同时算两本账：**将来保障** vs **闹了之后现职还在不在**。

## 详细说明

- 大冰：「一方面你得考虑将来有没有保障，另一方面你也得考虑现在你跟人家这么一闹，你这个工作呀还有没有的这个问题，这都得考虑一下。」
- 方向：找劳动仲裁/法律援助，但决策权在本人；直播间不提供专业法律意见。
