# 路径依赖锁死自己

## 一句话定义

有工作后全身心押一条道、以为道不通就会饿死——**路径依赖**是锁死很多人的四个字；须先解开意识之锁，再谈能力提升。

## 详细说明

- 能力唯一决定论也是锁；能力提升漫长，在此之前先打开意识
- 与 [人生成长-职场-游牧思维替代农耕思维](/dabing/atoms/人生成长-职场-游牧思维替代农耕思维.html) 相对：农民没地可种，游牧民可转场
- 「选择设陷」：以为啥都能学、啥都能选 → 实际大部分选项够不着
