# 给方案前先摸清上级沟通

## 一句话定义

连麦者职场困境复杂时，**先追问与直接主管的沟通频次、互动质量**，信息足够后再给具体方案，避免空泛说教。

## 详细说明

- 大冰在给出「私下认错」方案前：「麻烦你给我描述一下你和你的上级主管领导……沟通的频次以及沟通的情况如何。」
- 连麦者答：每天多次汇报大小事、表格审阅、定调拿主意；领导包容、欣赏、费大力气调她过来。
- 大冰：「这些信息足够了」——随后才提认错联盟方案，并强调自己「没有资格教」，只是提醒。
