# 过去的事与当下分离

## 一句话定义

用「萨拉热窝今昔不同」隐喻：创伤中的「当年的你」与「现在的你」关联仍在，但决定量在当下，不必再拯救过去那个自己。

## 详细说明

大冰问教授聊完后是否松快——「因为你在跟陌生人聊，清楚知道这叫过去的事」。进而用国际关系举例：当下萨拉热窝与一战时期「既相同又不相同」，「并不需要今天的萨拉热窝再出来一个瓦尔特保卫他」——「当下的你并不需要用拯救当年的那个你」。
