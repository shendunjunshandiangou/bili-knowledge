# 平台承载希望值

## 一句话定义

快手、某音等互联网平台是贫寒子弟**为数不多的向上气口**；「一个人成了会抚慰一堆人」——像彩票有人中奖才有人买，平台承载的是广谱人群的**希望**。

## 详细说明

大冰论证链：

- 中国社会上升通道封闭，互联网平台是为数不多可完成「底层向上覆盖」的空间
- 若通道存在，意义「非常巨大和伟大」——叫**希望**
- 类比彩票：不可能连买 3 年无人中奖；有人中奖才继续买
- 延伸：年轻人不结婚可归因于对人生未来丧失希望、无想象力空间
- 主播速朽不是坏事——阶段性既得利益者赚钱后把空间让给下一个
