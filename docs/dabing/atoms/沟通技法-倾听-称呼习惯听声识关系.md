# 称呼习惯听声识关系

## 一句话定义

从连麦者提及亲属的措辞习惯（如是否会自然说「哥哥」）判断其叙述是否与真实关系一致。

## 详细说明

45岁女性称要聊哥哥，大冰即时打断：「你没有哥哥。有哥哥的人说话都不是这样的，一个『哥哥』都不会」。随后改按「梳理」继续，并追问寻找行动、照片等。与 [沟通技法-倾听-措辞丝滑度识背景](/dabing/atoms/沟通技法-倾听-措辞丝滑度识背景.html) 同族，本集为亲属称谓专例。
