# 自我介绍五段式模板

## 一句话定义

求职/连麦自我介绍：**直接年龄 → 籍贯到市区 → 最终学历与专业 → 工作标题列举（亮点单点）→ 现状 + 邀请对方提问**；别说「93 年」让人算。

## 详细说明

- 大冰模板：① 三十几（别说几几年）② 省市区（不必大山成长史）③ 最终学历+专业（不必说后来没干）④ 第一/二/三份工作标题列举，可圈可点的一句成绩 ⑤ 未婚/失业等现状 + 「根据我的情况您有什么要补充问的」。
- 「不是挤牙膏」——对方没耐心；今天连麦做了三次自我介绍皆发散。
