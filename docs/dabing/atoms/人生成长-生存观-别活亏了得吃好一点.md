# 别活亏了得吃好一点

## 一句话定义

**不能靠答案活着**——「无所吊谓，都得死，别活亏了」；活着本身配得上世界上的一切，从吃好喝好、买漂亮衣服等小处爱自己。

## 详细说明

姑娘找到答案、买了房后失动力，觉得生活「脏」。大冰：

- 「我们靠答案活着吗？……它是阶段性辅助你的工具，不能够靠答案活着」
- 带领复述：「无所吊谓，都得死，别活亏了，我得吃好点喝好点，这个世界上所有一切我都配得上」
- 「别人配的我都配得上」——没做错，干干净净；过去湿棉袄早已脱下
- 具体动作：挂麦后刷淘宝买漂亮衣裳、点外卖——细节上爱自己
