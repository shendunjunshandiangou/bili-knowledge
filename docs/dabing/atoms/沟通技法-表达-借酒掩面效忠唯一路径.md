# 借酒掩面效忠唯一路径

## 一句话定义

与现老大修复关系：**勿单独掏心窝**——仅大场面圆台酒至份上、自由散打时，借酒掩面、放弃尊严低声效忠；有人听见反而更好。

## 详细说明

- 连麦者问能否单独约现老大喝酒道歉
- 大冰：「千万不要骑到家里拍大腿掏心窝，人家把你当傻瘸子」
- 「圆桌子酒喝到一定份上开始自由散打……甚至不害怕旁边有人听到，挤耳朵脸先不要」
- 要碰机会，不是单独约
- 生存极重要时「锅愿意往谁身上推就往谁推」——大冰不教供人名，称现实最优解非鼓励告密
