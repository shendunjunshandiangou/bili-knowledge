# 公屏摇人匹配资源

## 一句话定义

连麦中识别本地/专业需求后，**公屏呼叫**对口人选上麦对接，并强调**未成年人须家长建联**。

## 详细说明

- 女孩英语弱、县城教育差；大冰公屏找济南高校英语/培训老师 → 留学生口语互练 + 马校长免费长期培训。
- 未成年人保护：「跟她的妈妈有建联一下，让他妈妈知道这回事」；切片发给双方。
- 「孩子向成人社会发出的要求，我们大人必须接招。」
