# 金钱观不同频比吃睡更要命

## 一句话定义

婚后基本盘含**经济生活**；南北吃睡习惯尚可迁就，**牵扯到钱**则直接威胁婚姻长期存续——家庭矛盾拆解到底多是经济账。

## 详细说明

- 「两口子吃不到一块去……还可以凑合；一个早上六点醒一个中午十二点醒还可以迁就；但是牵扯到钱，不好意思」
- 「基本盘有很重要的面叫做经济生活……不利于经济生活开展以及婚姻是否长期存续」
- 「开门七件事柴米油盐酱醋茶，哪一件不对接的钱」
- 「大部分老百姓家庭矛盾……拆解到最后不就是一笔经济账」
- 「豪门恩怨有多少跟钱没关系……九子夺嫡也是为了钱」
