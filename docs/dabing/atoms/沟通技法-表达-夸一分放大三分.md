# 夸一分放大三分

## 一句话定义

破冰聊天**赞美要放大**：「有一分放大成三分，从头夸到脚」——但须基于真实欣赏，非聊骚。

## 详细说明

- 工程师只会「嗨你好我是谁谁谁」；大冰：「夸他呀，赞美他呀，接着夸啊。」
- 确定不是聊骚、真喜欢再夸；夸完可请吃饭。
- 话题深入靠**内容**非纯话术；不了解对方时纯技术搭讪「很难往下聊」。
