# 先问报班目的再评判

## 一句话定义

批评家长育儿选择前，先问**报什么班、什么机构、主要目的是什么**——听完再讲本质，而非上来否定孩子。

## 详细说明

大冰对妈妈：

- 「孩子报的是什么班？什么样的机构？」
- 「让孩子去学小小主持人、口才，你们主要的目的是什么呢？」

妈妈答：孩子外向、话多、喜欢被关注、背稿快、喜欢舞台感。

大冰再重框：喜欢的是被关注≈被爱，不是真口才；七岁不知城市名却能长篇背稿，看不到正向加分——**先听目的，再输出专业判断**。
