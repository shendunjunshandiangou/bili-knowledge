# 考公后别用小农意识背房贷

## 一句话定义

已通过考公跳出农村路径后，不必为「房本=安全感/美好生活」急于背几十年房贷；可等订婚后再买小二手房。

## 详细说明

连麦者住单位宿舍、想买房。大冰：「农村孩子通过你的道路已经走出来……尽量把小农意识扔一扔」；商品房历史才几十年，别把对美好生活的向往置换成大负债；建议确定订婚结婚后再买，「第一套房甚至建议买二手房、小一点，生孩子前买一套就行」。父母想法可理解，但你已「蹦出来了」。
