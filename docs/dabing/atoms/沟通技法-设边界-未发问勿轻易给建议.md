# 未发问勿轻易给建议

## 一句话定义

**成年人际边界**：对方未主动发问时，实无必要以「好心」提意见或建议——尤其涉私人议题；每人有城池，Opening 须本人授权。

## 详细说明

- 城池比喻：不容他人搭梯子爬墙头；「我觉得你们城市建得不好，想进来翻新」→ 越界
- 时代变化：单位不再等于小社会，须捍卫自我边界也勿侵犯他人
- 与 [人生成长-处世-主动提建议者必是是非人](/dabing/atoms/人生成长-处世-主动提建议者必是是非人.html) 一体两面
