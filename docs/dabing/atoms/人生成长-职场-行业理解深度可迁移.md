# 行业理解深度可迁移

## 一句话定义

现代职场**岗位带着行业走**——对行业理解深度是核心可迁移技能；甲方设计+行业懂行比纯设计更溢价。

## 详细说明

- 聚生智能 PR/视频运营，大概率未来仍在科技/AI 行业
- 「行业理解的深度是绝对可迁移的……求职时的附加值」
- 「设计+行业理解赚的钱多；单干设计 vs 行业加设计 profile 不同」
- 招聘筛简历希望「人工智能干过、聚生智能方向」——「他做出来的东西老板更喜欢，开会能听懂我们说什么」
- 已选甲方路径：「你就是给一家公司服务的甲方设计」，别整天想纯设计工作室路子
