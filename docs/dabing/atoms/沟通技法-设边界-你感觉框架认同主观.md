# 你感觉框架：认同主观不辩内容

## 一句话定义

面对带有主观色彩的指责，用「你说得对」接招，并点明「只要你加上你感觉三个字，你说的一切都成立」——认同对方**主观感受的成立性**，但不辩驳、不解释具体内容。

## 详细说明

连麦者指控大冰连麦时有「高高在上的感觉」「很有优越感」。大冰回应「你说的对」，继而展开：

> 「只要你加上你感觉三个字，你说的一切都成立。因为在这个世界上，没有什么比一个人的主观个体认知更大的东西了。别人没有必要和义务去改变你。」

后续攻击（书毫无价值、表演意义大于实际、阴阳怪气）均重复同一模式：「你说得对」+ 把表述框定在「你感觉」范围内。

**操作要点**：

| 步骤 | 做法 |
| --- | --- |
| 1 | 先简短认同：「你说得对」 |
| 2 | 区分：不是认同事实内容，而是认同「站在你的角度、加上你感觉之后」的表述成立 |
| 3 | 不进入解释、辩护、反驳 |
| 4 | 必要时总括：「您在我这儿咋都对」 |

连麦者说「你并不认同我说的对」时，大冰仍坚持框架：「总而言之，您在我这儿咋都对行不行？」
