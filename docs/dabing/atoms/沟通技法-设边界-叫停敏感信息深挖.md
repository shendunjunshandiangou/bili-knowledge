# 叫停敏感信息深挖

## 一句话定义

连麦者追问老板永居、护照、海外布局等敏感信息时，明确制止：**「不该了解的，别去了解了，干好你的事就行了。」**

## 详细说明

大冰已推测公司可能有「伞」、业务非核心，连麦者继续问大领导是否拿永居/小国护照、香港分公司筹备等。

大冰回应：

- 「不该了解的，别去了解了」
- 「干好你的事就行了」
- 「归根到底这些跟你没啥关系」

收尾再宽慰「存活率可能稍长」但补「万一我说错了你别冤枉好人」——给判断同时留法律与道德余地。
