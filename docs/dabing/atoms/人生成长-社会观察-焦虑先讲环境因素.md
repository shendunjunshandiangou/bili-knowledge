# 焦虑先讲环境因素

## 一句话定义

讨论**社会群体焦虑**时，应先讲社会环境因素，不能把现象只归因为个体「不行、想要太多、不努力」——对平行维度的老百姓不公平。

## 详细说明

连麦后半段另一听众：焦虑因想要太多、送外卖忙就不焦虑等。

大冰驳：

- 焦虑是社会议题 → 先讲环境：为何 10 年前不焦虑、现在焦虑
- 年轻人不结婚 → 对未来看不到希望
- 外卖小哥可能更焦虑，只是没空间表达，会在某点情绪迸发
- 「咱们都是一般老百姓，不能以高位置俯视归因为个人问题」
- 安全感缺失 vs 权利名声财富是不同维度
