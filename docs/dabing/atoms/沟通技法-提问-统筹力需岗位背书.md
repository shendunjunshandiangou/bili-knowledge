# 统筹力需岗位背书

## 一句话定义

强调统筹力等硬实力时，需**岗位职务名称 + 完成案例**作信任背书——「嘴上自证不明智」，HR 不看标配三项。

## 详细说明

- 连麦者称客诉协同、地接资源统筹；大冰追问：原单位岗位名称？项目经理期限完成了什么？
- 「这块目前还没有」→ 话题应转向「可碰到啥学啥」，而非继续自证。
- 与「亲和力抗压是标配非硬实力」配套：统筹力比三项更显化，但仍要背书。
