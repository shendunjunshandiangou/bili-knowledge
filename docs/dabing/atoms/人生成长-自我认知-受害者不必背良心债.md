# 受害者不必背良心债

## 一句话定义

在合法边界内满足正常需求、且**未伤害他人**时，不必背负良心债或情感负担；被敲诈者更是货真价实的受害者。

## 详细说明

大冰对残障兄弟：

- 正常婚恋困难下的需求「百分之百能够理解」
- 未强迫异性、未尾随偷窥——「内心挺干净的人」
- 「你不应该背任何良心上的债呀，或者是情感上的负担」
- 作恶者在知其残障后仍死咬，比一般恶人更恶；连基本悲悯都没有的人将来没好下场
