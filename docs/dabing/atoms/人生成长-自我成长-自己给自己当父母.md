# 自己给自己当父母

## 一句话定义

成年后缺什么自己补什么——**谁告诉你不可以自己给自己当妈妈、当爸爸**；在存量历史中批判父母意义不大。

## 详细说明

- 「你缺啥，你自己直接去补，谁告诉你不可以自己给自己当妈妈、自己给自己当爸爸呀」
- 「你希望拥有什么样的爸爸妈妈，你可以自己先成为自己，先成为了将来再去给你的孩子成为」
- 「我们再去在存量历史当中批判他们，意义已经不大了，怎么样对你最有利」
