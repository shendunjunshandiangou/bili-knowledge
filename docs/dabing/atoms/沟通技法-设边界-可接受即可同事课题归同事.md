# 可接受即可同事课题归同事

## 一句话定义

同事代你拒绝额外工作时：你若**可接受**加班做完就没问题；同事接受不了是他的课题——「做你自己就好」。

## 详细说明

分公司态度好却压活；连麦者因好态度不好意思，同事坚定推回。

大冰：

- 「如果你的同事不做这个反应……你可接受还是不可接受？」→ 可接受就会加班做
- 「那可接受就没问题啊，没毛病啊；你可接受你的同事接受不了，那是他的课题，他的议题」
- 「做你自己就好，即便加班但是你可接受，那可接受是可接受的」
