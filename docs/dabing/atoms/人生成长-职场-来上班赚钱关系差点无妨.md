# 来上班赚钱关系差点无妨

## 一句话定义

同事站队、关系差若**不影响职务稳定与饭碗**，叫小事——「我是来赚钱上班的，关系不好会不会丢饭碗？不会就别消耗心神。」

## 详细说明

- 连麦者担心小团体、组长与 C 位关系、升职被说坏话
- 大冰：无末位淘汰→「好好上你的班，其他事不重要」
- 核心对冲句：「我关系跟你稍微差一点……我是来干嘛的？来赚钱上班。会不会影响丢掉饭碗？不会。」
- 升职加薪主因：「工作做得好」——「不要把领导智商想太低」
- 组长对 C 位「关系好」也不必 worry——工作完成度与运转对其更重要
