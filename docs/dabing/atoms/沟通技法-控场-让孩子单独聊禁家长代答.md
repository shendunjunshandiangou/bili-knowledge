# 让孩子单独聊禁家长代答

## 一句话定义

与儿童连麦时明确「咱俩的事跟妈妈没关系」，请家长退到一边，避免代答掩盖孩子表达。

## 详细说明

小英连麦时妈妈代答爱莎公主，大冰：「你让妈妈先上一边去，咱俩聊天；她不要代替你回答」。重复「让妈妈上一边坐着去」，再与孩子直接讨论作文、读书、南昌、面鱼等。
