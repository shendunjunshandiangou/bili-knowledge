# 攻击力应向外不向内

## 一句话定义

有自我的人需要**对外攻击力**；对外不展现就会对内攻击自己——妥协、无法拒绝即「对自己使劲」。

## 详细说明

大冰对创伤后过度妥协的上海姑娘：

- 「人呐有自我的人是一定需要有对外的攻击力的，这个攻击力如果对外不展露展现的话，就会对内冲着自己使劲」
- 对外攻击力可以是：爱别人、主动链接、积极回应挑衅——「帮别人也是一种攻击力」
- 「你一触即退，退回来之后这个东西不是安静在这了，就会攻击你自己」

与心理咨询互补：心理医生梳理，大冰给具体动作通道。
