# 轮子先滚健身换环境

## 一句话定义

推动躺平者：**健身**（可初期陪伴，建立话题与外在约束）+ **换城市**（存量环境无可能性）——轮子先滚起来，不管往哪滚，总比不动强。

## 详细说明

- 家里健身设备 ≠ 真练（跑步机摆拍效应）
- 换城：对可触小世界不满 ≠ 对世界不满；新环境 = 可能性（酒吧婚恋类比）
- 可选方向：滑雪教练（连麦者有证，体重限制另论）
- 趁父亲给钱换城，主要任务健身 + 该玩玩
