# 不辩论原则

## 一句话定义

当对方进入**辩护模式**（如「我家庭很幸福」）时，明确「咱俩不辩论好不好」，**不否定对方感受**，把论点改述为更中性的「路径/方式或许有问题」——避免陷入事实争执，保持引导方向。

## 详细说明

大冰指出「您从小获得爱的方式就有问题」后，连麦者辩护：「我的家庭其实很幸福，有一个很大的家庭。」

大冰回应：

- **不辩论**：「我没说您过得不幸福，咱俩不辩论好不好」
- **改述论点**：「获得爱的路径及方式或许是有问题的」
- **展开而不纠缠**：不是「你不幸福」，而是「不是表现好、乖巧懂事才配得到爱；爱不应通过交换行为博得」→ 自我缺失、打压自己

适用：

- 对方把批评听成对整个人生/家庭的否定
- 需要继续推进重构，但不能让对方关闭倾听
- 与「镜像反馈」不同：这里是**主动降冲突**，不是指出对抗姿态
