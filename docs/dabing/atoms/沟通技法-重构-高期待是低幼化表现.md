# 高期待是低幼化表现

## 一句话定义

对领导、伴侣持续**高期待**（求承载、求包容、求对等回报），在职场与感情中反复踩同一坑——这是**偏低幼**的模式，不指向成年人的成熟度与独立性。

## 详细说明

- 连麦者自认独立，实则未经得起现实考验；对男友、对主管都是高期待
- 逻辑原点：「我的高期待没有问题」→ 大冰：这种模式偏低幼
- 「人是社会关系的总称……老在同一个坑里踩」——可统一理解为高期待
