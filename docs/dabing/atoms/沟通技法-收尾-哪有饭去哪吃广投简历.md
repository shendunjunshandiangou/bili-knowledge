# 哪有饭去哪吃广投简历

## 一句话定义

求职**勿锁地域**——「哪有饭去哪儿吃，广投简历，拼基数」；排序：本专业相关→薪资与成长→不相关但可活。

## 详细说明

- 22 岁大数据应届只投重庆；大冰：「卡住你的是地域……全中国这么大。」
- 父亲也说「世界这么大，你想怎样走自己去走」。
- 排序：先跟本专业相关且指向过得去薪资与可能性；找不到再取舍到不相关岗位。
- 「二季度末了，抓紧时间满世界撒简历。」
