# 轨迹分叉减少旧链接

## 一句话定义

人生轨迹分叉后，与「过得更好」的发小同学**不必维持太多横向联系**——欢聚易引发不对等比较，对你多是情绪负面叠加。

## 详细说明

- 「如无必要，其实不需要有太多这样的横向联系。」
- 人家只展示过得好的一面；潜意识也在比较「别人低我高」获自我认可。
- 可真心为友高兴，但「轨迹不是多么重叠、生命链接不紧密时，不必太多联络。」
- 建议：建筑当下人生阶段新的社交圈、人际网络。
