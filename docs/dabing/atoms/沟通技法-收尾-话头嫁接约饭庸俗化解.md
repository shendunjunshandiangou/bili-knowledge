# 话头嫁接约饭庸俗化解

## 一句话定义

用刚结束的第三方对话（连麦内容）作由头 reopen 话题，约吃饭后以亲密行动而非说理化解情侣小矛盾。

## 详细说明

大冰给具体脚本：挂麦后跟女友分享「老狗逼」关于不买房的理论 → 约干锅/米线 → 「吃饱后趁意志力薄弱该啃就啃，干点庸俗的事，这一页就翻过去了」。强调「怕冷战，不怕吵架」「大部分情况都可以被睡服/说服」。
