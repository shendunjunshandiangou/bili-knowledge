# 默则危拿话语权

## 一句话定义

面对罗嗦或难主导的对象，用**长段沉默**（「默则危」）听其说完，从而夺取话题主导权。

## 详细说明

- 「最长我可以3分钟一个字儿不吭，我就听你说」
- 原理：压着别人话讲的人易处下位；要梳理/help对方时需拿主动权
- 与「少说」配合：对方说时少应声（不嗯嗯附和）
