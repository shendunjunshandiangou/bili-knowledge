# 家乡共情致谢

## 一句话定义

当第三方善意惠及与自己有**地域或身份关联**的人时，点明共同身份并**代为致谢**，把感谢从个人层面扩展到群体共情。

## 详细说明

张雪峰汇报对烟台残疾女孩的持续帮扶（山东组关注、免费教辅、学费就业帮扶等）后，大冰：

- 纠正「汇报」一词，表达意外与感谢。
- 自报身份：「我是山东烟台人，那是我们家乡的小孩，我替我们家乡的孩子谢谢你。」

**操作要点**：

1. 找到与对方行动的交集（同乡、同校、同行业等）。
2. 用「替……谢谢你」把个人感谢升级为群体代言。
3. 避免空泛客套，紧扣具体被帮助的对象（家乡的孩子）。
