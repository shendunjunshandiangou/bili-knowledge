# 连麦阐述限时约定

## 一句话定义

连麦者表达无序时，**约定阐述 5 分钟内完成**，且「但是」转折频次控制在 **3 回以内**，防发散占麦。

## 详细说明

- 大冰对第二段连麦：「阐述部分尽量控制在 5 分钟之内完成；如果你要说，尽量把『但是』的频次控制在三回以内。」
- 连麦者仍说不清 → 「让你去找劲松你又不去……连一个问句你都不知道怎么组织。」
