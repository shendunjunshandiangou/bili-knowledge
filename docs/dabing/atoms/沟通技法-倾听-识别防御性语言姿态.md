# 识别防御性语言姿态

## 一句话定义

在连麦或对话初期，留意对方**前几句**是否带有礼貌外壳下的对抗/防御——往往对方自己未意识到，但陌生人（或 HR）能感知。

## 详细说明

大冰对连麦者的观察：头六句话里**对抗性很强**，是「天然的防御动作」——本心并非不礼貌，但声音、语言姿态会被接收为凶、难管、有对抗性。

连带建议：

- 面试前用小镜子看**当时的脸和表情**（自以为正常，在别人眼里可能凶或不好管）
- 做**表情管理**和**语言姿态软化**

与父亲专制、严肃的成长环境关联：权威映射未处理好，会在面对 HR/领导时走「卑微」或「冒犯被挑战」两个极端。
