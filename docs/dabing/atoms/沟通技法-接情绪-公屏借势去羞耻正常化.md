# 公屏借势去羞耻正常化

## 一句话定义

当连麦者因私密遭遇（如被敲诈视频）感到「脏」时，**让直播间公屏集体表态「理解」**，用多数人的正常化消解个体羞耻。

## 详细说明

残障兄弟遭视频敲诈后觉得自己「脏了」。大冰：

- 「如果一半是男人的话，找不出几个干净的」
- 「现在看一眼公屏……老爷们儿都留个言，能不能理解，理解一下咱们这个残疾人兄弟，正常男人的正常需求，心里边有这样的需求，这不是一个错的，它永远不是错」
- 进一步：没强迫异性、没做邪恶之事，「内心挺干净的人」
- 「这根刺……平台上八九十%的男人都别活了」——相对化羞耻
