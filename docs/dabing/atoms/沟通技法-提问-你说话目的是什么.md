# 你说话目的是什么

## 一句话定义

连麦梳理时**反复追问**「你跟我讲这些话的目的是什么」——不听表面内容，听背后真实的人与真实诉求（常是求抚慰、拒被切顶）。

## 详细说明

- 「我不看你说了啥，听你背后的话……尽量看背后真实的那个人」
- 连麦者目的：诉怨恨、证自己问心无愧、或求抚慰——说着说着逻辑混乱
- 发现求抚慰时尝试掰话题，逻辑不够则暴露**阻断被扒开**
- 与 [沟通技法-控场-阻断求抚慰](/dabing/atoms/沟通技法-控场-阻断求抚慰.html) 配合使用
