# 正规出家非逃避避世

## 一句话定义

正规出家是匹配「弘法利生」的严谨工作，极苦极自律；受影视影响以为受打击出家即解脱，是误解——**为逃避而出家不是正道，也出不成**。

## 详细说明

- 大冰列举门槛：背楞言、每日凌晨三点半四点起床、三坛大戒「不是那么好受的」
- 工作内容四字：「弘法利生」——需匹配职业属性、要求与职业道德（指正儿八经一批）
- 影视误导：「受了重大打击人出家就斩断解脱——不是那道理，两码事」
- 逃避特征：想风吹不着雨淋不着、不累不饿滋润过活——「我不会走磨走这条道，走的那条道也未必是正道哦」
- 提醒：担心「魔子魔孙披着袈裟坏法乱道」；「小庙为了靠劳的心思都不能有，但凡有这心思道就歪」
- 替代：「不出家也能是修行」；大乘本有居士，佛教最初也在家人中
