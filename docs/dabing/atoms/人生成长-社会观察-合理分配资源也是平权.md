# 合理分配资源也是平权

## 一句话定义

有能力者自愿向主动求学的未成年人**免费提供优质教育资源**，是社会资源合理分配，「某种意义上的一种平权」。

## 详细说明

- 马校长愿免费长期教女孩英语；大冰强调机会是女孩自己连麦博得的，与家庭经济无关。
- 「一切皆合理……我们期待着在您手里边把这个孩子教得英语水平突飞猛进。」
- 同时要求机构与女孩母亲建联，流程周密。
