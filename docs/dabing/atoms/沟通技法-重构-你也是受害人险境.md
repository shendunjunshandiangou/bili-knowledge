# 你也是受害人险境

## 一句话定义

当倾诉者聚焦他人受害时，点出**叙述者本人也多次处于险境**，只是运气稍好——重构其受害人身份。

## 详细说明

- 「不是光你妹妹受到了侵害——你每一次也都陪着她站在悬崖边上，你只不过运气好一点而已」
- 「我只不过是他未遂而已」——叙述者可能未意识到自身受害程度
- 作用： legitimizing 其切割/自保决定，减少「我没资格」的 guilt
