# 做人是做事的一部分

## 一句话定义

职场协同中**不要把做人与做事切割**——「你做人本身就是你做事的一部分」；你不是单位外包商，也得维护渠道。

## 详细说明

连麦者问：「难道把事做好还不行，还得先学会把人围好了？」

大冰：

- 「你做人本身就是你做事的一部分呀……谁告诉你们的要把做人和做事切割分开来看？」
- 「我们是以活生生的人和其他人协同作战、团队合作，你本身做人就是你做事的一部分」
- 连麦者像不适合上班、可多琢磨独立创业——因长期人际与边界问题
