# 五句话剥离自我攻击

## 一句话定义

对自我攻击型决策者，用**固定五句话**剥离归因、指向环境、给路径，再解释测试逻辑。

## 详细说明

五句话：
1. 摒弃「是我做错了/不够努力/状态不好」等自我攻击
2. 状态不好主因是环境（学业压力≤30%，家庭因素更大）
3. 尽快脱离现有环境去新高校
4. 二本不决定一生，也可考研211/985
5. （测试说明）喝水是观察能否从高压阈值穿刺
