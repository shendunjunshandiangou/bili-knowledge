# 点破反讽就是骂在脸上

## 一句话定义

连麦者用看似「看透世界」的概括（如「没有好男人好女人，看谁降得住谁」）反击时，直接说：**不是反讽，是骂在你脸上**——因固化观念已害过自己。

## 详细说明

女生：「这个世界上没有真正所谓的好男人好女人，就是看谁降得住谁。」

大冰：

- 「你对这个世界足够了解？……理论依据数据支撑这块链路都打通了？」（反讽其样本不足）
- 女生：「不要这么说，我要这样说……」
- 大冰：「他不仅是反讽啊，这个叫做他骂在你脸上」
- 理由：「你给自己的固化植入的观点已经害过你了」，如「别人不能跟我提分手」——「你能扔掉多少扔掉多少」
- 区分：「不要拿情绪覆盖观点立场……这只是情绪而已」
