# 越级诉苦极傻

## 一句话定义

为解释委屈**越级联系一把手**，一把手关注团队稳定大于你个体委不委屈——极傻；且易成前后老大之间的**枪与棋子**。

## 详细说明

- 连麦者信同事传话「拉帮结派」，问前老大是否该向老板解释
- 越级打电话→老板要供人名→拒说→微信被删
- 大冰：「老板永远关注团队内部运作稳定性，大于关注你个体委不委屈」
- 「从这一刻你变成前老大和现老大之间的枪和棋子」
- 传话者：「非者必是是非人」；递话朋友可能变下阶段对立面
