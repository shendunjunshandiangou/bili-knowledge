# 法无禁止与公序良俗双框架

## 一句话定义

讨论主播盈利、网红现象时，先立**双前置条件**：「法无禁止即可为」+「公序良俗不破坏」，在此罩子下再展开论述。

## 详细说明

大冰回答「直播怎么赚钱」前的 slogan：

1. **法无禁止即可为**——衡量一切主播盈利模式的前置条件
2. **公序良俗不破坏**——再推一步的前提
3. 「在这两个罩子之下，你干啥都行」

之后才能谈：不反对带货、平台是上升通道、包容低俗等——避免听众以为在鼓励违法或突破底线。
