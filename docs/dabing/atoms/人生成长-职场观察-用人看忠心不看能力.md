# 用人看忠心不看能力

## 一句话定义

某些有体制内背景、业务可能是「壳」的公司里，老板用人大概率看**谁比较忠心**，而非谁能力多强或资历多深。

## 详细说明

大冰对女硕士小领导局面的解读：

- 小领导技术可以但管人不行，被 CEO 头衔吊着、又被同级大将制衡——「可能是人家基于自己角度的缜密选择」
- 「小领导的野心有点明显」
- 这份工作：能干就干，薪酬亏不着就干着；这种公司不亏、有伞，不易轻易死掉

帮助连麦者停止用「能力梯队」框架理解公司，减少内耗。
