# 如果我是你降资格

## 一句话定义

给强烈人生/财务建议前，反复声明**「我不能教导/指导，没那个资格，如果我是你的话……」**——用假设降权威与责任。

## 详细说明

本集全文高频：

- 「我不能教导你，也不能指导你，我没那个资格哈，我就说如果我是你的话」
- 「我真没法来劝你啥，我也不专业，我只是把我带入你我会怎么干」
- 「我只是在假设，如果我是你」

使卖房、提收入等重建议可被接受，听众知是参考非命令。
