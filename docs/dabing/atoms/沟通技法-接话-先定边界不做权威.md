# 先定边界不做权威

## 一句话定义

被当作行业专家请教时， upfront 声明：**「我不做主播/不是运营，咱俩只是聊聊天」**，降低权威预期与后续责任。

## 详细说明

79 年软件从业者连麦请教快手主播盈利模式、平台良性发展。

大冰前置：

- 「你找我有个啥事」
- 对方夸逻辑思维、学识 → 「我不做主播，我也不是这个，您看我有没有粉丝啥的，咱俩只是聊聊天」
- 后文再答「具体怎么做轮不到咱们想」「我不是运营」「有些话不适合在这上面聊」

形成一致的「非权威聊天」定位，可长谈观点而不必给实操方案。
