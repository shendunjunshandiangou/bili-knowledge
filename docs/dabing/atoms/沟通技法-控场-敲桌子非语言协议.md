# 敲桌子非语言协议

## 一句话定义

对不愿开口的连麦者，约定敲击次数代替语言（听见/名字字数/授权），降低表达压力。

## 详细说明

休学少女几乎不说话。大冰：「不需要非得说话」；请敲桌确认听见；名字几个字敲几下；「我捏鸭子你敲桌=授权」；认同敲一下、反对敲两下。并解释：「小朋友最大，不必按迂腐规则有来有往」。
