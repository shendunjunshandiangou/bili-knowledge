# 只跟昨天比

## 一句话定义

求职成长期**勿横向比**大冰或同龄人——「唯一要比的是迈出去之前和之后的自己：比昨天强一点，正反馈来得快。」

## 详细说明

- 跟大冰比：差几十年阅历，「你到我这岁数说不定还比我强」——无意义。
- 跟同龄人比：易生自我攻击，拖后腿。
- 「渴望成功」本身没问题；比较产生波动、自损、扣帽子时「迅速停止」。
- 有限对比：「我都不要求比我去年强，比我昨天强就行。」
