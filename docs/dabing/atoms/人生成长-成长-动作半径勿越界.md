# 动作半径勿越界

## 一句话定义

领导、同事、伴侣各有**合理动作半径**——超出半径叫**越界**，不是「 merely 高期待」；婚恋里对年轻人过高期待也不公平（逼女友整容类比）。

## 详细说明

- 「你有领导，领导也有领导……超出半径叫越界，并不是高期待而已、没有原罪——你越界了」
- 同事比连麦者更会「忍耐」，因知半径在哪
- 婚恋：选了对方又过高期待 = 不公平
