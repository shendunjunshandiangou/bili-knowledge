# 拒替连麦者承担敏感风险

## 一句话定义

连麦者自我评估话题敏感却仍要求主播「替其承诺承担风险」时，点破**风险转嫁**并拒绝——「要不然你就不说那个话，您直接说就可以了」。

## 详细说明

连麦者：「不确定会不会涉及敏感词……我可以说我的经历吗？」

大冰解读语言模式：

- 「我知道我的发言有风险，但我希望你主动替我来承诺愿意承担风险」
- 若自己判断敏感，应换直播间或更合适平台，「不可能把风险这么快速地转嫁让我们来承担」
- 强调敏感词=已完成自我评估；下意识动作=希望主播预先背书——「我不太愿意的」
- **仅未成年人**一种情况下愿意替承担
