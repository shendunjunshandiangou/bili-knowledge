# 平台弹窗及时切断连麦

## 一句话定义

平台规则弹窗提示违规风险时，**立刻关窗并断开连麦**，同时保持对连麦者的尊重表述。

## 详细说明

与小杨连麦时：

- 「平台有平台的规则，我这边弹窗了……和您连麦容易让我炸号了」
- 「我需要把这个窗关上，然后下一步关掉您的连麦」
- 同时：「我对您抱有最大的尊敬，但是不好意思……你得提供客观依据」
