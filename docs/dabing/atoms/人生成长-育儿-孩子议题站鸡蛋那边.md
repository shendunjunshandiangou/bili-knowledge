# 孩子议题站鸡蛋那边

## 一句话定义

直播间涉及**你家孩子**时，立场天然站最弱势者——「石头和鸡蛋里站到鸡蛋那边」；大人议题往后稍，孩子是孩子里的 king。

## 详细说明

大冰一年多反复强调：

- 话题涉及孩子 → 重心**完全围绕孩子**，大人议题都要往后稍
- 判断关系图谱最弱势者是不是孩子；是孩子则站孩子
- 「在我的直播间，没有任何议题比一个小孩子更重要」
- 延伸：真希望小朋友过得好，尤其小小孩；大人「就这样吧」，心力优先给小孩
