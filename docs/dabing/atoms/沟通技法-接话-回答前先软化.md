# 回答前先软化（表达认可建立链接）

## 一句话定义

在正式回答对方问题之前，先加一句**对问题的认可或接收**，再进入内容——俗称「软化动作」，用于建立语言链接，而非因为问题本身有多深刻。

## 详细说明

场景：HR 问「中文专业为什么选影视广告行业？」

大冰示范：不管后面答什么，**先追一句**——「老师你问的这个问题，以前没有人问过我」「你问了一个很深入的问题」，然后再说「我来告诉你我的想法」。

大冰自评：前面这句话「屁用没有」、没有含金量，但作用是：

- **表达认可**，让对方感到被听到
- **建立语言链接**——「您说的我接收到了」
- 把面试官当**人**，而不只是考核机器

这是社会化生存中的**标配互动动作**，不是虚伪奉承。
