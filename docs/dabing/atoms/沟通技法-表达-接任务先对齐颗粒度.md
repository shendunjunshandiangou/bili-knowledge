# 接任务先对齐颗粒度

## 一句话定义

任务下达时**第一时间**说明执行计划、需对方拍板/协调/资源处——「对齐颗粒度」，别瞎答应再临时抓瞎。

## 详细说明

- 领导临时加任务、一天要出视频——连麦者不敢约时间
- 大冰：「约半小时汇报方向」——「不存在敢不敢，这是工作流」
- 接任务话术：「没问题我马上去做，两点钟出初步想法和模板，需要跟您商量一下」
- 「交代任务下来就要把任务怎么做、你的计划、需要它的地方提出来」
- 连麦类比：「你问他啥明白啥也不明白，瞎答应什么呀」——先想需不需资源、拍板、协调
