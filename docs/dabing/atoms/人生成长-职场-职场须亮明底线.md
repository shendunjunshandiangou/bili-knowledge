# 职场须亮明底线

## 一句话定义

应对职场恶意前，须先**亮明底线**——哪些话绝对不能讲、哪些行为算越界；缺底线则易被拿捏。

## 详细说明

- 大冰第二点：「没有一个清晰的底线让出来……哪些东西越界了，这个是要亮明的」
- 与 [人生成长-职场-开脱或斗争无第三选项](/dabing/atoms/人生成长-职场-开脱或斗争无第三选项.html) 并列——要么战斗要么开脱，都需边界清晰
- 连麦者县城长大、有效积累少，更像女生宿舍矛盾重演
