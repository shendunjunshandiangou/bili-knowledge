# 不灌鸡汤扎心实话

## 一句话定义

拒绝「努力加油」式片汤话，用**扎心但真实**的分析替代情绪价值，确认对方要的是解决而非哄。

## 详细说明

- 「我们完全可以连上麦把你哄得很好……但这种片汤话有用吗？明天想到主要议题得到推进了吗？」
- 「您连这个麦只是为了要点情绪价值吗？既然不是的话，话扎心你得听」
- 「我不给你们盲目打鸡血，见到踩坑浪费时间的孩子太多了」
