# 月薪3w裸辞做自媒体

**摘要**：30K投放经理离职全职自媒体，6–7万粉但月入1k，二创混剪与原创纠结、负债30万。大冰纠偏：**做个人IP ≠ 原创/二创之争**；别等爆款牵着走；需稳定输出人格而非追10万赞。信息齐备后「直接聊、不上课式」开讲。

## 内容轨 · 人生观点

- 个人IP 是长期人格积累，不是单次爆款逻辑。

## 技法轨 · 沟通拆解

- **接话**：对方信息一次性给齐，「喜欢跟爽快人聊天」→ 直入主题。
- **重构**：从「二创还是原创」重框为「个人IP路径」。

## 文章正文

一位做过九年互联网广告投放的连麦者介绍自己：山西祖籍、北京户口、湖南科技大学应用心理学本科，16年毕业，游戏行业4年、教育行业5年，投放营销预算超10亿元，月薪3万，今年6月裸辞全职做自媒体。5月21日开始发视频，全网约6到7万粉丝，赛道是冷知识、奇闻异事、泛知识，多为二创混剪，少量原创口播。最近两三周流量很差，二创像营销号，心里过不去，想做原创但爆款率低，月入仅1000多元；打折后还输了30多万，现负债约10万，体重从150斤涨到198斤。6月中旬在北医六院确诊重度抑郁症，有一点双向情感障碍。

大冰判断：问题不在二创还是原创，而在应抓紧时间做个人IP。做号会被爆款牵着鼻子走；应找本子、找两三个好友头脑风暴，把长板、短板、人生阅历、职业行业、技能全部列出来，看能提供什么内容，且必须跟"你就是这样的人"契合。他用赵雷举例：歌适合他唱，不是天下第一好，而是完全成立的对标个人IP。

连麦者说契机是前下属"疯狂大兵哥"（快手两百万粉）做短视频成功，他交6000元学费去山东学了一个月；女朋友建议先按对方路径验证闭环再转型。他特别想做游戏类内容，与女友有分歧。大冰说copy别人几乎无法重复或超越；样本案例太小，像周围有人炒股成功你去学，他不认可这种学习价值。应让大家对"你这个人"感兴趣，而非只对账号感兴趣——"是要做人啊"。他举吃鸡账号"队长"带剧情、随机组小姐姐的例子：看完后对人不只对节目感兴趣。

关于收入下降是否坚持半年，大冰说"耐不住寂寞别来做这个事"，运气成分占比不小。但连麦者已被动告别上一段人生，实际在做的是"接下来该怎么活"。问存款够吃多久，答一年多。大冰说：若笃定不想回去打工，别给自己设时间线，"百米冲刺容不得回头"，带破釜沉舟的劲反而容易成事；最差就是重新打工，"又掉不了地上"。他提醒别着急，一乱就浪费时间、船不停换方向——连麦者承认选题最近很乱。
