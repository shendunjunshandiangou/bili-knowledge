# 职场要带点「攻击性」：别总客气，分档设底线

<BiliPlayer :bvid="'BV1RAkABKEr8'" />


**摘要**：北京职场人自认讨好型、老好人，与谁都淡淡、易被欺。大冰：你不是懦弱，是**过度琢磨可被感知**；**客气在职场常被读作示弱**；人人需「加引号攻击性」——自我、原则底线；**老好人包容度反而更低**；解法：**分档对人**，该直则直、该亮底线则反弹；平级可翻脸，**别跟发薪人翻脸**；勿把朋友的朋友都发展成哥们——「如无必要，勿增实体」。

## 内容轨 · 人生观点

- **攻击性定义**：「不是侵略性，是自我、原则底线」；人人需一定攻击性，否则易被排异为办公室末尾。
- **客气=示弱**：「客气在很多人眼里是一种示弱，因为你没有攻击性。」
- **老好人悖论**：说话难听有标签的人包容度高；老好人「底线设得非常低、自我没那么强」，稍不符合预期就被说「你怎么这样」。
- **琢磨超阈值**：开口前过度美化、琢磨，别人可感知→升起防备心。
- **朋友≠职场**：朋友有陪伴供需与沉没成本；职场是另一套逻辑。
- **朋友的朋友**：「不是病态，叫很二逼」——大概率对方把你当资源方；「如无必要，勿增实体」。
- **专注当下**：此性格「专注当下比学多技能更好」——目标锁定更高年薪。

## 技法轨 · 沟通拆解

### 互动场景

在京工作，自认讨好、想与所有人搞好关系包括朋友的朋友；感觉被欺；对心理学感兴趣、女友做特殊教育；可能有工作调动。

### 大冰的回应序列

1. **重框性格**：「你不是懦弱软弱，有的是思量、小算计——周围人可感知。」
2. **琢磨诊断**：开口前过度美化→超正常琢磨阈值→别人防备。
3. **分档方案**：画人物谱系分档，「该摔贫摔贫、该生平生平」——勿把现状态对标所有人。
4. **攻击性论述**：人人需原则底线型攻击性；低于众人预期→沦为被排异对象。
5. **老好人对比**：有「说话难听」标签者包容度高 vs 老好人稍越界就被否。
6. **预警机制**：到程度该反弹、亮明底线；新工作/调动=重新开始，「现在设立不算晚」。
7. **翻脸边界**：平级被攻击可翻脸；「跟发薪人轻易翻啥脸」。
8. **朋友的朋友**：不必全发展哥们；熟人即可，「别给自己惹麻烦」。
9. **收尾**：锁定更高年薪，勿盲目多学技能。

## 文章正文

一位连麦者说自己在职场性格软弱、想跟所有人搞好关系，包括朋友的朋友、上下级同事，结果关系都平平淡淡，感觉被人"欺"。大冰说：不是懦弱也不是软弱，有的是思量、考虑和偶尔的小"算计"（加引号），周围人可感知——微表情、姿态、别人聊天时的反应都能被解读；抠搜、老在乎存款等会慢慢塑造一个人，自己看不见别人看得见，"别人知道你在琢磨"。

开口前过度琢磨、美化话术，话到嗓子眼说出来与预期不符——议题好解决：找本子画人物谱系图，把职场沟通对象分档，只对其中一两档保持现在的琢磨状态，其他的"该摔贫摔贫，该生平生平"，要分人。

大冰展开"攻击性"概念：客气在浅表是礼貌、指向尊重，但在很多人眼里和一定量的交流过程中是一种示弱——"因为你没有攻击性"。每个人都需要具备一定的"攻击性"（加引号），不是侵略性，而是自我、原则底线。攻击性低于众人预期水平线，容易沦为办公室末尾被共同瞧不上、排斥的人——几乎每间办公室都有这样一个人，说不清做错啥但所有人排斥他。

他观察到：被贴"说话难听、情绪不稳定"标签的人，大家包容度反而高；老好人说一点不符合预期的话，大家"受不了"——因为感知到你底线设得很低、自我不强、没有太多"攻击性"。所以要给自己设框架，玩笑有度，超出度要表达不舒服；分档后该反弹反弹、该亮明底线亮明底线，现在设立不算晚。平级同事若攻击可以翻脸——"都是爹生妈养的"，侵犯底线、自我必须第一时间站出来。但跟直接领导、发薪人不要轻易翻脸。

朋友与职场是两码事：朋友有陪伴需求和沉没成本。他想把朋友的朋友都发展成哥们儿，大冰说"不是病态，叫很二逼"——朋友介绍的本地朋友大概率没你高，把你当有资源的一方希望将来效劳，而你没看清交换点，"如无必要不做实体的事"。多点熟人不是坏事，求医问药有用，但不宜走太近——在北京的普通孩子，在本乡本土别人觉得有本事，自己知道几斤几两，"别给自己惹麻烦"。

他女朋友从事特殊教育，连麦者也对心理学感兴趣，大冰说已上轨道有前景就专注当下工作；若不创业，把目标锁定更高年薪的工作，"专注当下比着手选择更好"——今天聊的只针对他，别人听了别往身上套。
