# 职场大忌：不要在单位聊私事

<BiliPlayer :bvid="'BV1VvkkBmEoi'" />


**摘要**：23 岁郑州省直事业单位 211 女，在职硕士，恋爱不顺被同事灌输「25 岁前可挑、过了被挑」。大冰：23 岁别急，恋爱拼**见人的基数**；同事多是闲磨嘴皮子的**人生导师**，别信。由此展开：**不要在单位跟任何人聊私事**；「来说是非者必是是非人」扩大化——**未主动发问却主动提建议者，必是是非人**；每人有城池，勿让人爬墙头；闲聊勿把话题拐到自己身上——「开始谈论你，你就已经是一盘菜了」。

## 内容轨 · 人生观点

- **恋爱拼基数**：见的人越多，概率越大——菜单只有四道菜选不出怪饭店，世界菜谱极多。
- **同事人生导师不可信**：事业单位闲时间多、无有价值输出，拿结婚生孩子俯视他人；「他爱你吗？份子钱愿随 2000 吗？」
- **单位勿聊私事**：婚恋、家庭、想不想结婚——尽量别让人知道。
- **主动建议者 = 是非人**：时代扩大化旧语——未主动发问却「好心建议」者，尤其涉私人议题，必是是非人。
- **人际边界**：每人有城池国度，Opening 城门须本人授权；勿搭梯子爬墙头。
- **别成菜**：话题拐到自己身上 = 成一盘菜；可八卦不在场的第三者，勿八卦自己。

## 技法轨 · 沟通拆解

### 互动场景

23 岁省直事业单位女性，恋爱不顺，受同事「25 岁分水岭」话术影响；问正确观念与是否该有动作。

### 大冰的回应序列

1. **降焦虑**：23 岁经验少，火候未到；32 也不用急。
2. **给动作**：皮肤身材管理 + 捕捉自由恋爱 + **工会相亲别浪费**（社会属性初筛）。
3. **拆同事话术**：开发商「精装拎包入住」类比——问对方是否真爱你。
4. **扩题私事边界**：「以后长点心，不要在单位聊私事」。
5. **扩大化是非人**：主动建议 = 刷存在感；价值感低者逮机会当导师。
6. **城池比喻**：勿攻伐他人城池；集体主义时代单位即社会，今天须捍卫边界。
7. **收尾**：别成菜；23 岁还小。
