# 在职场被人当软柿子，要如何改变成长？

<BiliPlayer :bvid="'BV18DK46yEpu'" />


**摘要**：与本集「不当职场软柿子」为同一连麦切片（BV1crKH6jERu）。山东 32 岁事业编代理科长遭同事架空、甩锅、说「没底气」，曾办公室崩溃大哭。大冰：**尽量别当软柿子**，捏完还会捏；**办公室发疯不是自救，反而坐实软且烂**。正解是冷静观察、**会上有的放矢扳他**，点事实疏漏后接「为工作负责、帮你成长」；德服不够需立威；向主管**对称汇报**阻碍；**私下向领导认错**换联盟；分清**大小王**——主持工作谁说了算；成长方向：**业务底线 + 及格政治意识**。

## 内容轨 · 人生观点

- **软柿子定律**：「软柿子一旦当了，是捏不完的，一直把你捏呲了捏出汤来。」
- **发疯≠强硬**：办公室发疯不仅坐实软柿子，还是「有点烂了的软柿子」。
- **德服不够**：事业单位打开局面往往不能只靠德服人；立威后仍不配合，就是工作态度问题。
- **业务 + 政治**：优秀业务干部也要提升「加引号的政治」意识做自我保护，目的不是私利，而是工作绩效、服务弱势群体不出疏漏。
- **成长代价**：「责任越大压力越大，一切指向成长——没有哪种成长不痛苦。」

## 技法轨 · 沟通拆解

### 互动场景

32 岁山东小城事业编，科室代理主持工作；前任科长累垮调离，原业务骨干兼科长后崩溃拒任，连麦者被架上代理科长；同事防御姿态、抛棘手问题、说她没底气；曾天天加班哭、办公室发疯自救。

### 大冰的回应序列

1. **定调**：「尽量别当软柿子」——一旦当了会被一直捏。
2. **纠偏自救**：发疯不是变强，是更证软且烂；应「做好战斗准备，认真研究对手」。
3. **扳他话术**：冷静观察、有的放矢，**最好在会上**点行为事实/不作为/疏漏，追一句：「我是为了对咱们的工作负责，也是帮助同志你更好成长，你能理解我的良苦用心吗？」
4. **立威逻辑**：德服不够需威力；收拾过后仍不配合=态度问题；矛盾公开、主管知情；自己零疏漏 + 积极推进需配合的工作。
5. **中心工作**：一切动作为服务好人民群众（弱势群体）。
6. **向上对称**：向主管汇报时对称造成工作阻碍的因素——前提是告知领导「不是添麻烦，是为工作不出疏漏」。
7. **认错联盟**：私下向专门调她来的领导承认哭/发飙错误，承诺不再犯；顺势说急着想干好活、同事使绊子，请领导教应对；换领导安抚与兜底。
9. **大小王**：主持工作不以年资论，「谁是班长谁说了算」。
10. **收尾**：业务放首位 + 及格政治意识，避免路子走歪。

### 可复用沟通模式

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 别当软柿子 | 拒绝无限退让；发疯不是解 | 被同事拿捏的基层管理岗 |
| 会上扳他 | 点事实 + 工作负责/助成长话术 | 事业单位科室内斗 |
| 向上对称阻碍 | 汇报时对称卡点 | 主持工作被架空 |
| 私下认错换联盟 | 向赏识你的主管认情绪失控错 | 与领导关系好但被同事整 |

## 文章正文

（同场主条目 BV1crKH6jERu 切片。）32岁山东小城市事业编，保障困难群众基本生活，入职3年8个月。前任科长累垮带怨调离，业务熟同事兼科长崩溃拒绝，她调来后被分管稀里糊涂架上代理科长——无人交接、天天加班哭，同事卸下重担后只干旧业务，棘手问题抛给她，说她说话没底气。

她曾在办公室「发疯」自救——大冰：那反而佐证是烂掉的软柿子。**尽量别当软柿子，捏不完。** 应认真研究对手，把握中心思想：为工作、为服务弱势群体；可在**会上**点出疏漏——「我是为了对咱们的工作负责，也是为了帮助同志你更好的成长，你怎么理解我的良苦用心呀？」威立住后，矛盾公开化，领导已知，不配合就是工作态度问题。

与上级汇报多、被欣赏——建议私下找领导**承认在办公室发飙的错误**，承诺不再犯，同时适当对称同事使绊子因素；领导安抚后表达想干好工作、对得起伯乐。合理借力，但勿心理过度依赖——领导想培养干将，不是认干女儿。

分清职场**大小王**——主持工作谁说了算；上天不是反复出题，「一两道题直接把你将死。」别奔着涨工资进这行；热爱谈不上但负责，底线是服务人民不出错。业务放在首位，搭配及格的政治意识与自保动作，方向就不会太歪。承认幼稚比含糊过去更好——**合理借力**可预先摆平或兜底问题。

问经历痛苦向哪成长——业务干部若政治意识弱，可能「整容比谁都会六」走歪；你要的是业务底线+及格政治意识+自保的综合点。3年内可成主导工作的代理科长，「没有哪种成长不痛苦。」
