# 职场生存：没有斗争意识，就开脱或战斗

<BiliPlayer :bvid="'BV1rCCSBEEGi'" />


**摘要**：27 岁上海国企财务（岗级略高于 A、B 两同事），遭平级排挤、孤立，向大冰倾诉时哭泣。大冰：你是来**领工资**的，关系差正常；要么**反复加强「来赚钱」认知**开脱，要么**树立斗争意识**——「谁骂你骂回去」，没理搅浑；**没有第三个选项**。职场哭坐实**弱者**；此刻哭因被关注而非事情本身，折射城市生活孤独。建议：亮明底线；注会赶紧过——「见识太少，没长大，还需要涨一涨」。

## 内容轨 · 人生观点

- **来上班领工资**：「您就是过来打个工，领份工资，干完活儿走人」——关系不好正常，关系好才奇怪。
- **两选一**：开脱（像零零后「我管你们呢」）或**斗争意识**——有理得理不饶人，没理把水搅浑；办公室政治基础斗争。
- **哭 = 弱者标签**：「并不是你哭你就有理」——哭加强他人认知你是弱者，潜意识讨可怜、求饶。
- **被关注才哭**：平常不哭，连麦被倾听才起伏——折射在这座城市**很孤独**，工作寄托了几乎全部生活。
- **亮明底线**：缺清晰底线——哪些话不能讲、哪些越界必须亮明。
- **有效积累少**：不是心理脆弱，是**见识太少、经历太少**；像女生宿舍矛盾不该在职场重复。

## 技法轨 · 沟通拆解

### 互动场景

27 岁上海国企财务，部门三人，与 A、B 磁场不合；出差时向领导直说 B 未做好遭 B 电话骂；工位搬人事后被刻意跳过；考注会进度慢，倾诉时哭泣。

### 大冰的回应序列

1. **目的三连问**：来上班目的？A/B 给你发工资？有过失动摇岗位？——否 → 不值得气。
2. **大面宽慰**：来领工资，关系差正常。
3. **两选一框架**：无斗争意识 → 开脱 or 战斗，无第三选项。
4. **斗争定义**：谁骂骂回去；有理不让，没理搅浑。
5. **判断做不到**：「你做不到去战斗」→ 适合加强「来赚钱」认知 + 注会。
6. **揭穿哭因**：被关注才哭 → 孤独；职场哭有害。
7. **收尾**：注会赶紧下——政策级建议。

### 可复用沟通模式

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 目的三连问 | 上班目的/谁发工资/岗位风险 | 为同事关系内耗者 |
| 两选一框架 | 开脱 or 斗争，无中间路 | 被孤立却不敢反击者 |
| 揭穿哭因 | 被倾听才哭 → 查孤独 | 职场倾诉崩溃者 |

## 文章正文

一位27岁上海国企财务分析、总账、报表、会计核算岗位的女生连麦，考注会只过两门最简单的，估计还要两年，感觉被部门孤立、有恶意，有时胸口痛但未确诊抑郁。

部门三人，称A和B：A强势、推诿扯皮、推活、曾让她帮中午回家的A干活；B因系统换新出差，她在外汇报B工作没做好，B电话骂她。她比A多两个岗级、比B多一个岗级，负责局域网电脑搬到人事工位，人事和部门AB会送礼吃喝故意跳过她，原要好的同事也疏远。上级领导对她应无直接不满。

大冰先宽慰：目的是赚钱挣工资，A和B都不给你发工资；没有明显过失动摇岗位稳固性——"你不是来走亲戚交朋友的，关系不好正常，关系好了才奇怪"。

第二点：没有清晰底线，要亮明哪些绝对不能碰、哪些话绝不可讲。第三：县城长大、家里开店——"从你求学到现在没有树立过斗争意识，也没有斗争实践"。若意识上真认知是来挣钱，就开脱一点像零零后，"我管你们呢"；要不然就树立斗争意识，"没有第三个选项"——谁骂你骂回去；有理得理不饶人，没理把水搅浑，"办公室政治基础斗争"。

她哭，大冰说事情本身不至于起伏这么大，是因为此刻被关注、被倾听——折射在这座城市生活非常孤独，工作寄托几乎全部，"你哭反而加强他人认知你是弱者"，潜意识是讨可怜求饶恕，"把委屈收起来"。结论：大概率做不到战斗，适合反复加强"我来赚这份工资"的认知，把注会快点过；换别人会讲联吴抗魏，她绝对做不到。不是心理脆弱，是"见识太少、有效积累不多"，像女生宿舍矛盾不该在工作后再重复——"没长大，还需要涨一涨"。
