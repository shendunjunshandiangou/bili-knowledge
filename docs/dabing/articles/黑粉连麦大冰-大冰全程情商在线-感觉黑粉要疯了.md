# 黑粉连麦大冰：全程「你说得对」的控场现场

<BiliPlayer :bvid="'BV1nDPte7Ezn'" />


**摘要**：一名自称来「试问」的连麦者预设四个攻击性提问，指责大冰连麦有优越感、书无价值、做事表演意义大于实际、深夜直播害学生。大冰全程以「你说得对」接招，配合「你感觉」框架把主观攻击还原为个体感受，不辩内容、不解释、不生气；连麦者越说越激动，大冰则用降温话术、认同策略、闲聊转场和「过客心态」卸力。连麦末尾平台弹出「引人不适言行」警告，大冰为保护直播间主动挂断。连麦者最后坦承「刚才是开玩笑，看你会不会生气」。

## 内容轨 · 人生观点

- **主观感受无需辩驳**：「在这个世界上，没有什么比一个人的主观个体认知更大的东西了；别人没有必要和义务去改变你。」只要对方加上「我感觉」，其表述在其自身框架内就成立。
- **生命链接可以很浅**：「有可能这辈子我们只聊这一次天，只说这一次话。」连麦者可能是过客，「在我生命当中可能只有这几分钟时间有一个交汇」，因此「我没有必要生你的气」——「归根到底对我不重要的一件事」。
- **生气有选择性，且与己无关时更淡**：大冰称「我都奔五的人了，没有那么多值得我去生气的东西」。他举例唯一真生气的情况：山东一位父亲不让考上大学的孩子读书、想让其回家结婚——「眼睁睁看着一个年轻人的前程因为父亲的一点愚昧可能发生方向性转化」；但「前提这个事跟我没关系，如果是我自己的事的话，大概率我是不会生气的」。
- **认同是让对方高兴的方式**：「我觉得没有什么比认同一个人可以更让一个人高兴的吧」——连麦者却回应「你这种语气就越来越让我生气」。
- **连麦者最终目的**：连麦者称「我刚刚在跟你开玩笑……看你会不会生气」，并提到此前录屏评论说他「气急败坏」——大冰回应不记得对哪位粉丝气急败坏过。

## 技法轨 · 沟通拆解

### 互动场景

深夜直播连麦。对方开场「请开始你的表演」，称要「试问」四个问题，语气带挑衅。四个攻击点依次为：① 连麦时高高在上、有优越感；② 写的书毫无价值；③ 做事（如百城百校免费音乐会）表演意义大于实际意义；④ 深夜连麦时间不对、粉丝多是学生、大冰「害了小屁孩」。连麦过程中对方多次指责大冰「阴阳怪气」「不正常」，情绪逐步升高；末尾平台出现违规警告弹窗，连麦中断。

### 大冰的回应序列

1. **开场设边界**：「你可以问，但是我也可以选择答还是不答。」
2. **攻击一（优越感）→ 认同主观**：「你说的对。」继而展开框架——「只要你加上你感觉三个字，你说的一切都成立……别人没有必要和义务去改变你。」
3. **攻击二（书无价值）→ 区分「认同内容」与「认同角度」**：「你说得对。」澄清：「我不是认同你说话的内容，我认同您站在您的角度，只要加上了你感觉三个字之后说的一切话。」
4. **对方搬出「大多数网友」→ 打回个体身份**：「这个大多数这三个字的话，大概率不是您缜密的数据整理过之后得出的结论……如果您代表您自己的话，您说得就对。」
5. **对方激动 → 降温控场**：「别激动，我这是好好跟你说话呢」「不要激动啊」「慢慢想，没事，咱们俩有时间」「别紧张别紧张，喝口水慢慢说」。
6. **攻击三（表演大于实际）→ 继续「你说得对」+ 不解释**：「你说得对，虽然我不知道你指的是啥，但是不重要，总之你说得对。」并补边界：「我没有必要向您解释什么呀……我也不在乎你喜不喜欢我呀，但是你不喜欢我这件事一定是对的。」
7. **对方指控「在乎却说不乎」→ 顺接不辩**：「那我就在乎就好，这也不能够改变我认为您说的这个话。」
8. **对方称阴阳怪气 → 再次点「你感觉」**：「因为你加上了你感觉三个字……能不能正常聊天呐？……我现在一定比你正常。」
9. **重申底线 + 平台规则内可接受**：「你只要上来了，不违反平台的规则，不搞人身攻击的话，刚才那一些我们都可以接受……你说的都对，那你还想干嘛嘛？」
10. **等第四问 → 给时间**：「你要不要再组织一下语言，我可以等你一下，我点根白将等你一下，慢慢想不着急啊。」
11. **攻击四（害学生）→ 认同 + 请公屏报年龄**：「你说得对。」并请公屏报年龄；对方报完后大冰接：「当然他们即便告诉你他们30岁、40岁有可能他们也在上学……但是这个不能够改变您的这个观点，我对您的观点不否认啊，您说得对。」
12. **对方说「你并不认同我说的对」→ 总括**：「总而言之，您在我这儿咋都对行不行？因为我主要的一个核心是，我不希望大晚上您太生气……不要带着气儿睡觉。」
13. **认同策略 → 对方更气 → 自降重要性**：「我对您是不重要的人呐，我不吃你们家大米啊……消消气消消气。」
14. **话题切换卸力**：「咱们聊一点轻松的，昨天晚上吃的什么呀？」——从攻击转入面条、成都饮食等闲聊。
15. **连麦者摊牌 → 过客框架**：「你觉着咱们俩的生命链接能够有多深？……在我的生命当中，我的认知是因为您是我的过客……我没有必要生你的气嘛。」
16. **平台警告 → 保护直播间边界**：出现「引人不适的言行」弹窗后，「你先不要接着往下聊了……咱俩不聊了，咱俩再聊我估计我可能就今天晚上的直播会被封禁的，那我先挂掉了好吧。」

### 可复用沟通模式

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 你感觉框架 | 认同「你说得对」，把攻击还原为对方主观感受，不辩内容 | 面对主观指责、黑粉挑衅 |
| 个体不代言群体 | 对方说「大多数/网友都……」时，请其只代表自己 | 对方搬出集体名义施压 |
| 可选择答与不答 | 开场声明有权不回答 | 预设多题拷问、面试式连麦 |
| 降温慢说 | 「别激动」「慢慢说」「喝口水」「有时间」 | 对方情绪升高 |
| 不解释不辩 | 「没有必要向您解释什么」 | 攻击意图大于信息交换 |
| 认同卸力 | 「您在我这儿咋都对」+ 关心对方别带气睡觉 | 想让对方下台阶（效果因人而异） |
| 自降重要性 | 「我对您是不重要的人」 | 对方因你而持续生气 |
| 话题切换 | 从冲突切到饮食等无害闲聊 | 对峙僵局、需要冷却 |
| 过客心态 | 生命链接浅、不必生气 | 一次性冲突、网络陌生人 |
| 平台/场合边界 | 警告出现时主动终止，保护主场景 | 直播、公开场合 |

## 文章正文

一位自称黑粉的用户连麦，说要问四个问题。大冰：可以问，但可以选择答或不答。

第一：连麦时高高在上、有优越感——"你说得对"，加"你感觉"三字一切成立，别人无义务改变你。第二：书毫无价值——"你说得对"，认同的是"你感觉"后的表述； "大多数网友"大概率非缜密数据结论。第三：表演意义大于现实，例百城百校音乐会——"你说得对"，不喜欢他他也不在乎。第四：连麦时间不对、粉丝都是上学小孩、害了小孩——"你说得对"，请公屏报年龄。

对方越聊越气，大冰反复"您在我这儿咋都对"，说不希望对方带气睡觉，"认同一个人最能让一个人高兴"——反效果。聊昨晚吃面条、成都吃面配菜等缓和。对方称开玩笑试探会不会生气；大冰说生命链接只有几分钟，对方是过客。平台弹窗"引人不适""脏话谩骂"，疑公屏举报，主动挂麦防封。
