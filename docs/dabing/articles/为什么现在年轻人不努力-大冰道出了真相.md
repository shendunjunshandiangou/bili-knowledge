# 为什么现在年轻人不努力？大冰道出了真相

<BiliPlayer :bvid="'BV171T56sE7B'" />


**摘要**：软件从业者请教主播盈利模式。大冰先立「法无禁止即可为 + 公序良俗不破」双框架，再论平台是寒门少数上升通道、承载希望；对比线下小饭馆创业风险，主张包容普通人线上谋生；强调八零后是时代红利受益者，「没有资格教别人」；希望平台「两头小中间大」，砍掉塔尖让流量下沉。

## 一、双框架前置

- 「法无禁止即可为」+「公序良俗不破本」——在此之下再谈主播与盈利。

## 二、三层认知

1. **希望值**：上升通道封闭，平台是气口；一人成功抚慰众人；速朽让空间轮转。
2. **轻资产**：普通人线上创业优于没经验就开小饭馆——「你不赔谁赔」。
3. **代际包容**：七九八零最幸运一代，「只是运气好」；零零后一零后没做错什么；「自己吃饱了瞧不起他们，凭什么呀」。

## 三、平台理想形态

- 希望无头部大主播，腰部+大量普通人直播间满足具体诉求（如凑学费）。
- 良性模型：「两头小肚子大」，金字塔最不科学；具体怎么做「轮不到咱们想」。

## 文章正文

做软件的连麦者请教：互联网门槛相对低、主播多，靠什么模式从小主播到大主播赚钱？考虑了很久，想请教心得。大冰：了解，不做主播，没有粉丝，咱俩只是聊天。在谈认知之前先放两句话：**法无禁止即可为**——衡量一切主播、盈利模式的前置条件；在此之下再推一步，**公序良俗不破本**。两个罩子之下干啥都行。

**第一层认知——希望值。** 不古板认为当主播网红不好、带货不对；可以不带但从不反对任何人去带。残酷现实：当今中国社会上升通道非常封闭，寒门子弟上升通道早闭合；互联网平台是为数不多的气口，可实现底层向上覆盖——叫机会、机会空间。其他行业还有哪些天然存在这么多向上通道？若还存在，意义巨大——**这个东西叫希望**；一个人成了会安抚抚慰一堆人的心，让大家都觉得还有希望。广谱人群若完全丧失希望太糟糕——举例年轻人不结婚，若从希望归因：对人生未来丧失希望，觉得没太多可能性、想象力空间，没希望再结个屁婚。平台保有希望值、上升通道、奇迹可能性（郭有才等），是好事；模式快速成名、**速朽不是坏事**——阶段性既得利益者赚到钱，把空间让出来给下一人，空间有限，尽量多人有希望值。

**第二层认知——轻资产。** 实业不好干；对一般普通人，这是为数不多的**轻资产、低投入、拼运气**投资项目——建号捕获机遇是创业行为，可最大化规避现实创业风险。直播间碰到中年失业有点积蓄想开小饭馆创业的，一定拦住——以前干过吗？没干过凭什么干成？不懂还拿积蓄创业，一般人第一层就小饭馆，**你不赔谁赔**。能不能琢磨轻资产方向，即便干不成不伤筋动骨。包容普通人把生活期许、盈利手段放网上；希望未来大主播都不复存在，只有中部腰部和大量普通人——为解决家用，目的单纯，如孩子重病需凑学费，直播两个月赚四年生活费，爆火一阵子满足后号别干了。目前阶段要牺牲头部满足底部；跟相关人士聊过说做梦，但有权利做梦——不做这种梦还玩什么互联网。

**第三层认知——代际包容。** 八零后（七九八零）是中国社会自有史以来**最幸运一代**，真正既得利益者，伴随改革开放出生；年轻时代比九零后零零后一零后幸运太多，稍微努力就有所成，相信天道酬勤，吃到时代红利。人到中年：**第一没有资格教年轻人太多东西，只是运气好；第二必须在某些时刻语境下底层认知相对理想化一点，否则对不起吃过的红利。** 中华文明有史以来最美好的 40 年（79–19）没有大灾难饥荒瘟疫，咱们赶上了，知足了。零零后一零后做错了什么？空间窄太多，上升通道封闭太厉害，机会值太少，**干嘛苛责他们？** 提到网红博主或多或少污名化，尤其年轻博主耍宝搞怪低俗——他们有没有权利吃饱吃好、追求美好生活？不像七零八零天然获得时代红利；生下来面临压力无比巨大，让人吃碗饱饭怎么地？凭什么骂？耍宝搞怪出丑为了吃饭，配不配吃饭、养好饭、有爹妈要养？**哪怕低俗，法无禁止、不破坏公序良俗，怎么着了？**

连麦者第二个问题关于平台运作原理——大冰也回答了方向。真实观点：限于刚才几点；具体怎么做轮不到咱们想。希望所有平台现有头部主播让空间、流量向下覆盖——以后不要头部，只有腰部和大量能盈利的普通人直播间。具体怎么做不知道实操方法，但知道任何产业不要资源集中金字塔尖，否则平台也分阶级。**良性发展应该是两头小肚子大**——社会学模型里金字塔永远最不科学，两头小中间大最好。连麦者表示学到很多，感谢。
