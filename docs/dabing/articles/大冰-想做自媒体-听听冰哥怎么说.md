# 《大冰》想做自媒体？听听冰哥怎么说

<BiliPlayer :bvid="'BV1ri5D6DEsQ'" />


**摘要**：32岁北漂程序员（月薪3万、存款约70万）计划转型自媒体，想做举火把的VLOG（健身做饭等），观察「丧文化」反向操作。大冰：不了解具体就没法梳理；**生活方式分享不稀罕**（衣食住行有什么稀罕的）；**丧文化早就过时**（一年前说还行）；社会观察「太空了」；需更具体赛道与差异化，不能只有意识形态空壳。

## 内容轨 · 人生观点

- **生活方式赛道不稀罕**：健身做饭等衣食住行切入，「这个有什么稀罕的吗」。
- **丧文化已过时**：主流/反向宣扬的观察若滞后，选题有问题。
- **自信需成事支撑**：程序员自述自信像空中楼阁，需具体做成的事支撑。

## 技法轨 · 沟通拆解

### 互动场景

32岁程序员转型自媒体咨询。

### 大冰的回应序列

1. 问赛道——对方不便细说→「不了解就没法帮你梳理」
2. 归类：大概率生活方式相关→「有什么稀罕的吗」
3. 拆社会观察：丧文化过时、观点太空
4. 要求具体：形式/内容需可执行差异化

### 可复用沟通模式

| 模式 | 做法 | 适用 |

## 文章正文

三十二岁北漂在职程序员连麦，计划转型做自媒体，想听看法。自我介绍：存款大概三十万现金、炒股三十万，公积金，没有贷款、房、车，未婚不要孩子；父母非富贵但能自理工作；月薪三万不忙。互联网行业感受需求萎缩、难找下一份工作，努力工作不会带来成长意义、也不能保住工作。从原生家庭自卑内耗中完成自我认同和家庭和解，有稳定三观和自信——但自信像**空中楼阁**，没有具体做成什么事支撑，极度渴望做对自己有意义的事；对未来社会发展有判断，对自媒体方向有想法，调研了信息，能支撑一段时间内容输出。

问做哪几个赛道——不太方便直播具体说，可简单说一下：大概形式 VLOG，比喻是**举着火把的人**不是灯塔，普通人明白——是否涉及意识形态探讨？不涉及，纯个人，**生活方式分享**。具体展现内容是健身还是什么——有点敏感。大冰：没什么可保密，即便说了也没人立马抄得去；**不了解就没法帮你梳理**。大概率跟生活方式相关，切入点生活起居、衣食住行——**这个有什么稀罕的吗？**

连麦者较立体介绍：锻炼、做饭完全不会；对社会感觉——目前躺平、社会文化会偏离：主流宣扬正能量时群体反向宣扬躺平，主流宣扬躺平时又反向正能量；至少北京大家宣扬比较丧的文化时，想通过自己做到一些启发、吸引同类。大冰：**首先社会观察这块有问题，丧文化早就过时了**——一年前说都已是尾巴，放在当下稍微扩大，政策面自上而下或广普受众已在慢慢摒弃，一定圈层摒弃速度非常快。

另一支撑点：资本主义导致个体空心化，大家忘了坚持还有自我——是否涉及意识形态？连麦者以为指宗教等，理解错了。若这个方面，跟具体内容填充的**关联度**——通过什么具体内容呈现是最主要课题，否则**坐而论道**，一定不好弄。

具体内容：非常瘦弱、不会做饭；16 型人格分析本质对生活感官非常弱，重思想不重行动，三分钟热度（不认为有问题但确实存在）；想以我为基点坚持做——可概括为**自我养成系**。瘦弱反而是优势——当下非常瘦弱，但能让大家看到浑身是块的过程，某项技能从零到满分；不同赛道很多人在实践，如李子柒；另一例子像歌手阿牛、做特殊菜、一苇渡江横渡长江去学——都叫自我养成。

**尽量不要做这个赛道，特别难**——最好有一定量积累后再做，粉丝基数最起码百万以上，否则很危险；除非打算拿出一年甚至更长碰运气（时间跨度甚至只是碰运气），留两年存款——**现有自媒体 VLOG 形式一定会被淘汰**（不说原因，成辩论了，只对标结论且很笃定）。

要结合自己实际，**最擅长输出的是啥**——即便只是程序员，一定有最擅长的：编程层面、逻辑思考、秩序化梳理问题、程序员思维等，都叫储备基础能力；拿出来怎么用，而不是现在想什么东西好都要从 0 到 1。**尽量避免从 0 到 1**，哪怕 0.1 开始也比从零好。依据个体本位做纸面推销或找同学朋友头脑风暴，发现最适合、最擅长干啥——能力在哪，根据能力做演变，转换成适合自媒体的内容形式；不要一上来先想大标大方向，要**从本位出发**。举例：特别看好某赛道、知道一定会火，但评估自己实力发现不匹配——**再好跟我没关系**。再举例：大盘上扬但需借贷才能完成初始投入——不会做，不叫真实具备的基础实力；考验个人能力。建议自我梳理，让朋友配合想想都会哪些，跟想做的有没有关联度；没有就放弃，根据会的做演化——**就怕你不会，你但凡会点啥**；有人没文化但善于说话、掌握民间语言极生动依然跑得通，看对谁说、怎么说、说什么；出发点是**个体本位**。玩这个大领域卖的是你自己、拼的也是你自己；区块链思维落到实地是每个节点，把节点个性特色差异化竞争力拿出来打天下。

还有小问题：人生阶段推荐的书或学习？不敢扮演这角色，但可以说短板——**成长向上**；接下来两年需要**忘年交**（女朋友除非五十岁否则不算）。年轻一代不要否定看上一代人，即便思想僵化也有完成一定量的积累有价值部分，缺这块需中和。几个好的忘年交可匹配思维助力——不是你不成熟所以需要，而是到了这个节点需要。**社会观察不够**；若指望个体信息汲取和书籍阅读辅助社会观察深入性或相对客观性，不太相信这条路径在你身上实现的合理性；希望有具体单位的人，通过良好交流、日常相处、耳濡目染辅助完成社会观察。参考消息算一种途径，但忘年交可遇不可求，**并不是通过阅读就可以完成**——一条腿是信息量索取（手机自媒体），另一条腿是**有质量的人际沟通交流**；天然愿意横向交流，不愿意向上或向下（打包年龄概念）。若书籍解决一切、自我教育真那么强，世界不需要老师——忘年交是生活当中亦师亦友、社会观察比我们更缜密的上一代人甚至上上一代人。

自媒体会再谨慎想，写表汇总评估。最后一个问题：沟通表达有问题吗？**很正常**——因准备好来的，听不出太多端倪；不设防无准备可能能观察到最真实表达。先分析再评论——对你而言未必是坏事。女朋友说有时咬字比较重「你怎么这样」——不是问题，只要不带情绪；**你有一个温柔的妈妈**（不是猜，是社会观察）。直播间对孩子温柔一点，长大外化表达时正向信息信号他人即可感知。
