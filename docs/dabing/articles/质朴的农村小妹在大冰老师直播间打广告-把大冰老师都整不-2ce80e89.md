# 质朴的农村小妹在大冰老师直播间打广告

**摘要**：农村小妹第一次连麦，口音像许三多，坦白做腌黄瓜不会推销，想借直播间打广告。大冰当场放行：「这会儿7万多人，抓紧做」；但她没挂小黄车、不会夸自己，几十斤酱黄瓜卖不出。大冰收尾提醒下播后开购物车，并肯定其「简单直接、特别诚实」。

## 一、连麦诉求

- 第一次连麦，做腌黄瓜（酱香型），一次腌几十到百十来斤，在县城卖。
- 不会推销、不会挂小黄车，「分享高兴的事来的」。

## 二、大冰现场帮推

- 识别口音像王宝强，主动给广告时间。
- 催她自夸产品、针对南方观众说明口味；指出没购物车「广告落不了地」。
- 建议下播后主页留言私信会增多，尽快上线小黄车。

## 三、收尾评价

- 「没那么多弯弯绕」「上来打过广告连个小黄车都没有，我替你打完这广告都落不了地」。
- 玩笑说直播间能帮她卖300斤，她要给邮十斤，大冰婉拒。

## 内容轨 · 人生观点

- **诚实直接**：「我就想着上来打个广告」——大冰认为「特别的好」。
- **机会要落地**：有曝光渠道还要配购买路径（小黄车），否则流量浪费。

## 技法轨 · 沟通拆解

### 互动场景

质朴农村小妹首次连麦求帮卖自制腌菜，无电商基础。

### 大冰的回应序列

1. **接话**：口音识别拉近距离（许三多/王宝强），立刻同意「来打吧」。
2. **控场**：告知在线人数，限时鼓励现场推销。
3. **追问**：辣/甜/酱香？挂小黄车了吗？教她主动夸产品。
4. **收尾**：提醒下播后开购物车；肯定诚实诉求；幽默收束（300斤/十斤）。

### 可复用沟通模式

## 文章正文

农村小妹第一次连麦，开口「大兵哥你好、爷爷好、晚上好」，说自己是农村来的，特别幸运。她做腌黄瓜（酱香型），一次腌几十斤到百十来斤，在县城卖，不会推销，「就想借着你这打个广告呗」。

大冰听出口音像许三多、王宝强，立刻说「没问题，来打吧」。他提醒：若是借直播间宣传，这会儿人多——「好像有7万多人」——要抓紧做。问有没有购物车、淹了多少黄瓜；得知一次几十斤、村里人、没有小黄车、不会弄之后，仍鼓励她在连麦过程中尽量把几十斤卖出去，并教她主动夸产品：是辣口、甜口还是酱香？南方观众多，得好好推销，「这样也算是连上麦，我给你帮个小忙」。

小妹形容腌制过程麻烦、用新鲜蔬菜，推销不出去因为「没人认识我」；真不会夸自己。大冰指出：没有小黄车，广告打完别人也没地方买——她是来「分享高兴的事」，但下播后主页应有人留言私信，要尽快上线小黄车，把腌黄瓜、萝卜、辣椒等卖出去。他肯定她「简单直接、特别诚实」——「我就想着上来打个广告，特别的好」。小妹说准备少了，大冰半开玩笑：小看直播间了，100斤不够，「最起码能够帮你卖300斤」；她要给邮十斤，大冰婉拒「十斤黄瓜我们家也不用买别的了」。

连麦结束后大冰向观众点评：多质朴、直接，没那么多弯弯绕；但上来打广告连小黄车都没有，「我替你打完这广告都落不了地」。
