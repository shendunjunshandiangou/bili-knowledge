# 职场大忌：跟着第一支舞的人跳

<BiliPlayer :bvid="'BV1vZFezrEZT'" />


**摘要**：94 年男从「前老大」跳到「现老大」部门，犯**职场大忌**：「永远跟着第一个带你去舞厅的人跳第一支舞」；信同事传话（非者必是是非人）；越级打电话给一把手诉苦→被要求供人名→微信被删；成前现老大之间**枪与棋子**，前老大不回信息=弃子。大冰：**效忠现老大后不得再连前老大**，「没得选」；效忠只能**大场面喝酒借酒掩面**；需求决定关系，非父兄；父位缺失→对高位者恐惧与讨好。

## 内容轨 · 人生观点

- 从引路人跳到对立面=面上背叛，他人界定难纠。
- 一把手关注团队稳定大于个体委屈，越级诉苦极傻。
- 传话难避放大镜；递话「朋友」可能变下阶段对立面。

## 技法轨 · 沟通拆解

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 第一支舞大忌 | 勿从第一个带你的老大跳到新老大 | 站队/转岗 |
| 效忠后断前联 | 决定效忠现管者后，不得再与前老大正向连接 | 办公室政治 |
| 借酒掩面效忠 | 圆台酒至份上、有人时低声，勿单独掏心窝 | 修复上下级 |
| 不听不信传话 | 只观测现老大真实态度，逮机效忠 | 被告密 |

## 文章正文

94岁男，北京985本科，私企从专员到部门主管、办公室副主任（最高职），现新疆乌鲁木齐合资民企做技术岗。

**三位领导**：老一=董事长；老二=前老大（原民企集团副总裁，带连麦者进公司，有知遇之恩）；老三=现老大（与前老大同公司原老一老二，来新疆后迅速与前老大闹掰）。连麦者从前老大部门转技术岗——自认管理能力不行，算「逃避」。

**困境**：同事（一男一女平级，连麦者帮忙牵线进公司）传话——现老大认为连麦者拉帮结派；年会老板也「听说」此事。连麦者信传话（大冰：「非者必是是非人」，传话有放大镜效应，不必信他们「报恩」）。现老大上任不到一月把连麦者踢出办公室，跟老板谎称「自己想回家休息」——连麦者跟老板否认，老板「知道了」但事过。

**致命操作**：焦虑下问前老大是否向老板解释拉帮结派→前老大建议可电话说明→连麦者**越级打给老板**摘清自己→老板强要供人名→连麦者不说→老板删微信。前老大后两条信息不回——「这杆枪不好使」「弃子」。

**大冰核心观点**：
- 职场大忌：「**永远跟着第一个带你去舞厅的人跳第一支舞**」——从引路人跳到对立面=面上背叛。
- 决定向现老大效忠后，**不得再与前老大有任何正向连接**，「**没得选**」。
- 越级向一把手诉苦换公平=极傻；老板关注团队稳定大于个体委屈。
- 标配：传话「不听不信」，观测现老大真实态度，逮机**效忠**（大场面圆台酒、自由散打时低声，**借酒掩面**，勿单独掏心窝）。
- 若再效忠又不说人名=「当汉奸也得认真一点」——生存最优解可推锅，但大冰不教。
- **需求决定关系**，非父兄；慕强/认父兄=职场大忌；父位缺失→对高位者恐惧与讨好。

**Q&A**：能否调回前老大公司？——打现老大和一把手脸，前老大不会为你忤逆；信息不回=事实弃子。效忠道歉？——只有喝酒借酒掩面一条路。
