# 大冰在线教学：如何追女孩

**摘要**：27 岁厦门设备工程师，东北人，i 人、地铁搭讪加微信聊死。大冰：**别把女孩当练手道具**；**别在地铁大马路要微信**——易构成冒犯、女孩或出于害怕/不善拒绝；应通过**志趣相投的场域**（户外团队、摄影同行）建联；聊天靠**话题内容**非纯话术；**夸要放大**（一分夸三分）；**表白一次**、勿死缠烂打；保留**绅士风度**与边界尊重。

## 内容轨 · 人生观点

- **心态纠偏**：「那些女孩子成了你的道具了，拿她们练手——那不行。」
- **搭讪边界**：地铁随机加微信「老套」且可能冒犯；1 米 85 高个让女孩「大概率是害怕」；打破安全距离未必善后有美好回忆。
- **建联路径**：户外运动民间团队（骑行慢跑等）、摄影同行——需共同志趣与信任背书；勿借相机在大学周边「乌烟瘴气」搭讪。
- **相亲可行**：「嘴笨的很多」也可相亲；厦门户外团队多。
- **尊重意愿**：国人「不太习惯主动拒绝」——给了微信可能是懵+不会拒，勿罪化对方随便。

## 技法轨 · 沟通拆解

### 互动场景

27 岁工程师，周围女生少，地铁搭讪加微信；自认外形尚可但不会说话；朋友教「看怎么做不看怎么说」仍无效；最近买相机。

### 大冰的回应序列

1. **追问具体**：你是谁、对方是谁——「具体问题具体分析」。
2. **纠偏心态**：经历少可理解，但「女孩不是练手道具」。
3. **路径建议**：相亲 + 厦门户外运动小团队 + 摄影志趣。
4. **否定纯技术**：不了解对方无共同语言，「很难往下聊，成了纯玩技术」——反对快速占便宜类搭讪课。
5. **摄影路径**：节假日鼓浪屿与同行讲明非抢活、可义务帮忙；警惕相机搭讪变味。
6. **自评盘点**：185、七八千、工作稳定——「相亲市场不至于不受欢迎」。
7. **收尾禁令**：「别再干地铁大马路要微信」；保留绅士风度；表白一次、勿死缠烂打。

## 文章正文

连麦者想咨询追女生——「这活一般的居委会大妈都干不了」，不知道怎么跟人说话，脑袋里没有想法，不太喜欢说话。大冰说：先告诉我你是谁，再告诉我他是谁，**具体问题具体分析**。

连麦者27岁，厦门设备工程师，父母在东北，周围女生少。之前地铁等地搭讪加微信，加上聊几句就聊死。自认外形比别人好看一点，但不太会说话，一说话别人觉得没意思；农村孩子，朋友说要怎么做不要看怎么说，但不知道人家什么时候喜欢或不喜欢自己。约吃饭往往聊不到那一步——一般「嗨你好，我是谁谁谁」，后面就不会聊了。大冰说：**夸他呀，赞美他呀，有一分放大成三分，从头夸到脚。**

地铁随便加微信「有点老套」——网上见过，现实生活还没遇见过。连麦者说也不是为了什么，想多经历，知道自己经历少。大冰说：**心态有问题——那些女孩子成了你的道具了，拿她们练手，那不行。**

建议：**相亲**——不是找不到另一半才相，很多像你这样嘴笨的也可以。**厦门户外运动**——不是极限户外，骑行慢跑等民间小团队，有共同志趣才有共同话题；大马路上随便拽一个人谈恋爱，「你是哪根毛啊」——先找建立建联的契机。最近买了相机搞摄影，有些话题但只聊几句。大冰说：话题深入靠**话题内容**不是话术；不了解对方、没有共同语言秩序，很难往下聊，就成了**纯玩技术**——网上专门教搭讪沟通的那批，发心目的是快速占便宜，不鼓励，会影响社会风气。

摄影路径：**略有了解**——有人背相机在漫展、大学周边（如厦大附近）以拍照名义建联搭讪，晚上带去酒吧，搞得乌烟瘴气，不太建议。想认识女孩子不错，但要志趣相投、合适场域、有信任背书和共同话题。节假日背着相机上鼓浪屿，跟同行讲明白不是抢活，免费义务帮忙。

喜欢漂亮的觉得看不上自己——大冰让他逆推：女孩子看上你，你值得被认同喜欢的点有哪些？形象打扮后比周围人帅、185身高、月入七八千、工作稳定、最近有些内耗——**大概率在相亲市场不至于不受欢迎**。踏踏实实想真想谈恋爱结婚，先聊到这。

最后一句：**别再干地铁或大马路上问人要微信**——南方小土豆抬头看1米85快1米9的大高个，大概率不是喜欢或有可能性，是**有点害怕**；动作不太礼貌，显得没大有品位。事实上会对一部分女孩子形成心理上的**冒犯**——人际有边界意识和安全距离，轻易打破有没有办法很好善后？女孩子事后想起来是美好回忆吗？保留一点点**绅士风度**（词有点过时）。将来表白**一次就行**，人家愿意不愿意直接告诉你，别死缠烂打——不是让人无限尝试，要尊重对方主观意愿。很多女孩子大马路上被要微信反应没那么快，懵懵的状态；国人不太习惯主动拒绝，给了微信可能只是**不善于拒绝**，不是授权——既不要罪化人家随便，也别当小流氓；可以不当好人，但不能当小流氓。别乱学网上搭讪课——快手有人教搭讪中年小少妇还成功案例，「那可没品了」。
