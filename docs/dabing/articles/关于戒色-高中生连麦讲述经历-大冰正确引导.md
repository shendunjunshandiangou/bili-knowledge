# 关于戒色，高中生连麦讲述经历

<BiliPlayer :bvid="'BV11MNU6fEUz'" />


**摘要**：18岁高中生问「怎么戒色」。大冰：18岁「变啥色」——阳刚气血滚烫，**合理管控即可**，有点想法不是错；自称「重度成瘾」→「大概率属于医疗范畴，帮不了，得医」。后半段连麦：体育生十字韧带断裂、康复放弃、游戏账号1.5万粉月入约1万——建议**学还要上**、毕业证要拿、游戏当兴趣爱好；体考相关专业可琢磨；燕子幼鸟应送回窝。

## 内容轨 · 人生观点

- **18岁不需戒色**：合理管控，非医疗问题不必病理化。
- **游戏当爱好别押宝**：1.5万粉小游赚不了太多，押宝风险大；高中毕业证必要（当兵等都需要）。
- **康复不能因赚钱停**：挣到1万后停康复是错误路径。

## 技法轨 · 沟通拆解

### 互动场景

①戒色咨询；②韧带伤+游戏账号+是否返校。

### 大冰的回应序列

1. 戒色：直接否定必要性→区分医疗/正常
2. 游戏段：「你这段话想告诉我的信息是啥？」——逼中心诉求
3. 给结论：学要上、游戏当兴趣、体考方向可琢磨

## 文章正文

18岁连麦者开口问「怎么戒色」。大冰：18岁变啥色，这不很正常吗，阳刚之气血都是滚烫的，**18岁不需要戒，合理管控好自己就行**，有点想法不是错。连麦者说现在有神经症、韧带也断掉了、重度成瘾。大冰：这跟戒色有什么关系？**大概率属于医疗范畴的问题，帮不了，得医。**

连麦者主要想倾诉：2022年高一下半学期，他是体育生，在足球队训练，一次训练导致**十字韧带前交叉韧带扭断**。他进入市里较好的一所重点高中，当时是踢得最差的，运气好跑得快、撞个人，教练稀里糊涂把他整进去，还交了2万块钱。进去后看旁边的人，不甘心自己特别差，拼命训练往死拼，确实进步了，韧带却断了——好不容易得到教练信任，已经发挥比较好、准备打市里比赛，努力这么长时间，给队友和教练……最后韧带断了，心里非常不好受，没打成比赛，后边做手术康复。

他和奶奶去北京——父母离异，姐姐跟妈在江苏，他跟爸，爸有点不务正业，姐姐有点叛逆。当地县医院做韧带手术，听朋友说有人韧带被做坏了，就自己拉着奶奶去北京租房，去北医三院。坐火车时非常幸运预约到教授级别的好医生；到了北医三院，医生看他是学生还加号插队，可能要等一个月，两天就办住院。做手术时**细菌感染**，经历十天非常痛苦的煎熬，十天没睡觉也没吃饭——主要特别对不起奶奶，奶奶在旁边借钱。大冰：将来长大参加工作，对奶奶好点就是了，现在没有必要这种想法太多。

康复后一直在家里颓废，有点没希望，问如何找到心里边的火苗。现已高三毕业，**没去参加高考**，学籍还保留，可以复读。大冰：先不去考虑体育生，考虑正常文科；学习成绩确实很差，**高考也必须参加**——谁说光考实力，还有点运气成分，万一蒙上了呢；两项还不行，最起码拿高中毕业证，将来当兵也能当得了。

还有一件事：2025年11月想连麦，在家没事干，在抖音创足球游戏账号，尝试接单子，账号做起来后一个月能赚到1万块钱——腿还在康复，但**第一个月挣到1万之后腿就没有再去康复了**，简而言之放弃了，学到了这种（赚钱）状态。大冰追问：**你这段话想告诉我的信息是啥？** 是否犹豫不想回学校、一门心思做游戏账号？粉丝量1万5左右，但游戏比较小，**偶尔把它当个兴趣爱好运营着**，有点收入是点，1万5也赚不了太多钱，**学还是要上的**；押宝押到那上面，担心将来风险很大，竞争力没有那么强。尽量保证高中毕业证——没有高中毕业证，回头想当兵都当不了。学考老师让他参加学考，应该可以拿到高中毕业证；**下一茬高考必须要参加，碰碰运气**。纯运动体考项目参加不了，跟运动相关专业可以琢磨，往这个方向。

做完手术后接触因果类东西，门前燕子筑巢，幼鸟掉下来——**第一天应搬梯子重新放回窝里**，家长也告知这是最好的办法；他第一只幼鸟放回去了，早上一看一天咣咣咣掉四只。大冰：一下子掉四只，跟因果做太深连接关系不大，**好像就是给它喂死了**——又筑了个巢，把唯一活下来的鸟放进去，自己喂肯定喂不活。连麦者心里觉得把鸟给整死了、像杀生了。大冰：**万事看发心**——主要看你面对这个鸟是想弄死还是想救活；发心是想救活，但能力、实力、知识信息匹配不到，最后没能救活，**不算你把它弄死**；不是玩弄生命、玩死，与虐猫虐狗、故意捅燕子窝完全不同性质。
