# 从初中自学到三硕士：学不动了怎么坚持？

<BiliPlayer :bvid="'BV13ugk6YEfL'" />


**摘要**：浙江山村唇腭裂女孩，17 岁获免费治疗后才敢出门；初中辍学打工十年，自学大专本科、同济 MBA、澳洲硕士，现远程任美资中层。连麦分享励志路径后，大冰延伸：**意志力低谷时用「一点点再坚持」**；**别俯视躺平者**；励志多是「没回头路只能往前走」；**攻击力宁可向外不对内**；人生巅峰有三条路——活在过去自我认可、服务众生、**翻篇放下昨日之你**；低谷需**找小战绩自我赞美**作自救抓手。

## 内容轨 · 人生观点

- **精神寄托**：累到看不见头时，心中存一个更高愿景，「不管能不能实现，对当下困境的意志力非常有帮助」。
- **正反馈链**：上海咖啡厅因会英文多 100 元/月，「现实正向反馈可以激励继续往前」；考完大专立刻报本科，「一步一步让自己有更高一点的目标」。
- **不俯视躺平**：「不应俯视不励志、甚至有点躺平的人……拿着我们的正确去否定他人一定不正确」；某些躺平是「不良环境中的短暂自我疗愈」。
- **励志真相**：「所谓外界看到的励志，往往是生活一巴掌打过来，你没有回头路……若干年后自我美化成奋斗励志」。
- **三条路径**（人生走到顺了该怎么活）：① 让一部分自己活在过去，加强自我认定；② 服务社会利益众生重建价值；③ **翻篇**——忘记来时路好坏，「昨日之你非今日之你，咋活都得活」。
- **攻击力向外**：「宁可对外，尽量别对内」；自卑自我攻击时，「宁可你狂一点，都不希望你当个怂人帮世界杀你自己」。

## 技法轨 · 沟通拆解

### 互动场景

连麦者主动分享从服务员到三硕士的励志样本；大冰控场（勿看公屏、边吃点心边听），随后从「如何对抗想放弃」切入，再延伸励志观、攻击力、人生路径。

### 大冰的回应序列

1. **追问干货**：「不可能整个过程没有想放弃——最后实现不放弃，心里产生放弃时如何对抗？」
2. **一点点坚持法**：意志力薄弱期「目标太远坚持不到，就先坚持一点点——三天、一个月、这一票难关」；正反馈像本钱，「一点点往下坚持，回头一看好像没有放弃，动作被无限拉长」。
3. **承认普通人**：「不太相信个体有多大意志力……普通人能做的是一点点做，甚至都得死、能活一天多白赚一天」。
4. **打样价值**：有参照模板，普通人可投射反弹，「重叠度反向给我们增加信心」。
5. **延伸躺平**：客观原因下只能温饱的人不见得做错；励志成功者应广谱认知——不俯视他们。
6. **攻击力重构**：社会教育压抑「翘尾巴」，但当下年轻人易自卑对内攻击；「爱别人、帮别人、自证、讨要赞美都是攻击力向外」。
7. **两害取其轻**：阶段性骄傲膨胀像可乐解当下，长线要置换成白水，「先从这个坑出来再说」。
8. **三条路径收尾**：建议选第三条翻篇；若走第二条可搭配第一条；原生家庭议题对此样本「明显不需要」。

### 可复用沟通模式

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 一点点坚持 | 意志力低谷只承诺最小单位 | 长期目标受挫 |
| 打样投射 | 承认参照系对信心的价值 | 励志样本连麦 |
| 不俯视躺平 | 成功者不拿路径否定他人 | 代际/阶层对话 |
| 攻击力向外 | 自证、赞美、助人作对外通道 | 自卑自我攻击 |
| 三条路径 | 过去认可/服务众生/翻篇 | 人生巅峰迷茫 |

## 文章正文

连麦者来自浙江小山村，初中毕业后没有书读，出来打工约十年，服务员等各种活都干过，但一直没有放弃读书——当时在学校有梦想要读书，客观原因读不了，就自己学，资料不多、网络不发达，听收音机捡资料。她自考大专，想告诉前面那个被骗的小姑娘：自考是一条路。专科没读完时，拿着初中文凭加大专在读，找到外企工作——那时机会比较好，她不怕，很自信去面试；面试的人里有天津大学等名校的，她就冲，经历很多拒绝，不断尝试，最后被一家美资公司录用。虽然做文职，对她已是天大的好事——十来年一直做服务员站着的工作，上班也闹过笑话，但第二次不犯，一直在总结。

那段婚姻出现问题，她仍没放弃读书，之后又读本科，认识现在的先生。孩子大概三岁时开始考研，读同济与英国大学双学位 MBA；那时工作已经蛮好，外资做到经理、部门老大。考研阶段数学太差，没日没夜刷题，出差美国时在飞机上给老师讲题，四个月每天大概睡四小时，后来考上。全职工作还要读两校研究生，课也比较大。2022年带家人在澳洲又读研究生，现在拿到三个硕士，远程为美国外资公司做中层管理。

她还有比别人更难的一点：出生唇腭裂，17岁才由美国公益机构免费治疗，那之前17年一直很自卑，没法出去打工——那是她重生的开始。她希望把经历分享给需要启发的人。

大冰请她讲干货：「不可能整个过程没有想放弃——最后实现不放弃，心里产生放弃、甚至找出一堆理由不得不放弃时，如何对抗？」她说：打工很累，靠体力时间换很低收入，要活下去；不要陷在当前困境里——当时做法是心中找寄托、目标，比如学英语是精神支柱。最初十年没有自考机会，但没饭吃时可以去酒店食堂打卡、下个月工资扣；心存希望，觉得「我不会待在这里，我有更高的想法或愿景」，不管能不能实现，对当下意志力非常有帮助。自己学六七年效果不好，底子很薄，但在上海一家咖啡厅做服务员700块一个月（别人600），因为会英文，外宾都由她接待——老板说「你这小姑娘不错，能讲英文」，那是莫大鼓舞，现实正向反馈可以激励继续往前。考完大专还没考完就报本科，一步一步让自己有更高一点的目标；初中到大专是很大的跨越，够不着的那种痛苦不怕，但不知道方向时真的要靠意志力。

大冰提供**一点点再坚持**法：人不可能没有逃避心理，意志力是反人性的，人性图安逸。常用方法就是**一点点再坚持**——意志力薄弱时觉得目标太远坚持不到，就先坚持一点点：三天、一个月、这一票难关。坚持下来了正反馈就有了，像二级市场得有点本钱才能博下一个一点点，一点点往下坚持，回头一看好像没有放弃，动作被无限拉长了。他不太相信普通人有多大意志力，建议一点点做，甚至「都得死，能活一天多白赚一天」——像上阵杀敌，多拉几个垫背就赚了。连麦者认同：像项目切分成小任务，完成一个阶段有一个正反馈，长期坚持，回头一看每个节点最终成功交付，心理特别好。

大冰说她的样本价值在于：普通人很恐惧的是没有人打样、没有参照系；有模板在前，即便跟我没关系，重叠度反向给我们增加信心——年轻人把自我往你身上投射，反弹回来捕获的信息一定有益。连麦者说：很多人半路就被打倒了，情况机遇都不一样，不能说是样板，但有些东西可以自己掌控——要不要人身意向。

大冰延伸：**不应俯视不励志、甚至有点躺平的人**。除了励志路径，其他不见得都错；基于客观原因只能在有限范围捕获温饱、平稳生存的年轻人，出于基本自保短暂躺平，在彻底躺平之下反而不易出现太极端的心理问题——某些情况下的躺平是一定量的积极自我治愈，是在不良环境中短暂摆脱。励志成功者应广谱认知：不能拿着我们的正确去否定他人一定不正确，要因地制宜、因材施教。

连麦者非常同意：她也不是一直励志的人，人生有很多躺平时刻；考研阶段冲刺，之后要调整，不可能一直那样。工作都完成、很平稳后，压制很多东西突然爆发，所以在找心理咨询师聊——20几年一直努力，也会有起起伏伏，允许自己在某段时间什么都不干、刷剧玩，什么都不想干。

大冰说：搞清楚自己想成为什么样的人，并拟定一条自己肯相信并愿秉行的路径就可以了。平静恬静的一生、相爱的爱人、健康的父母、良好养育的孩童、温饱不忧，即便没有惊涛骇浪、世俗意义上的打拼，也挺好。关于励志，他不太相信励志这个东西——对外界看到的励志，往往是生活一巴掌打过来没有回头路，只能往前走；走过艰难路段后自我美化成多么奋斗励志，但放到真实当年场景，没有回头路，躺平会被干掉，倒退会被干掉，生如逆旅单行道，只能往前走。连麦者说：很多时候只是标签而已。

有没有必要强调自己某段历史时期励志？**一定有**——这叫积极的自我认可。外界认同、赞美、被看见机会很少，人都需要自己看到自己、爱自己、认同自己；实现路径包含很重要的一条叫**自己赞美自己**。尤其还没成事的人，越没成事越要积极认可自己——人生低谷还能往前走，一方面没有回头路，一方面越在低谷越不能否定自己，要从过往找小战绩甚至夸大化，强化「我成过事儿、我不那么糟糕」——这是抓手，从光滑的深渊墙壁上慢慢往上爬。不仅困境需要自救，活着一场经常需要自救；事来之前先把自救机制建好，学会夸奖赞美自己，还要主动向外讨要赞美。

连麦者说谷底的方法有点麻木——「我已经在谷底了还能怕什么」，类似钝感力。大冰接下来说可能得罪人的话：从小到大被教育不能骄傲、别翘尾巴、别飘别浮，但他活到今天不再那样看——如果没有影响到他人，只是个体自我认知，很多情况下是好东西。当下年轻人容易陷入自卑、自我否定、自我攻击；与其如此，不如重新抓起被否定的「翘尾巴」——心理学讲攻击力，不拿出来对外一定向内；爱别人、帮别人、自证、向外获得赞美都是攻击力向外。**宁可对外，尽量别对内**——焦虑抑郁很多是对内难为自己。跟明显自卑的人说「你要自信」太空了，他宁可对方「狂一点」都不希望当个怂人，宁可交往过正一点点中和，都不希望帮世界杀你自己。

连麦者很有感触：「你说我不行，我就证明给你看」——自证在某个阶段一定有用，像可乐解当下，长线要置换成白水，但任何問題先解决眼巴前；阶段性骄傲膨胀有利于不陷入谷底对内攻击自己，两害取其轻，先从这个坑出来再说。

人生走到这一步该怎么活？大冰给三条路径：**第一条**，让一部分自己活在过去，加强自我认定认可，从外界获得认定——你太不容易了，终于有了今天，可以形成心理辅助，玩到生命终结前对自己认可度不衰竭，OK。**第二条**，做服务社会、利益众生、帮助他人，在此过程中重新建筑价值——连麦者说这就是她的想法。**第三条**，稍微高级一点：忘记之前好的坏的一切，翻篇——不管来时路多艰难、成就多高，都不算数了，接下来怎么活都行。若心理咨询师建议第二条，可以搭配第一条，但他希望她选第三条——即便选了第二条，将来还是要面对第三条。

连麦者说第二条可以做，第三条很难——像潮水一样涌来很难排解，写作做视频抒发做不到，是不是要有时间，是不是一种恐慌，是不是原生家庭。大冰：建议把所有问题都往原生家庭归因，尤其是像她这样已经很好重新养育自己的人——明显不需要非得往原生家庭上考，别被词汇绑架；可能就是这段时间过度一下，静下来就好了。她自我掌控力很强——当年当领导能厉害得要死，手下兵见到她打嘚瑟，像穿普拉达的女魔头；她比较较真、要求比较高，后来也在慢慢学。
