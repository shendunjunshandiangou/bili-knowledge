# 北大老教授与大冰心灵对话

<BiliPlayer :bvid="'BV1vcPjzhEYp'" />


**摘要**：六零后北大国际关系学院退休教授（长期抗抑郁用药、脊柱问题只能躺聊）因「情商59」想向大冰学习。大冰先纠正其「帮年轻人」逻辑——悲悯可以，不必痛心实操；公屏维护秩序让老人讲完；用萨拉热窝比喻「过去的事与现在的你是两个人」；劝别信情商测试、自在最重要；送歌《未名湖是个海洋》后拒留微信。

## 一、教授诉求

- 退休多年，先生提醒情商不够；刷到大冰视频想学习。
- 自述残疾、重男轻女、北大求学、哥哥入狱、母亲精神分裂、资助贫困生、支教等漫长创伤史。

## 二、大冰核心回应

1. **设边界（帮年轻人）**：「谁都帮不了……你可以悲悯，大可不必痛心」；起了善念已有功德，不必实操指导。
2. **控场**：公屏提醒勿有罪推论、爱幼尊老；「大概率好多年没人认真听他讲讲话」。
3. **重构**：萨拉热窝——「现在的你跟当年的你已经是两个人」；跟陌生人聊过去事会「像说别人的事」。
4. **反测试**：智商147情商59「相信个六」；大冰自曝测出智商65；「写卷子的人智商有问题」。
5. **自在**：「没有什么比自在更重要」；别较劲、别硬找烦恼；「烦恼即菩提，别给自己没烦恼硬找烦恼」。
6. **收尾**：认定对方不需情商加强；拒留微信——「活特简单」。

## 内容轨 · 人生观点

- 自在、不较劲，比量化情商更重要。
- 帮人的念头够了，不必背负他人因果。

## 文章正文

六零后北大国际关系学院退休教授连麦（脊柱问题只能躺着聊），最大剂量服用北医六院专家会诊后的抗抑郁抗焦虑药，大脑有器质性损害，有时说话断片。测出智商147、情商59，先生提醒情商不够，刷到大冰视频想学习交流——底层逻辑：**跟任何人说话得自己先自在**。

她讲述一生：出生残疾，农村重男轻女，母亲精神分裂常无故暴打——「你长得这个样子还在我面前晃，让我烦，你赶紧死去，我偏不我偏要活着。」九岁才让上学，不交学杂费不准坐桌椅，站后排听一学期，考200分仍被霸凌，老师说「你们家是大地主你还想怎样」。无伙伴，拼命读书、割草、干家务；父亲察哈尔师范学院旧书繁体字自学。高中每月10元吃不饱，骑车100多公里回家拿咸菜拌辣椒吃一个月；高考体重不够90斤，老师求情让医生写正常才考上北大，哥哥同榜落榜。

父亲托关系让哥哥复读，哥哥几次不念，当代课教师、做小生意、奸淫幼女罪判8年。父亲要求她别读书回家捞人、嫁人，她以死相逼；大二起中关村家教，一小时从五毛涨到两元，自己挣完四年北大。84年入学88年毕业，不包分配，手写简历投到北京医科大学做辅导员；自学法律考律师，母亲医疗费2000、5000要她负担，父亲打耳光说六亲不认，她曾吞安眠药洗胃，父亲仍来要钱。

研究生考上，资助特困生（每月工资分给十人，最高不超3000），做心理咨询、支教。母亲临终说「不该来的你倒来了」。见年轻人自杀很**痛心**，想继续帮年轻人，用病例当「大体老师」攒资粮。

大冰先拦：**谁都帮不了，每人有自己的因果**——可以悲悯，不必痛心，不必基于痛心做指导；起了善念已有功德，不必实操。公屏维护：勿有罪推论、爱幼尊老；「大概率好多年没有人认真听他讲讲话。」

**萨拉热窝比喻**：现在的你跟当年的你已是两个人，不必再拯救当年的你；跟陌生人聊过去事「像说别人的事」。IQ/EQ测试——「相信个六」；大冰自曝智商测出65，「写卷子的人智商有问题」。**没有什么比自在更重要**——怎么舒服怎么来，别较劲，「烦恼即菩提，别给自己没烦恼硬找烦恼」。认定不需情商加强；送歌《未名湖是个海洋》；拒留微信——「活特简单。」
