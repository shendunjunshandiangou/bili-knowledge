# 出生杀人犯连麦大冰：去你奶奶的

<BiliPlayer :bvid="'BV1TaK268Edp'" />


**摘要**：41岁云南男子，4年前失手杀人服刑10年已出，母亲患癌、抑郁严重，曾用自杀威胁大冰小屋老板要求连麦。大冰念其威胁留言后**护短兄弟**：「我不骂你，但我就俩字——滚蛋」；对连麦者先缓情绪，劝有车票钱就回家陪母亲；「一辈子又不是现在没赚到钱意味着永远赚不到」。（注：本字幕文件后半段含另一连麦，已另记。）

## 内容轨 · 人生观点

- **护短自己人**：威胁跟过自己的人（小屋老板钟不悔及妻儿），零容忍——「这是我兄弟，我罩着的」。
- **先回家陪母亲**：母亲患癌，有车票钱就该回去；没脸见也要陪。
- **价值非一次性**：现在没赚到钱≠永远赚不到；别因阶段性困境判自己一辈子。

## 技法轨 · 沟通拆解

### 互动场景

刑满释放人员，愧疚父母、抑郁，曾威胁小屋老板；同时是大冰要保护的小屋体系成员被骚扰。

### 大冰的回应序列

1. 先缓情绪：「先缓缓情绪，要不然没法聊天」
2. 了解案由、服刑、现况、母亲病情
3. **念威胁留言对照**：让小屋老板报警该不该——「威胁谁呢，法制社会」
4. 讲述老板处境（妻重病、孩子一岁、曾卖唱攒钱）——建立对比
5. **护短爆发**：「我不骂你，但我实在忍不住就俩字儿——滚蛋」
6. 对连麦者本体：劝回家陪母亲；车票钱有没有——有就能回
7. 别把自己判死刑：一辈子很长

### 可复用沟通模式

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 护短自己人 | 对威胁自己体系的人 hard boundary | 骚扰关联弱者 |
| 念证据对照 | 读出对方威胁留言逐条对质 | 装可怜实威胁 |
| 车票钱回家 | 降到最小可执行动作（陪母亲） | 愧疚型连麦 |

## 文章正文

### 刑满释放者连麦

连麦者本打算月初离开，刷到大冰视频后排麦多次未连上。当年喝酒与人打斗，对方有刀，他也借了刀，失手杀人后自首，服刑4年，出来近10年。在江苏打零工，41–42岁，身体不佳，抑郁严重，几个月无法上班。母亲患癌，没能在生前孝敬，没脸见家人；过年只回去两次，今年想回但没车票钱——大冰：有车票钱就能回家，**先回家陪陪妈妈**。

41岁云南男子，4年前失手杀人（对方有刀，他也借了刀，打斗中失手），自首服刑4年，已出来近10年。打零工没赚到钱，抑郁严重，几个月无法上班。母亲患癌几年，觉得亏欠父母，没脸见家人，过年只回去过两次；今年想回但没车票钱。大冰：**一辈子又不是现在没赚到钱意味着永远赚不到**；有车票钱就该回家陪母亲，没脸见也要陪。

### 威胁小屋老板

曾联系苏州小屋老板钟不悔要求连麦。大冰念其留言：「今晚崩溃到真的好想杀人报复……在我离开之前会让全网都知道这个社会多么虚伪冷漠……我会去你们小屋自杀的。」钟不悔妻曾生重病、孩子一岁不到，曾到铁路上卖唱攒钱给老婆治病——出于好心加过微信。大冰：**这是我兄弟，我罩着的**——「我不骂你，但我实在忍不住就俩字儿——滚蛋。」威胁谁呢？法制社会。老板报警该不该？该。

### 字幕后半段：22岁行政前台（同一 raw 文件）

22岁大专行政前台，同岗位已走两位，担心被裁，自认吃青春饭、价值不足。老板问：你凭什么涨工资？一天两三小时干完活，凭什么5500？她知职场第一性原理是价值而非努力，缺乏储蓄与抗风险能力。

大冰：**用游牧文明思维而非农耕文明思维**——不要路径依赖，这块地种不了就转场；能力之外还有概率和运气，多试总能找到。该学什么取决于**碰得着什么**，在有限选项里碰见啥学啥；没找到下一份方向前不必辞，有收入又不累。

当下及未来别害怕失业，没有铁饭碗，要有转场意识——「逐水草而居，草枯了该转就转，死赖着会饿死自己。」

大冰对前台补充：先放弃「什么都可以学」的心态——在能碰着的有限选项里碰见啥学啥；没掌握下一技能、没对标下一份工作之前，**不必主动辞**，有收入又不累，「等着人家辞你就是了。」直播间顺带提醒同龄年轻人要有转场意识、放弃铁饭碗思维。
