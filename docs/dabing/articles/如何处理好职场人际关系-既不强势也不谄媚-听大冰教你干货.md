# 如何处理好职场人际关系：既不强势也不谄媚

<BiliPlayer :bvid="'BV1KJzzBnEa9'" />


**摘要**：小镇女孩连麦，童年寄宿、父亲赌博去世，自述社交受挫、像「出于好意却招人反感」。大冰纠偏：不是「困扰」是**小反感**；慎做心理咨询只指出问题不解决。给三件套——**语速慢、开口微笑、勿秀聪明改温和**；用剑宗（外化动作）而非气宗（根本重塑）。后续案例：分公司态度好却压活——可接受就接受，同事课题归同事；讨软怕硬用温和恒定，不必谄媚。

## 内容轨 · 人生观点

- 戒备与进攻性大面上是「有自我」的好事，但具体场景会招致反感——对标动作调整即可，不必扒皮抽筋式归因。
- 「对不起造成困扰」预设不平等关系；改说/理解为**反感**更可操作。
- 心理咨询：社会面上很多只帮你看清问题，不掌握方法论解决——审慎去做，否则发现问题又解决不了。

## 技法轨 · 沟通拆解

### 互动场景

自述社恐实为受挫后不敢说话；工作中把握不好尺度；讨好感、谄媚笑、边界不清。

### 大冰的回应序列

1. **纠偏概念**：「不是困扰，叫反感小反感，可以规避」
2. **慎心理咨询**：直播间少推咨询，因多数只指出问题不解决
3. **三件套**：①语速慢——避免灌输压迫 ②开口微笑——软化措辞 ③勿秀聪明——改温和慢缓
4. **剑宗 vs 气宗**：外化表达调整，别加心理包袱
5. **分公司案例**：态度遮蔽行为——同事推回不推回是他的课题；你可接受加班就可接受
6. **恒定温和**：「笑的时候别谄媚，微笑就行」——勿高频压迫与谄媚切换

### 可复用沟通模式

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 困扰改反感 | 把「造成困扰」改理解为可规避的小反感 | 社交焦虑 |
| 语速微笑温和 | 慢+笑+不秀聪明 | 职场人际初调 |
| 剑宗外化调整 | 动作层改，不承诺根本人格重塑 | 长期创伤背景者 |
| 可接受即没问题 | 你能接受加班就做；同事不接受是他的议题 | 职场边界 |

## 文章正文

一位小镇女孩连麦，先说明基本情况：幼儿园起寄宿亲戚家，后父亲赌博、欠债、父母离婚不离家，父亲在她初中时去世；母亲无学历、上白夜班，接触少，她常在各亲戚家吃饭或学校吃，家庭教育与社会化有缺失。高中住校，大学条件好一些、沟通改善，算过得较轻松的一段；但从初中起沉迷小说，与朋友交流少。工作后社交受挫严重。她曾寻求心理咨询，咨询师帮她分析出更多问题却没有解决办法——这也是大冰很少在直播间推心理咨询的原因：社会面上很多从业者能指出问题，却不掌握方法论一起解决，审慎去做，否则陷入「发现问题又解决不了」。

她最大的困境是社交和工作中的社交：把握不好尺度，像切片里那个男孩说「老师您可以边吃饺子边跟我聊天」——出于好意却可能打扰对方。大冰纠偏：对那个小伙没有「困扰」，有的是需要在最短时间明白：**尽量不要招致反感**——不是困扰，叫**反感**，「对不起我对你造成了困扰」这句话本身有问题，关系位置非常不平等；反感、小反感可以规避，不是多大的问题。

她说多次受挫后养成不敢说话，自称社恐，但聊着反应过来了——不是社恐，有表达欲，是受挫后不敢说话。大冰：这个问题太好解决，甚至不用归因得那么细或看得那么严重，在具体动作上做几个很小调整，局面迅速可以转变，先吃颗定心丸，不是多大的事。

三件套（不是上课，只是聊天）：

**第一，语速慢下来。** 慢下来就不是灌输、倾倒状态；灌输和倾倒带有压迫感、侵略性，对方会感知压力。语速很快、高频输出时，你会忽略对方具体表达——不仅是语言，还有表情、接受度等；太快像水龙头拧很大，泡沫冲走了。

**第二，开口说话先微笑。** 笑着讲话会软化一部分可能招致反感的措辞，即便表达方式有点问题，预先做软化；真的要笑出声。

**第三，不要表现得太聪明。** 越是害怕别人发现不够聪明、越努力证明自己达标，越会引发前两个问题。对冲方式：温柔一点、慢一点、缓一点、**温和**一点。把握这三点，即便措辞有点问题，反感度大概率不至于上升到影响人际社交关系。该厉害时再厉害，该反弹时再反弹，不能频率、节奏一直调到同一状态。

关于戒备与进攻性：鉴于成长境遇，潜意识层面天然戒备、随时准备自卫——大面上没问题，指向你有进攻性、有自我，是好事；宁可这样也不希望太畏缩。但在具体生活场景会招致不快、反感，对标直接解决即可，不必扒皮抽筋把骨头看见——那是系统工程，重新养育一遍需要漫长时间。在动作上做表皮、浅层次调整：语速慢一点、态度温和一点、笑着讲话、软化表达，不要高频、侵略性、压迫感地招致不快；宁可温和说不太达标的话，也不要用其他姿态。

她问太软弱怎么办：讨好感、谄媚笑、边界不清，别人怎么说就怎么做，甚至对非领导也如此，镜中看自己白天的笑觉得太丑、太谄媚。大冰：没有让你谄媚地笑，**微笑就好**，状态恒定住；不要有时特别谄媚、有时又是高频快语速压迫感——这种切换让人不适，且你往往不自知。在乎他人情绪表达和态度，之前多在乎、当下多在乎、将来大概率也如此——这个没法帮你改。用**剑宗**（外化表达调整）而非**气宗**（从根本重塑），不累、不加心理包袱。

**案例一：** 分公司任务完不成，她和同事去协助，对方语言态度很好却把大量工作压给他们；她觉得不好意思，同事则坚定推回「我们完不成、晚上还有别的事」。她震惊：原来可以拒绝。大冰问：若同事不揭露，你可接受还是不可接受？她说可能会加班做完——**可接受就没问题**，你可接受、同事接受不了是他的课题，做你自己就好。

**案例二：** 她一人去协助，资历低负责端茶倒水，同事说「你怎么会去做这些」——两个例子都指向：别人告诉你不顿时，瞬间倒向认同，主导权不在你这；事实上别人不告诉你时，你很自洽、可接受，是外力打破内在平衡，太容易受别人影响。涉及根本：没有特别坚硬笃定的自我，不是做错了什么，是养育、成长过程有缺失，不存在几句话就重塑自我。相对改善可在职场生态环境里占据稍微优势的位置——但可能附带不好指向。当下建议：再有人提醒「这事不对、你不该端茶倒水」时，一听就行，问一句——他是不是特别爱你？跟你说这话的目的是什么？是不是打破你的自洽平衡？若客观意义上被使唤、加班、被剥削，但你的状态可接受、自洽，他人来打破时你又明显受影响，先想对方发心和目的。自我不够强大就容易被影响。没有治本的法子，实话实说；不建议靠心理学书刻意学就有强大自我；以他所学，掌握的理论在她这儿不好用，只告诉最粗浅的动作调整。
