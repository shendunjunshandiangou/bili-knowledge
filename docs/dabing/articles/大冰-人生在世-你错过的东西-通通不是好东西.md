# 大冰：人生在世，你错过的东西，通通不是好东西

<BiliPlayer :bvid="'BV169KN6AEif'" />


**摘要**：大冰对连麦者强调两句：**错过的一律不是好东西**（前男友、前工作亦然）；**没有明天**——不要预设明天，「所谓的后天都是今天」；苹果比喻：最好吃的苹果只属于今天，留着「明天」口感每天下降一点；不是「活在当下」口号，是**活在现在**；可以想有用的——今天怎么活好、怎么不活亏；别为昨天焦虑错失、为明天焦虑没着落。

## 内容轨 · 人生观点

- **错过皆非好物**：曾经错过的通通不是好的选择，重复说三遍强化。
- **没有明天**：预设的明天都是今天；专注今天怎么活好、怎么不活亏。
- **苹果只属于今天**：不舍得吃、留给明天——没领会苹果跟明天没关系，一天不吃口感降一点（时光/日子）。

## 文章正文

（开篇切片）大冰：**人生在世，只要是你错过的东西，通通不是好东西**——前男友、前好工作，错过了就不是最好选择；别神化错过之物。

国外博后连麦（芝加哥），来美近3年，30岁，学医。毕业时导师留高校任教（有编），她因访学一年像旅游、不想回国面对现实，选择留美；年底母亲大病，靠父亲远程指导挂号，后悔仓促、放弃国内机会。老板经费问题，半年每周组会说未来不确定，她渐抑郁；父母来美觉得环境不好、工作家庭未定，电话逼她；掉发、气血亏，心理医生文化背景不同沟通不畅。

她认为价值排序：健康、父母、自由、自我提升、金钱稳定工作——30岁后与父母期待冲突，理智懂只能做好手头事，情感反复陷漩涡。

大冰：最大痛苦来自**错失机会被父母评估影响**，与当下未来对比——但那是**有问题的思考方法**。留校有编？你扯淡，非升即走、站队——你在**神化**错过的东西。

第二观点：**没有明天**——预设的明天都是今天；别为昨天明天活，**今天怎么活好、怎么不活亏**。苹果比喻：最好吃的苹果只属于今天，留着「明天」口感每天下降。你懂个屁——把别人生活确定化，「不确定才是最大的确定」；唯一不变的是变化。

第三：父母介入你的**价值评判体系**让你动摇——左脚东右脚西把自己撕裂。对策：父母操心时说**「我有数，放心」**——先解决情绪，拆弹；即便没数也先说有数。不要试图让他人满意——最不利的路径；要有**防火墙**，别让任何人介入价值判断。

别跟留校任教的人比——勾心斗角压力阈值你没法想象。强大自我≠内核稳，可从**未体验的人生体验**找支点（如火人节等，保证安全合法）。大冰现场「掉牙」演示生活乐趣。最后：不会过得差，别跟别人比。
