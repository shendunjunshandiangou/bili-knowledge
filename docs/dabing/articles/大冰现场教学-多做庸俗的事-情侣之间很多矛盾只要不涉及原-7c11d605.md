# 大冰现场教学：多做庸俗的事

<BiliPlayer :bvid="'BV1jVb1zVEPm'" />


**摘要**：28岁农村出身公务员与相亲女友分手又藕断丝连。大冰连环追问（介绍人、女方消费、是否一路人），判断无原则问题、分手正常；劝别自卑——人家已做过算法评估；别为安全感背房贷；挂麦后拿连麦话题约饭，「该庸俗时庸俗」——约出来吃饱后亲密，很多矛盾可化解。

## 一、案情梳理

- 公务员，去年8月相亲认识同龄稳定工作女孩；近期因爽约、抽烟喝酒、未来规划（他想结婚生子、她暂不考虑）、女方家庭条件更好而分手。
- 周末仍偶尔见面，双方都说想对方。

## 二、大冰判断

- 「问啥人说啥」——追问介绍人、车、包、吉他等，结论「不太像是一路人，但不见得不可以谈」。
- 无底线原则矛盾；「谁谈恋爱不得分个十几回手」；「没有哄不回来的女朋友，就看你怎么哄」。
- 女方家庭已评估过基本面；自卑「不用瞎想」。
- 买房：跳出小农意识，别为「房本=美好生活」背大负债；可等订婚后再买小二手。

## 三、行动建议

- 挂麦后用连麦老头的话头跟女友聊买房观 → 约干锅/米线 → 「趁意志力薄弱该啃就啃，干点庸俗的事，这一页就翻过去了」。
- 「该庸俗的时候是要庸俗的……一身正气到立阳麦了」。

## 内容轨 · 人生观点

- 非原则情侣矛盾可主动修复，别冷战。
- 农村孩子考公后不必用买房证明成功；朋友问房与自己无关。

## 技法轨 · 沟通拆解

### 互动场景

憨直公务员情感困惑 + 自卑 + 买房焦虑。

### 大冰的回应序列

1. **信息对称追问**：谁介绍、女方家境消费、是否一路人。
2. **正常化**：分手可逆、改造≠掌控欲。
3. **重构**：别小农意识背贷；人家已评估你。
4. **收尾给话术+行动**：连麦话题作由头 → 约饭 → 庸俗亲密化解。

### 可复用沟通模式

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 问啥人说啥 | 细节追问再下判断 | 关系/相亲 |
| 话头嫁接 | 第三方观点引出新对话 | 冷战破冰 |
| 庸俗化解 | 吃饭+亲密替代讲道理 | 情侣小矛盾 |
