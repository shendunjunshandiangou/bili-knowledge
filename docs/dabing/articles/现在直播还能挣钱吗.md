# 现在直播还能挣钱吗

<BiliPlayer :bvid="'BV1v8VpzME7A'" />


**摘要**：79 年软件从业者连麦，请教互联网主播从小到大的盈利模式与平台运作原理。大冰先定边界「我不做主播，只是聊天」，再以**法无禁止即可为 + 公序良俗不破坏**为框架，阐述三层认知：① 平台是贫寒子弟为数不多上升通道，承载「希望」；② 轻资产低投入拼运气，比线下创业（如小饭馆）风险小；③ 八零后无资格苛责年轻人，应包容低俗主播——「他配不配吃饱饭」。对平台未来：希望砍掉塔尖、流量下沉，但具体怎么做「轮不到咱们想」。

## 内容轨 · 人生观点

- **法无禁止即可为 + 公序良俗**：衡量一切主播盈利模式的前置双条件；在此之下「你干啥都行」。
- **平台承载希望**：快手、某音等是底层向上为数不多的气口；「一个人成了会抚慰一堆人的心」——像彩票有人中奖才有人买；广谱人群丧失希望则社会糟糕（如年轻人不结婚归因于对未来无希望）。
- **速朽不是坏事**：快速成名又速朽，阶段性既得利益者赚到钱后把空间让给下一个——平台承载有限，尽量多人有希望值。
- **轻资产创业替代线下**：人到中年失业想开小饭馆——「没干过凭什么干成」；线上号是轻资产、低投入、拼运气，即便干不成不伤筋动骨。
- **八零后无资格教年轻人**：改革开放 40 年（1979–2019）最幸运一代，吃到红利、相信天道酬勤；零零后一零后没做错什么，空间窄、上升通道封闭，**没有资格俯视、苛责、鄙视**。
- **包容低俗主播**：耍宝搞怪出丑为吃饭，「他配不配吃好饭、吃饱饭？他有没有爹妈要养？」——自己吃饱了瞧不起别人，凭什么。
- **理想化平台形态**：希望头部主播「死」（打引号），流量向下覆盖，只有腰部和大量普通人直播间；不要托拉斯化、不要平台权贵阶级；具体怎么做没资格说，但「有没有权利做梦？有」。

## 技法轨 · 沟通拆解

### 互动场景

79 年（1980 年生）软件从业者，做过电视主持人到 30 岁，对快手等平台主播盈利模式困惑多年，连麦请教；语气恭敬称「大冰老师」。

### 大冰的回应序列

1. **定边界**：「你找我有个啥事」→ 对方夸逻辑思维 → 「我不做主播，我也不是这个，咱俩只是聊聊天」——降低被当作行业权威的预期。
2. **双框架前置**：先抛 slogan「法无禁止即可为」，再推一步「公序良俗不破坏」——作为全文论述罩子。
3. **不反对带货**：「我可以不带，但从来不反对任何人去带」——先解除道德批判立场。
4. **三层认知递进**：希望值 → 轻资产创业 → 包容认知；每层用生活例子（彩票、小饭馆、郭有才）锚定。
5. **回应第二个问题**：对方问平台如何良性发展 → 「也就限于刚才那几点了」→ 理想是砍掉塔尖但「轮不到咱们想」「我不是运营」。
6. **留半句**：「你肯定有想法，但有些话不适合在这上面聊」——知止。

### 可复用沟通模式

**本集技法层较弱**——以观点输出为主，连麦互动偏请教式；可复用模式：

| 模式 | 做法 | 适用 |
| --- | --- | --- |
| 先定边界 | 「我不做 X，只是聊天」 | 被当作行业权威时 |
| 双框架罩子 | 法无禁止 + 公序良俗，再展开 | 谈灰色盈利/网红现象 |
| 三层递进 | 希望→风险→包容，层层加深 | 复杂社会议题长答 |
| 知止留半句 | 「有些不适合在这聊」 | 敏感运营话题 |

## 文章正文

一位79年出生、做软件的连麦者请教快手等平台主播如何从小做到大、赚钱模式是什么。大冰说只做了解不做主播，前置 slogan 是"法无禁止即可为"，其下再加"公序良俗不破坏"。

他不反对当网红和带货——当今中国社会上升通道非常封闭，贫寒子弟往上走的缝早闭合了；快手、抖音等是为数不多的"气口"，可实现底层向上覆盖，"这叫机会空间"。若通道还存在，意义巨大——因为那是"希望"：一个人成了会抚慰一堆人，让大家都觉得还有希望，像买彩票有人中奖才有人继续买。年轻人不结婚，归因到对人生未来丧失希望、没有想象力空间。

第二层认知：对普通人而言，这是为数不多的轻资产、低投入、拼运气的"投资项目"，可规避线下创业风险——他直播间碰到中年失业想开小饭馆的人一定拦住，"你没干过凭什么干成"。第三层：应包容普通人把生活期许和盈利手段放网上；希望未来快手没有大主播，只有中部腰部和大量普通人直播间——比如为孩子学费直播两个月，"爆火一阵子就满足了他之后号就别干了"。塔尖机会和财富应向下覆盖。具体怎么做轮不到他们想，但他有权做梦。

连麦者认可，又问互联网如何良性发展。大冰说真实观点就限于以上几点；希望现有头部"死掉"（打引号），把空间让出来，不要托拉斯化、不要快手权贵阶级，"两头小肚子大"比金字塔更科学（原文此处截断）。

他还说八零后是改革开放最幸运一代，吃了时代红利，没资格教年轻人，有时底层逻辑须理想化一点才对得起吃过的红利；79到19这40年是中华文明几千年最美好的40年。不应污名化年轻博主耍宝搞怪——他们有没有追求美好生活的权利？零零后一零后空间窄、上升通道封闭，"你就让人家吃上碗饱饭又能怎么地"。
