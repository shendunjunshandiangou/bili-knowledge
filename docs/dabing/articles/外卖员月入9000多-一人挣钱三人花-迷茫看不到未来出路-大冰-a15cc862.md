# 外卖员：初一辍学与惜命

## 文章正文

23岁（字幕「223」应为23），初一辍学，送外卖月均9000多，闺女1岁，一人挣钱三人花。想稳定、不想10年20年一直如此，怕50、60岁仍无保障，怕媳妇过不好日子。

大冰直说：**命运在初一下学时已决定大半**——错失唯一一次改变命运的路径。寒门子弟只有尽量提升学业完成度，其他上升路径堵得差不多；别信农村出身大火的低概率。

现在若学技术最好，但学会放弃当前9000收入；第三，**锻炼身体、惜命、买保险**，不掌握其他谋生技能，只能靠出卖时间体力换钱——尽量良性延续，**垮不起、生不起病**，一旦病倒孩子老人没人管。

不服「凭什么只能干外卖」——不具备系统完整教育与职业技能培训两项基本条件。**只能干这个**，别拿积蓄创业，大概率赔掉；别搏运气，「赔不起拉饥荒咋办」。每次想迷茫就骂当年十几岁不懂事的自己——「现实世界没人在乎」童年没管、交不起杂费书费。

大冰三个月来遇不想上学连麦必劝回校——上升通道封闭，现在为数不多的方式不是上升而是尽量活下去甚至体面温饱；读书必须读。若研究哪项技术，做好心理准备——未必赚更多，或许为三十八九岁没力气下苦力时留一口温饱的伏笔。连麦者接受「只能干这个」，不再琢磨创业搏运气；大冰一边说一边心酸，但「兄弟没有办法。」
