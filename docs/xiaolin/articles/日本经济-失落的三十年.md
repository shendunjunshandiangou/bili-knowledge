> **摘要**：填「日本经济泡沫」坑。1991 泡沫破裂后，日本陷入通缩与增长停滞：1997 银行体系崩塌、零利率（1999）、**QE 由日本发明**（2001）、2006–07 短暂复苏后再遭次贷/福岛/中国竞争；2012 起**安倍经济学**（负利率+QQE）推升资产价格但 GDP/通胀仍弱。深层原因：老龄化、通缩预期锚定、终身雇佣下工资停滞。

## 泡沫破裂与信用危机

- 1985 广场协议 → 日元升值 + 过度放贷 + 财团哄抬 → 股楼市泡沫
- 1991 破裂：不仅是资产泡沫，更是**信用泡沫**（银行大量坏账）
- 1993 起财政货币双宽松短暂复苏；**1997** 多家银行破产，信贷收紧 → **通缩螺旋**：物价跌→企业收入降→工资降→需求降→物价再跌
- 1998 起连续 4 年通缩；2000 年国债/GDP 约 **130%**（欧盟标准 60%）

## 货币政策升级：零利率 → QE → 负利率 → QQE

| 阶段 | 要点 |
| --- | --- |
| 1999 零利率 | 隔夜拆借零利率；2000 短暂升回 0.5% 后 2001 再归零 |
| 2001 QE | **全球首创**量化宽松，央行买长期国债等；4 年资产负债表扩 2.5 倍；通胀略好转、信贷止跌但缓慢 |
| 2006–07 | 停 QE、加息至 0.5%；全球繁荣下日本略恢复 |
| 2008+ | 次贷、日元升值、2011 福岛（约 2350 亿美元）、中国/韩国竞争 → 再衰退 |
| 2012 安倍经济学 | 三支箭：激进货币（黑田东彦）、财政刺激、结构改革（效果慢/名存实亡） |
| 2016 负利率 | 对商业银行部分准备金 **-0.1%**，逼银行放贷 |
| QQE | 目标 10 年期国债 **≤0.25%**；买国债、公司债、甚至股票（成市场最大单一股东约 10%） |

2020 疫情：约 **1 万亿美元**刺激，债务/GDP 约 **260%**。2022 俄乌推高油价，日本终于出现输入性通胀，但是否走出「失落 30 年」存疑（视频留续集）。

## 为何猛刺激仍难通胀？

1. **人口**：2010 后婴儿潮退休，劳动人口减、老人需求低、储蓄高；赡养与福利压制总需求
2. **通缩预期锚定**：商家不敢涨价甚至道歉；成本升则削成本、降薪；需「几乎所有商品同时涨价」才易打破
3. **终身雇佣**：员工不跳槽→雇主无涨薪压力；工资 90 年代起几乎停滞
4. **企业/个人行为**：钱流入分红、买股买房买国债或海外，而非扩大生产消费

结果：流动性注入 → **资产价格狂欢**（股债涨），GDP/通胀仅「2P」级改善。

## 宏观政策速记

- **货币政策**：利率、准备金、QE/QQE，控通胀为主
- **财政政策**：税收、政府开支、基建、补贴（如新能源）
