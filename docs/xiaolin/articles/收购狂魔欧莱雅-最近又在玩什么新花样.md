# 收购狂魔欧莱雅，最近又在玩什么新花样？

<BiliPlayer :bvid="'BV1gM4m1S7ew'" />


> 摘要：借巴黎 Viva Tech 科技展实地探访，拆解全球最大美妆集团欧莱雅的成功内核——收购构建的集团化经营（研发/供应链/渠道协同 + 品牌营销内部竞争），以及行业老大面对增长放缓与 AI 浪潮的新打法：收购买当下市场（史上最大手笔 174 亿人民币收购 Aesop）与投资初创押注未来（BOLD 创投基金、开放式创新、专利导向）两条腿走路。

## 一、现场：美妆巨头逛科技展

欧洲最大科技展 Viva Tech（巴黎，年观展 15 万人）上，欧莱雅的展台异常繁忙。一个美妆巨头来科技展凑什么热闹？现场展品包括：

- **3D 打印"人皮"**：打印机挤出类似基底液的材料，培养皿中凝固成模拟皮肤——实验室版本复杂精准得多。用途：不必在动物或人体上做实验，用模拟皮肤测试产品功效、模拟环境对皮肤的伤害；
- **红外吹风机**：吹风 + 红外光，类似"风和太阳"的组合让头发自然吹干、伤害更小——不是欧莱雅自研，而是与深圳一家科创公司合作的产品；
- **My Hair ID**：检测头发颜色的设备；
- **纳米晶片导入仪**：内含大量纳米级硅晶片，每分钟 5000 次按摩帮助精华更好导入皮肤——专为怕做医美但想要更好效果的人设计（出自欧莱雅 2020 年在中国发起的"Big Bang 美妆科技创造营"孵化项目）；
- **皮肤检测仪**：用正常光、偏振光（拍出皮肤本质）、紫外光（看毛孔）三种光拍摄正面和左右侧分析皮肤。

展台的聪明之处：让人真体验——现场做头发、测皮肤、发小样。

## 二、欧莱雅是谁：全球最大美妆集团

兰蔻、美宝莲、赫莲娜、YSL 和阿玛尼的美妆香水线都是欧莱雅旗下品牌；产品覆盖护肤、美妆、香水、护发、男女全线。营收占全球美妆市场的七分之一，去年 400 多亿欧元、比第二名高 60%。

**成功的决定性因素：收购**。旗下近 40 个品牌，只有巴黎欧莱雅、欧莱雅 Pro 等三个是自己的，剩下全是买的。但收购绝不是砸钱买买买那么简单。

## 三、收购的第一重精髓：协同效应

欧莱雅 1909 年由一位化学家创立，最初是染发剂品牌，早期一直做染发剂细分品类。1957 年第二任 CEO 达勒上任——资本市场高手、野心勃勃：1963 年欧莱雅在法国上市；1964 年收购高端品牌**兰蔻**开始高端化；1971 年收购**碧欧泉**进军男士市场；1985 年收购阿玛尼香水线——逐步形成集团化经营格局。

**协同效应 = 1+1>2**：研发、市场渠道、供应链，所有子品牌共用一套，运营效率提高。案例：90 年代欧莱雅看中中国市场，1996 年收购**美宝莲**——看中的正是美宝莲已铺开的线下渠道；2003 年又收购国内生产线。此后欧莱雅大军的其他品牌（YSL、薇姿、理肤泉）进中国时，粮草和路都铺好了——像家族企业：爸爸先打江山疏通关系，孩子们进来跟叔叔阿姨喝喝酒就行了。

## 四、第二重精髓：品牌营销上的"竞争 + 共享"机制

营销品牌对消费品太重要了——全球前几大广告主里，欧莱雅排第四，第一是宝洁、第三联合利华，全是消费品公司。而营销比拼的是**创意**。

欧莱雅的机制（据其中国区高管交流所述）：旗下品牌共享供应链、研发资源，但**销售和营销完全独立**、各卖各的，彼此有竞争关系——虽是一个妈生的，也得争预算。同时集团不完全散养：尽可能促进"孩子们"交流合作、经验共享——兰蔻发现一个好用的营销招数，集团就让赫莲娜、修丽可也试试。类比基金公司：本来各基金经理单打独斗，老板让你把策略分享给同事——你原来靠这个策略躺赚，现在别人也会了，你就得继续创新。当然不是所有信息全盘托出，而是在可接受限度内促进共享、逼着共同进化。所以欧莱雅营销强，是因为有一套持续刺激新想法产生的机制。

**创意花絮**：兰蔻 2020 年买下南法尼斯一座私人玫瑰庄园并改造（背后故事有了、供应链条具象化了）；巴黎奥运会预算有限，不建大场馆，把赛场放到埃菲尔铁塔等名胜古迹旁，喊出"百年奥运重回法国"（1924 年奥运也在巴黎）——浪漫、仪式感全有还省钱。法国人搞创意确实有一套。

## 五、收购策略总结与外部环境

内因：先通过收购积累研发、供应链、渠道、营销体系；再通过收购和投资找到有潜力的小品牌，在体系共享助推下让小品牌迅速做大、占领全球市场、全球盈利——构建美妆帝国。外因：全球化几十年间总有高速增长的经济体承接产能扩张——最近十几年就是中国，现已是欧莱雅全球第二大市场，法国 CEO 都起了中文名"叶鸿慕"（原字幕"叶红木"，按公开资料修正）。

## 六、老大的烦恼与两条腿走路

大哥当久了也有高处不胜寒的挑战：大到一定程度可能最先碰到天花板。近几年全球经济不景气、中国美妆市场增速放缓（去年还下滑）；另一边 AI、数字化爆发式增长。动荡时期最慌的是既得利益者——"别哪天美妆界搞出个 OpenAI，把我老巢抄了"。

出路是创新，难在**怎么**创新。大型美妆集团大致两个方向：生物科学（多找 PhD）和科技。美妆的科技创新与谷歌、微软完全不同：欧莱雅自称"create beauty"的企业——"更美"是主观的、千人千面（情人眼里出西施），不像通信业从 4G 到 5G 有上传更快、延迟更低等明确指标指导研发投入。欧莱雅内部有个 packaging lab（包装实验室），专家整天研究什么材料、材质、颜色让人觉得更美更有质感——"美"更多元也更复杂。

**目标不确定 → 广撒网分散风险 → 适合投初创公司**：风险大但单笔投资小、网撒得广，比重金买成熟公司更合适。这正是欧莱雅展台大部分是初创合作项目的原因：红外吹风机来自 2018 年成立的创投基金 **BOLD**（原字幕"Bolt"，按公开资料修正）投的项目；纳米导入仪来自中国"Big Bang 美妆科技创造营"。欧莱雅把这种模式叫**开放式创新**。采访其开放式创新负责人："投初创最看中什么？"——本以为会是自上而下的赛道配置（AI 投 30%、AR/VR 投 30%），意外的是对方说没有明确的整体规划，最看中的是**专利**：你有没有独特的、与众不同的技术。

**收购 vs 初创投资的平衡**（站在 CEO 角度）：

| | 收购 | 投资早期初创 |
| --- | --- | --- |
| 标的状态 | 产品已跑通，买的是市场潜力 | 可能连产品都没有，买的是技术/工艺/科研 |
| 风险与金额 | 金额大、风险相对小 | 风险大、不确定性强 |
| 导向 | 市场盈利导向——赚当下的钱，保障集团盈利与增长 | 产品研发导向——押宝未来的机会 |

收购案例：去年 174 亿人民币收购澳洲护肤品牌 **Aesop**（欧莱雅史上最大收购），当时 Aesop 在中国刚起步、据说只有两家门店——典型的"产品成熟、市场未开发"。科研端还有欧莱雅基金会与联合国教科文组织合办的"杰出女科学家奖"（颜宁教授曾获奖）。

**总结**：欧莱雅先靠收购扩品类、搭集团化体系（研发/供应链/渠道协同 + 品牌营销竞合），乘全球经济增长构建美妆帝国；如今面对增长放缓，开始通过更早期的投资布局专利与技术、押宝未来。一个美妆消费品企业，我们看到的是琳琅满目的广告，背后是一个庞大、复杂、多元的体系。
